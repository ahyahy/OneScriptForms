НаблюдательФайловойСистемы.ПриПереименовании (FileSystemWatcher.Renamed) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает код для выполнения при переименовании файла или каталога по заданному пути.

Примечание

Переименование просматриваемого каталога не вызовет уведомления. Уведомления создаются только для записей в просматриваемом каталоге.

Пример

НаблюдательФайловойСистемы1.ПриПереименовании = "НаблюдательФайловойСистемы1_ПриПереименовании()";
