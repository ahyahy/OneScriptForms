РежимРазмераВкладок (TabSizeMode) Перечисление

Описание

Задает способ изменения размеров вкладок в наборе вкладок.

Члены

Имя члена Описание Значение
ЗаполнитьВправо (FillToRight) Ширина каждой вкладки изменяется таким образом, чтобы каждая строка вкладок заполнила всю ширину контейнерного элемента управления. Это применимо только к элементам управления с более чем одной вкладкой в строке. 1
Постоянный (Fixed) Все вкладки в элементе управления имеют одинаковую ширину. 2
Стандартный (Normal) Ширина каждой вкладки изменяется в соответствии со сведениями, отображаемыми на вкладке, а размер вкладок в строке не настраивается для заполнения во всю ширину контейнерного элемента управления. 0
