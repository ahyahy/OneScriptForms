ЭлементУправления.НайтиФорму (Control.FindForm) Метод

Синтаксис

НайтиФорму()

Параметры

Возвращаемое значение

Тип: Форма (Form).

Описание

Возвращает форму, в которой находится элемент управления.

Примечание

Значение свойства Родитель (Parent), для элемента управления может не совпадать со значением Форма (Form), возвращенным методом НайтиФорму (FindForm). Например, если элемент управления Переключатель (RadioButton) содержится в элементе управления РамкаГруппы (GroupBox), и РамкаГруппы (GroupBox) находится на Форма (Form), для Переключатель (RadioButton) свойство Родитель (Parent) - это РамкаГруппы (GroupBox), и для РамкаГруппы (GroupBox) свойство Родитель (Parent) - это Форма (Form).

Пример

Сообщить("Панель1.НайтиФорму().Текст = " + Панель1.НайтиФорму().Текст);

Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Форма1.Текст = "Форма1";

Панель1 = Ф.Панель();
Панель1.Родитель = Форма1;
Панель1.Верх = 20;
Панель1.Лево = 20;
Панель1.СтильГраницы = Ф.СтильГраницы.Трехмерная;
Надпись1 = Панель1.ЭлементыУправления.Добавить(Ф.Надпись());
Надпись1.Текст = "Панель1";
Надпись1.Стыковка = Ф.СтильСтыковки.Верх;

Надпись2 = Ф.Надпись();
Надпись2.Родитель = Панель1;
Надпись2.Размер = Ф.Размер(100, 60);
Надпись2.ЦветФона = Ф.Цвет().Бирюзовый;
Надпись2.Текст = "Надпись2";
Надпись2.СтильГраницы = Ф.СтильГраницы.Трехмерная;
Надпись2.Центр();

Кнопка1 = Ф.Кнопка();
Кнопка1.Родитель = Надпись2;
Кнопка1.Размер = Ф.Размер(70, 20);
Кнопка1.ЦветФона = Ф.Цвет().Лайм;
Кнопка1.Текст = "Кнопка1";
Кнопка1.Центр();

Сообщить("Кнопка1.НайтиФорму().Текст = " + Кнопка1.НайтиФорму().Текст);
