Прямоугольник (Rectangle) Свойства

Свойства класса Прямоугольник (Rectangle) перечислены ниже. Полный список членов класса Прямоугольник (Rectangle) смотрите в разделе Прямоугольник (Rectangle) Члены.

Свойства

Верх (Top) Возвращает координату по оси Y верхнего края прямоугольника.
Высота (Height) Возвращает или задает высоту в прямоугольника.
Игрек (Y) Возвращает или задает координату по оси Y левого верхнего угла прямоугольника.
Икс (X) Возвращает или задает координату по оси X левого верхнего угла прямоугольника.
Лево (Left) Возвращает координату по оси X левого края прямоугольника.
Низ (Bottom) Возвращает координату по оси Y, являющуюся суммой значений свойств Прямоугольник.Игрек (Rectangle.Y) и Прямоугольник.Высота (Rectangle.Height) данного прямоугольника.
Положение (Location) Возвращает или задает координаты левого верхнего угла прямоугольника.
Право (Right) Возвращает координату по оси X, являющуюся суммой значений свойств Прямоугольник.Икс (Rectangle.X) и Прямоугольник.Ширина (Rectangle.Width) данного прямоугольника.
Размер (Size) Возвращает или задает размер этого объекта Прямоугольник (Rectangle).
Ширина (Width) Возвращает или задает ширину прямоугольника.
