РезультатДиалога (DialogResult) Перечисление

Описание

Задает идентификаторы, которые определяют возвращаемое значение диалогового окна.

Члены

Имя члена Описание Значение
Да (Yes) Диалоговое окно возвращает значение Да. 6
Нет (No) Диалоговое окно возвращает значение Нет. 7
ОК (OK) Диалоговое окно возвращает значение ОК. 1
Отмена (Cancel) Диалоговое окно возвращает значение Отмена. 2
Отсутствие (None) Диалоговое окно возвращает значение Ничего. Это означает, что модальное диалоговое окно не закрывается. 0
Повторить (Retry) Диалоговое окно возвращает значение Повторить. 4
Прервать (Abort) Диалоговое окно возвращает значение Прервать. 3
Пропустить (Ignore) Диалоговое окно возвращает значение Пропустить. 5
