ТипЭлементаСетки (GridItemType) Перечисление

Описание

Действительный тип элемента сетки для СеткаСвойств (PropertyGrid).

Члены

Имя члена Описание Значение
Категория (Category) Компонент сетки, являющийся именем категории. Категория представляет собой описательную классификацию групп строк ЭлементСетки (GridItem). К типовым категориям относятся Поведение, Макет, Данные и Внешний вид. 1
Корневой (Root) Корневой элемент в иерархии сетки. 3
Свойство (Property) Компонент сетки, соответствующий свойству. 0
ЭлементМассива (ArrayValue) Компонент сетки, соответствующий элементу массива. 2
