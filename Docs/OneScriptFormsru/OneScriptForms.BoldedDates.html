ВыделенныеДаты (BoldedDates) Класс

Описание

Представляет массив объектов типа Дата, который определяет, какие непериодические даты будут выводиться полужирным шрифтом в объекте Календарь (MonthCalendar).

Список всех членов этого типа смотрите в разделе ВыделенныеДаты (BoldedDates) Члены.
