ПодэлементыСпискаЭлементов (ListViewSubItemCollection) Методы

Методы класса ПодэлементыСпискаЭлементов (ListViewSubItemCollection) перечислены ниже. Полный список членов класса ПодэлементыСпискаЭлементов (ListViewSubItemCollection) смотрите в разделе ПодэлементыСпискаЭлементов (ListViewSubItemCollection) Члены.

Методы

Вставить (Insert) Вставляет подэлемент в коллекцию по указанному индексу.
Добавить (Add) Добавляет подэлемент к коллекции подэлементов.
Очистить (Clear) (унаследовано от КоллекцияОснова (CollectionBase)) Удаляет все объекты из коллекции.
УдалитьПоИндексу (RemoveAt) (унаследовано от КоллекцияОснова (CollectionBase)) Удаляет объект по указанному индексу из коллекции.
Элемент (Item) Устанавливает и/или возвращает подэлемент по указанному индексу в коллекции.
