ПолеСписка (ListBox) Конструктор

Синтаксис

ПолеСписка()

Параметры

Описание

Создаёт новый экземпляр класса ПолеСписка (ListBox).

Примечание

Пример

  • Заполнение поля списка с помощью массива как источника данных:

    • ПолеСписка1 = Ф.ПолеСписка();
ПолеСписка1.ИсточникДанных = МассивСписок1;

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Ширина = 310;
Форма1.Отображать = Истина;
