Графика (Graphics) Класс

Описание

Формирует поверхность рисования GDI +.

Список всех членов этого типа смотрите в разделе Графика (Graphics) Члены.

Объект Графика (Graphics) можно получить с помощью метода СоздатьГрафику (CreateGraphics) какого либо элемента управления, либо с помощью метода ФормыДляОдноСкрипта.Графика (OneScriptForms.Graphics). Поведение объекта будет зависеть от способа его получения. Нагляднее это можно увидеть на примерах методов Графика.ИзИзображения (Graphics.FromImage) и Графика.РисоватьЛинию (Graphics.DrawLine).
