Синтаксис

ИнформацияЗапускаПроцесса(<ИмяФайла>, <Аргументы>)

Параметры

ИмяФайла (необязательный)
Тип: Строка.
Имя файла приложения, с которым должен запускаться процесс.
Аргументы (необязательный)
Тип: Строка.
Аргументы командной строки, передаваемые в приложение при запуске процесса.

Возвращаемое значение

Тип: ИнформацияЗапускаПроцесса (ProcessStartInfo).

Описание

Создаёт новый экземпляр класса ИнформацияЗапускаПроцесса (ProcessStartInfo).

Примечание
