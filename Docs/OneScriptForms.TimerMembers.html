Конструктор

Таймер (Timer) Конструктор Создаёт новый экземпляр класса Таймер (Timer).

Свойства

Интервал (Interval) Возвращает или задает время в миллисекундах между отметками таймера.
Метка (Tag) Возвращает объект, содержащий дополнительные данные.
ПриСрабатыванииТаймера (Tick) Возвращает или задает код для выполнения по истечении указанного интервала таймера и таймер включен.
Тип (Type) (унаследовано от Компонент (Component)) Возвращает тип текущего экземпляра.

Методы

Начать (Start) Запускает Таймер.
Освободить (Dispose) (унаследовано от Компонент (Component)) Освобождает ресурсы, используемые Компонент (Component).
Остановить (Stop) Останавливает Таймер.
