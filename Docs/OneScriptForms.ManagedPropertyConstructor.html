УправляемоеСвойство (ManagedProperty) Конструктор

Синтаксис

УправляемоеСвойство(<УправляемыйОбъект>, <УправляемоеСвойство>, <Коэффициент>)

Параметры

УправляемыйОбъект (обязательный)
Тип: Произвольный.
Объект, свойство которого связано с полосой прокрутки.
УправляемоеСвойство (обязательный)
Тип: Строка.
Свойство объекта связанного с полосой прокрутки.
Коэффициент (необязательный)
Тип: Число.
Множитель для масштабирования величины свойства объекта связанного с полосой прокрутки.

Описание

Используйте метод ФормыДляОдноСкрипта.УправляемоеСвойство (OneScriptForms.ManagedProperty) для создания нового экземпляра класса УправляемоеСвойство (ManagedProperty).

Примечание

Пример

УправляемоеСвойство1 = Ф.УправляемоеСвойство(Панель14, "Ширина", 3.9);

Панель14.Высота = ВертикальнаяПрокрутка1.Значение;

Надпись14.Текст = "" + ВертикальнаяПрокрутка1.Значение;
Надпись14.Шрифт = Ф.Шрифт(, 12, ); 

МассивСписок1 = Ф.МассивСписок();
МассивСписок1.Добавить(Ф.УправляемоеСвойство(Панель14, "Высота", 3.6));
МассивСписок1.Добавить(Ф.УправляемоеСвойство(Надпись14, "Текст"));
ВертикальнаяПрокрутка1.УправляемыеСвойства = МассивСписок1;

Надпись16 = Форма1.ЭлементыУправления.Добавить(Ф.Надпись());
Надпись16.Положение = Ф.Точка(ВертикальнаяПрокрутка1.Право + 20, Надпись14.Низ + 30);
Надпись16.Размер = Ф.Размер(Надпись13.Ширина, 20); 
Надпись16.Текст = "Позиция горизонтальной прокрутки (Min = " + ГоризонтальнаяПрокрутка1.Минимум + ", Max = " + ГоризонтальнаяПрокрутка1.Максимум + "):";
