ЭлементыПоляВыбора.Добавить (ComboBoxObjectCollection.Add) Метод

Синтаксис

Добавить(<Значение>)

Параметры

Значение (обязательный)
Тип: Произвольный.
Значение элемента, который будет добавлен в коллекцию.

Возвращаемое значение

Тип: ЭлементСписка (ListItem).

Элемент списка, который был добавлен в коллекцию. Если параметр Значение не является экземпляром класса ЭлементСписка (ListItem), то будет создан объект ЭлементСписка (ListItem) со свойством Текст (Text) равным параметру Значение, приведенному к строке и свойством Значение (Value) равным параметру Значение.

Описание

Добавляет элемент в список элементов для объекта ПолеВыбора (ComboBox).

Примечание

Пример

ЭлементыПоляВыбора1 = ПолеВыбора1.Элементы;
ЭлементыПоляВыбора1.Добавить("Просто строка");
