ВыравниваниеСодержимого (ContentAlignment) Перечисление

Описание

Указывает способ выравнивания содержимого на поверхности рисования.

Члены

Имя члена Описание Значение
ВерхЛево (TopLeft) Содержимое выравнивается вертикально сверху и горизонтально по левому краю. 1
ВерхПраво (TopRight) Содержимое выравнивается вертикально сверху и горизонтально по правому краю. 4
ВерхЦентр (TopCenter) Содержимое выравнивается вертикально сверху и горизонтально по центру. 2
НизЛево (BottomLeft) Содержимое выравнивается вертикально снизу и горизонтально по левому краю. 256
НизПраво (BottomRight) Содержимое выравнивается вертикально снизу и горизонтально по правому краю. 1024
НизЦентр (BottomCenter) Содержимое выравнивается вертикально снизу и горизонтально по центру. 512
СерединаЛево (MiddleLeft) Содержимое выравнивается вертикально по середине и горизонтально по левому краю. 16
СерединаПраво (MiddleRight) Содержимое выравнивается вертикально по середине и горизонтально по правому краю. 64
СерединаЦентр (MiddleCenter) Содержимое выравнивается вертикально по середине и горизонтально по центру. 32
