Картинка.Разблокировать (Bitmap.UnlockBits) Метод

Синтаксис

Разблокировать(<АтрибутыКартинки>)

Параметры

АтрибутыКартинки (обязательный)
Тип: АтрибутыКартинки (BitmapData).
Объект АтрибутыКартинки (BitmapData), задающий сведения об этой операции блокировки.

Возвращаемое значение

Описание

Разблокирует объект Картинка (Bitmap) из оперативной памяти.

Примечание

Пример

Картинка2.Разблокировать(АтрибутыКартинки1);

КонецЦикла;

	Картинка2.Разблокировать(АтрибутыКартинки1);
	СписокИзображений2.Изображения.Добавить(Картинка2);

КонецЦикла;
