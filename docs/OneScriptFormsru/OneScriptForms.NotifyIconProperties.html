ЗначокУведомления (NotifyIcon) Свойства

Свойства класса ЗначокУведомления (NotifyIcon) перечислены ниже. Полный список членов класса ЗначокУведомления (NotifyIcon) смотрите в разделе ЗначокУведомления (NotifyIcon) Члены.

Свойства

ДвойноеНажатие (DoubleClick) Возвращает или задает код для выполнения, когда пользователь дважды щелкает значок в области уведомлений на панели задач.
ЗаголовокВсплывающейПодсказки (BalloonTipTitle) Возвращает или задает заголовок всплывающей подсказки значка уведомления.
Значок (Icon) Возвращает или задает текущий значок.
ЗначокВсплывающейПодсказки (BalloonTipIcon) Возвращает или задает значок для всплывающей подсказки значка уведомления.
КонтекстноеМеню (ContextMenu) Возвращает контекстное меню для значка уведомления.
Нажатие (Click) Возвращает или задает код для выполнения, когда пользователь щелкает значок в области уведомлений.
Отображать (Visible) Возвращает или задает значение, определяющее видимость значка в области уведомлений на панели задач.
ПриНажатииКнопкиМыши (MouseDown) Возвращает или задает код для выполнения, когда пользователь нажимает кнопку мыши, а указатель мыши находится на значке в области уведомлений панели задач.
ПриОтпусканииМыши (MouseUp) Возвращает или задает код для выполнения, когда пользователь отпускает кнопку мыши, а указатель мыши находится на значке в области уведомлений панели задач.
ПриПеремещенииМыши (MouseMove) Возвращает или задает код для выполнения, когда пользователь перемещает мыши, в то время как указатель находится над значком в области уведомлений панели задач.
Текст (Text) Возвращает или задает текст подсказки, отображаемый при наведении указателя мыши на значок в области уведомлений.
ТекстВсплывающейПодсказки (BalloonTipText) Возвращает или задает текст, отображаемый во всплывающей подсказке значка уведомления.
Тип (Type) (унаследовано от Компонент (Component)) Возвращает тип текущего экземпляра.
