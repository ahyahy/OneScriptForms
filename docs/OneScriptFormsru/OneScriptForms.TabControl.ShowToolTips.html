ПанельВкладок.ПоказатьПодсказку (TabControl.ShowToolTips) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, отображается ли всплывающая подсказка вкладки при наведении мыши на вкладку.

Истина - всплывающие подсказки отображаются, Ложь - в противном случае. Значение по умолчанию: Ложь.

Примечание

Чтобы создать всплывающую подсказку для вкладки, задайте значение свойству ToolTipText.

Пример

ПанельВкладок1.ПоказатьПодсказку = Истина;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ПанельВкладок1 = Ф.ПанельВкладок();
ПанельВкладок1.Родитель = Форма1;
ПанельВкладок1.Стыковка = Ф.СтильСтыковки.Заполнение;
ПанельВкладок1.Ширина = 250;
Вкладки1 = ПанельВкладок1.Вкладки;

Вкладка1 = Вкладки1.Добавить(Ф.Вкладка("Вкладка1"));
Вкладка2 = Вкладки1.Добавить(Ф.Вкладка("Вкладка2"));
Вкладка3 = Вкладки1.Добавить(Ф.Вкладка("Вкладка3"));

ПанельВкладок1.ПоказатьПодсказку = Истина;

Вкладка1.ТекстПодсказки = "Это подсказка для Вкладка1";

Ф.ЗапуститьОбработкуСобытий();
