РежимАвтоРазмераКолонок (DataGridViewAutoSizeColumnsMode) Перечисление

Описание

Определяет значения для указания способа регулирования ширины колонок.

Члены

Имя члена Описание Значение
ВсеЯчейки (AllCells) Ширина колонок изменяется так, чтобы вместить содержимое всех ячеек колонок, включая ячейки заголовков. 6
ВсеЯчейкиБезЗаголовков (AllCellsExceptHeader) Ширина колонок изменяется так, чтобы вместить содержимое всех ячеек колонок, исключая ячейки заголовков. 4
ЗаголовокКолонки (ColumnHeader) Ширина колонок изменяется так, чтобы вместить содержимое ячеек заголовков колонок. 2
Заполнение (Fill) Ширина колонок подбирается таким образом, чтобы суммарная ширина всех колонок в точности заполняла отображаемую область элемента управления, а прокрутка по горизонтали требовалась только для того, чтобы не допускать уменьшения ширины колонок ниже значений свойства КолонкаТаблицы.МинимальнаяШирина (DataGridViewColumn.MinimumWidth). 16
ОтобразритьЯчейки (DisplayedCells) Ширина колонок изменяется так, чтобы вместить содержимое всех ячеек колонок, которые находятся в строках, отображающихся на экране в настоящий момент, включая ячейки заголовков. 10
ОтобразритьЯчейкиБезЗаголовков (DisplayedCellsExceptHeader) Ширина колонок изменяется так, чтобы вместить содержимое всех ячеек колонок, которые находятся в строках, отображающихся на экране в настоящий момент, исключая ячейки заголовков. 8
Отсутствие (None) Значения ширины колонок не изменяются автоматически. 1
