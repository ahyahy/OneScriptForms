ФормыДляОдноСкрипта.Предупреждение (OneScriptForms.DoMessageBox) Метод

Синтаксис

Предупреждение(<ТекстПредупреждения>, <Таймаут>, <Заголовок>)

Параметры

ТекстПредупреждения (обязательный)
Тип: Строка.
Таймаут (необязательный)
Тип: Число.
Заголовок (необязательный)
Тип: Строка.

Возвращаемое значение

Описание

Выводит на экран окно предупреждения.

Примечание

Важно: В системе отличной от семейства Windows данный метод неприменим.

Пример

Ф.Предупреждение("Текст предупреждения", 5, "Заголовок предупреждения");
