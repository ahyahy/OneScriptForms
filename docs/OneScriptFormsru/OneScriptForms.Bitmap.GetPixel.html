Картинка.ПолучитьПиксель (Bitmap.GetPixel) Метод

Синтаксис

ПолучитьПиксель(<X>, <Y>)

Параметры

X (обязательный)
Тип: Число.
Координата X пикселя.
Y (обязательный)
Тип: Число.
Координата Y пикселя.

Возвращаемое значение

Тип: Цвет (Color).

Экземпляр класса Цвет (Color), представляющий цвет пикселя по указанным координатам.

Описание

Возвращает цвет указанного пикселя в этом изображении.

Примечание

Пример

Сообщить("" + Картинка1.ПолучитьПиксель(10, 10).Имя);
