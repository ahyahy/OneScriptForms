КоллекцияСсылок (LinkCollection) Свойства

Свойства класса КоллекцияСсылок (LinkCollection) перечислены ниже. Полный список членов класса КоллекцияСсылок (LinkCollection) смотрите в разделе КоллекцияСсылок (LinkCollection) Члены.

Свойства

Количество (Count) Возвращает число ссылок в коллекции.
СсылкиДобавлены (LinksAdded) Возвращает значение, показывающее, были ли в коллекцию добавлены ссылки.
ТолькоЧтение (IsReadOnly) Возвращает значение, которое определяет, является ли коллекция доступной только для чтения.
