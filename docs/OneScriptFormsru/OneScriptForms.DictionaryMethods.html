Словарь (Dictionary) Методы

Методы класса Словарь (Dictionary) перечислены ниже. Полный список членов класса Словарь (Dictionary) смотрите в разделе Словарь (Dictionary) Члены.

Методы

Добавить (Add) Добавляет элемент с указанным ключом и значением в словарь.
Очистить (Clear) Удаляет все элементы из словаря.
СодержитЗначение (ContainsValue) Определяет, содержит ли словарь указанное значение.
СодержитКлюч (ContainsKey) Определяет, содержит ли словарь указанный ключ.
Удалить (Remove) Удаляет элемент с указанным ключом из словаря.
Установить (Set) Задает значение записи в словаре.
Элемент (Item) Возвращает значение, связанное с указанным ключом.
