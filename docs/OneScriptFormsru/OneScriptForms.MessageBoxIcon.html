ЗначокОкнаСообщений (MessageBoxIcon) Перечисление

Описание

Константы, определяющие значок для отображения на ОкноСообщений (MessageBox).

Члены

Имя члена Описание Значение
Вопрос (Question) Окно сообщения содержит символ, состоящий из вопросительного знака в кружке. 32
Восклицание (Exclamation) Окно сообщения содержит символ, состоящий из восклицательного знака в треугольнике с желтым фоном. 48
Звездочка (Asterisk) Окно сообщения содержит символ, состоящий из строчной буквы в кружке. 64
Информация (Information) Окно сообщения содержит символ, состоящий из строчной буквы в кружке. 64
Остановка (Stop) Окно сообщения содержит символ, состоящий из белого X в кружке с красным фоном. 16
Отсутствие (None) Окно сообщения не содержит символов. 0
Ошибка (Error) Окно сообщения содержит символ, состоящий из белого X в кружке с красным фоном. 16
Предупреждение (Warning) Окно сообщения содержит символ, состоящий из восклицательного знака в треугольнике с желтым фоном. 48
Рука (Hand) Окно сообщения содержит символ, состоящий из белого X в кружке с красным фоном. 16
