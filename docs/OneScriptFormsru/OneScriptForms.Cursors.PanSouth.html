Курсоры.КурсорЮ (Cursors.PanSouth) Свойство

Использование

Только чтение.

Значение

Тип: Курсор (Cursor).

Возвращает курсор, который появляется при выполнении операций с помощью колесика мыши и движения самой мыши при вертикальной прокрутке окна вниз.

Примечание

Пример

Форма1.Курсор = Ф.Курсоры().КурсорЮ;
