ПанельВкладок.РазмерЭлемента (TabControl.ItemSize) Свойство

Использование

Чтение и запись.

Значение

Тип: Размер (Size).

Возвращает или задает размер вкладок элемента управления.

По умолчанию автоматически размеры вкладок соответствуют значкам и меткам на вкладках.

Примечание

Чтобы изменить ширину вкладок с помощью свойства РазмерЭлемента, свойству ПанельВкладок.РежимМасштабирования (TabControl.SizeMode) необходимо присвоить значение Постоянный (Fixed).

Пример

ПанельВкладок1.РазмерЭлемента = Ф.Размер(35, 15);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ПанельВкладок1 = Ф.ПанельВкладок();
ПанельВкладок1.Родитель = Форма1;
ПанельВкладок1.Ширина = 250;
ПанельВкладок1.МногострочныйРежим = Истина;
ПанельВкладок1.РежимМасштабирования = Ф.РежимРазмераВкладок.Постоянный;
ПанельВкладок1.РазмерЭлемента = Ф.Размер(35, 15);

Сообщить("ПанельВкладок1.РазмерЭлемента = " + ПанельВкладок1.РазмерЭлемента.Ширина);
Сообщить("ПанельВкладок1.РазмерЭлемента = " + ПанельВкладок1.РазмерЭлемента.Высота);

Вкладки1 = ПанельВкладок1.Вкладки;
Для А = 1 По 6 Цикл
	Вкладки1.Добавить(Ф.Вкладка("Вкладка" + А));
КонецЦикла;

Ф.ЗапуститьОбработкуСобытий();
