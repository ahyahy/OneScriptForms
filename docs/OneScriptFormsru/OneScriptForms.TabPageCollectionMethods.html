Вкладки (TabPageCollection) Методы

Методы класса Вкладки (TabPageCollection) перечислены ниже. Полный список членов класса Вкладки (TabPageCollection) смотрите в разделе Вкладки (TabPageCollection) Члены .

Методы

Вставить (Insert) Создает новую страницу вкладки с заданным текстом и вставляет его в коллекцию по указанному индексу.
Добавить (Add) Создает страницу вкладки с заданным текстом и добавляет его в коллекцию.
Индекс (IndexOf) Возвращает индекс заданной вкладки в коллекции.
Очистить (Clear) Удаляет все страницы вкладок из коллекции.
Удалить (Remove) Удаляет объект Вкладка (TabPage) из коллекции.
УдалитьПоИндексу (RemoveAt) Удаляет страницу вкладки по указанному индексу из коллекции.
Элемент (Item) Устанавливает и/или возвращает объект Вкладка (TabPage) в коллекции.
