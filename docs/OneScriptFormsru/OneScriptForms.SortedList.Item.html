СортированныйСписок.Элемент (SortedList.Item) Метод

Синтаксис

Элемент(<Ключ>, <Элемент>)

Параметры

Ключ (обязательный)
Тип: Произвольный.
Ключ, связанный с получаемым значением.
Элемент (необязательный)
Тип: Произвольный.
Объект, представляющий устанавливаемое значение.

Возвращаемое значение

Тип: СловарнаяЗапись (DictionaryEntry).

Объект СловарнаяЗапись (DictionaryEntry), представляющий значение с помощью конкретного ключа.

Описание

Устанавливает и/или возвращает значение, связанное с указанным ключом в объекте СортированныйСписок (SortedList).

Примечание

Пример

СловарнаяЗапись1 = СортированныйСписок.Элемент(5.15);
СловарнаяЗапись1 = СортированныйСписок.Элемент(5.15, Дата(2020,05,05,05,05,05));
