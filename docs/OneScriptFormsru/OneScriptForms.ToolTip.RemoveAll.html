Подсказка.УдалитьВсе (ToolTip.RemoveAll) Метод

Синтаксис

УдалитьВсе()

Параметры

Возвращаемое значение

Описание

Удаляет весь текст всплывающей подсказки.

Примечание

Этот метод можно использовать, чтобы удалить весь текст всплывающей подсказки, связанный с компонентом Подсказка (ToolTip) для всех связанных элементов управления.

Чтобы отключить отображение текста, используйте вместо этого свойство Активна (Active).

Чтобы изменить текст для одного связанного элемента управления, используйте метод УстановитьПодсказку (SetToolTip).

Метод не применим к подсказкам, установленным для объектов с типом значения УзелДереваЗначений (TreeNodeAdv), или КолонкаДереваЗначений (TreeColumn), или ФлажокУзла (NodeCheckBox), или ЗначокУзла (NodeStateIcon), или ПолеВыбораУзла (NodeComboBox), или ЧисловоеПолеУзла (NodeDecimalTextBox), или РегуляторВверхВнизУзла (NodeNumericUpDown), или ПолеВводаУзла (NodeTextBox).

Пример

Подсказка1.УдалитьВсе();
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Кнопка1 = Форма1.ЭлементыУправления.Добавить(Ф.Кнопка());
Кнопка1.Размер = Ф.Размер(150, 25);
Кнопка1.Текст = "Кнопка1";
Кнопка1.Центр();

Подсказка1 = Ф.Подсказка();
Подсказка1.УстановитьПодсказку(Кнопка1, "Это подсказка для кнопки.");

Подсказка1.УдалитьВсе();

Ф.ЗапуститьОбработкуСобытий();
