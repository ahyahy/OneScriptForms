СписокЭлементов.Элементы (ListView.Items) Свойство

Использование

Только чтение.

Значение

Тип: ЭлементыСпискаЭлементов (ListViewItemCollection).

Возвращает коллекцию, содержащую все элементы элемента управления.

Примечание

Пример

Элементы = СписокЭлементов1.Элементы;
