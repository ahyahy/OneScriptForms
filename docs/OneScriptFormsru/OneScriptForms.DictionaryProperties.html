Свойства класса Словарь (Dictionary) перечислены ниже. Полный список членов класса Словарь (Dictionary) смотрите в разделе Словарь (Dictionary) Члены.

Свойства

Значения (Values) Возвращает МассивСписок (ArrayList), содержащий значения словаря в произвольном порядке.
Ключи (Keys) Возвращает МассивСписок (ArrayList), содержащий ключи словаря в произвольном порядке.
Количество (Count) Возвращает количество пар ключ-значение, содержащихся в словаре.
