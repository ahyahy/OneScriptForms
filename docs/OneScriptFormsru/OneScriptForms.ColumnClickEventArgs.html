КолонкаНажатиеАрг (ColumnClickEventArgs) Класс

Описание

Предоставляет данные для события КолонкаНажатие (ColumnClick).

Класс КолонкаНажатиеАрг (ColumnClickEventArgs) предоставляет отсчитываемый от нуля индекс в классе Колонки (ListViewColumnHeaderCollection) для щелкнутой колонки в элементе управления СписокЭлементов (ListView). Эту информацию можно использовать в обработчике событий КолонкаНажатие (ColumnClick) для определения щелкнутой колонки для выполнения задач по данным внутри колонки.

Список всех членов этого типа смотрите в разделе КолонкаНажатиеАрг (ColumnClickEventArgs) Члены.
