КольцевойИндикатор (CircularProgressBar) Конструктор

Синтаксис

КольцевойИндикатор()

Параметры

Описание

Создаёт новый экземпляр класса КольцевойИндикатор (CircularProgressBar).

Примечание

Для установки размера кольцевого индикатора достаточно установить одно из свойств: Высота (Height), Ширина (Width) или Размер (Size). Индикатор будет иметь равенство размеров по высоте и по ширине. В случае установки свойства Размер (Size) будет взято наибольшее значение из значений ширины и высоты.

Минимально допустимый размер прямоугольника кольцевого индикатора ограничивается стороной в 38 пикселей.

Пример

КольцевойИндикатор1 = Ф.КольцевойИндикатор();
