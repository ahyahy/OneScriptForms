Словарь.Удалить (Dictionary.Remove) Метод

Синтаксис

Удалить(<Ключ>)

Параметры

Ключ (обязательный)
Тип: Произвольный.
Ключ удаляемого элемента.

Возвращаемое значение

Описание

Удаляет элемент с указанным ключом из словаря.

Примечание

Пример

Словарь1.Удалить("Дата");
