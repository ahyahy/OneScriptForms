Кодировка (Encoding) Методы

Методы класса Кодировка (Encoding) перечислены ниже. Полный список членов класса Кодировка (Encoding) смотрите в разделе Кодировка (Encoding) Члены.

Методы

КоличествоБайтов (GetByteCount) Вычисляет количество байтов, необходимых для кодирования указанной строки.
ПолучитьБайты (GetBytes) Кодирует указанную строку в массив байтов.
ПолучитьКодировку (GetEncoding) Возвращает кодировку для указанной кодовой страницы.
ПолучитьСтроку (GetString) Возвращает строку, содержащую все декодированные байты из указанного массива байтов.
Преобразовать (Convert) Преобразует массив байтов из одной кодировки в другую.
