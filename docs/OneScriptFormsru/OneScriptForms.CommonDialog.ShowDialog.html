ОбщийДиалог.ПоказатьДиалог (CommonDialog.ShowDialog) Метод

Синтаксис

ПоказатьДиалог()

Параметры

Возвращаемое значение

Тип: РезультатДиалога (DialogResult).

Описание

Запускает общее диалоговое окно.

Примечание

Пример

Если ДиалогОткрытияФайла1.ПоказатьДиалог() = Ф.РезультатДиалога.ОК  Тогда
	Сообщить("Выбран файл: " + ДиалогОткрытияФайла1.ИмяФайла);
КонецЕсли;

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;

	ЗаписьТекста.Записать(Стр);    
	ЗаписьТекста.Закрыть();    
	Сообщить("Сохранён файл: " + ДиалогСохраненияФайла1.ИмяФайла);
