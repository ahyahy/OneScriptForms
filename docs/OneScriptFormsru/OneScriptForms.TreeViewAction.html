ДеревоДействие (TreeViewAction) Перечисление

Описание

Указывает действие, которое возникает при событии ПослеВыбора (AfterSelect) элемента управления Дерево (TreeView).

Члены

Имя члена Описание Значение
Неизвестно (Unknown) Действие, вызвавшее событие неизвестно. 0
ОтКлавиатуры (ByKeyboard) Событие вызвано нажатием клавиши. 1
ОтМыши (ByMouse) Событие вызвано операцией с мышью. 2
Развертывание (Expand) Вызвано событие развертывания узла дерева. 4
Свертывание (Collapse) Вызвано событие свертывания узла дерева. 3
