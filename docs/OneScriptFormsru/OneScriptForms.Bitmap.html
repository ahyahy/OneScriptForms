Картинка (Bitmap) Класс

Описание

Формирует точечный рисунок, который состоит из пиксельных данных для графического изображения и его атрибутов. Объект Картинка (Bitmap), это объект, используемый для работы с изображениями, определяемыми по пиксельным данным.

Список всех членов этого типа смотрите в разделе Картинка (Bitmap) Члены.
