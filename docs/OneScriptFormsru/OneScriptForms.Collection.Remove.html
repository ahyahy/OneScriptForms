Коллекция.Удалить (Collection.Remove) Метод

Синтаксис

Удалить(<Ключ>)

Параметры

Ключ (обязательный)
Тип: Строка.
Ключ, указанный при добавлении в коллекцию.

Возвращаемое значение

Описание

Удаляет указанный элемент из коллекции.

Примечание

Пример

Коллекция1.Удалить("Объект");
