Синтаксис

ГруппировщикТаблицы(<ГруппировкаТаблицы>)

Параметры

ГруппировкаТаблицы (обязательный)
Тип: ГруппировкаТаблицы (DataGridViewGrouper).
Компонент ГруппировкаТаблицы (DataGridViewGrouper), обеспечивающий группировку объекта Таблица (DataGridView).

Описание

Создаёт новый экземпляр класса ГруппировщикТаблицы (DataGridViewGrouperControl).

Примечание

Важно: В системе отличной от семейства Windows данный класс неприменим.

Пример

ГруппировщикТаблицы1 = Ф.ГруппировщикТаблицы(ГруппировкаТаблицы1);
