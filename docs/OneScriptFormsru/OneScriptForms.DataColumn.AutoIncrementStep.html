КолонкаДанных.ШагПриращения (DataColumn.AutoIncrementStep) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает шаг приращения для колонки, свойству которого АвтоПриращение (AutoIncrement) задано значение Истина.

Значение по умолчанию - 1.

Примечание

Пример

Колонка1.ШагПриращения = 5;
Сообщить("Колонка1.ШагПриращения = " + Колонка1.ШагПриращения); // Колонка1.ШагПриращения = 5
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ТаблицаДанных1 = Ф.ТаблицаДанных();
ТаблицаДанных1.ИмяТаблицы = "ТД1";
Колонки1 = ТаблицаДанных1.Колонки;

Колонка1 = Колонки1.Добавить(Ф.КолонкаДанных());
Колонка1.ТипДанных = Ф.ТипДанных.Число;
Колонка1.ИмяКолонки = "Инкремент";
Колонка1.АвтоПриращение = Истина;
Колонка1.НачальноеЧисло = 25;
Колонка1.ШагПриращения = 5;
Сообщить("Колонка1.ШагПриращения = " + Колонка1.ШагПриращения); // Колонка1.ШагПриращения = 5
	
Ф.ЗапуститьОбработкуСобытий();
