ПодэлементыСпискаЭлементов.Добавить (ListViewSubItemCollection.Add) Метод

Синтаксис

Добавить(<Элемент>)

Параметры

Элемент (обязательный)
Тип: Строка; ПодэлементСпискаЭлементов (ListViewSubItem).
Текст, отображаемый на подэлементе или объект ПодэлементСпискаЭлементов (ListViewSubItem), добавляемый в коллекцию.

Возвращаемое значение

Тип: ПодэлементСпискаЭлементов (ListViewSubItem).

Объект ПодэлементСпискаЭлементов (ListViewSubItem), который был добавлен в коллекцию.

Описание

Добавляет подэлемент к коллекции подэлементов.

Примечание

Пример

ПодэлементыРадуга = Радуга.Подэлементы;
ПодэлементыРадуга.Добавить(Ф.ПодэлементСпискаЭлементов("оранжевый"));
