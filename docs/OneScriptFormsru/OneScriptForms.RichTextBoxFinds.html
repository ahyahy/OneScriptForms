ФорматированноеПолеВводаПоиск (RichTextBoxFinds) Перечисление

Описание

Указывает способ выполнения полнотекстового поиска в элементе управления ФорматированноеПолеВвода (RichTextBox).

Члены

Имя члена Описание Значение
НеВыделять (NoHighlight) Найденный текст не будет выделен. 8
Отсутствие (None) Найти все экземпляры искомого текста, независимо от того целые это слова или нет. 0
ПоискВОбратномНаправлении (Reverse) Поиск начинается с конца и продолжается до начала документа. 16
ТолькоЦелыеСлова (WholeWord) Найти только экземпляры текста с целыми словами. 2
УчитыватьРегистр (MatchCase) Найти только экземпляры текста с точным соответствием регистра. 4
