КолонкаПолеВыбора.Элементы (DataGridViewComboBoxColumn.Items) Метод

Синтаксис

Элементы(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.
Индекс получаемого элемента в коллекции.

Возвращаемое значение

Тип: Произвольный.

Описание

Возвращает значение по указанному индексу.

Примечание

Пример

Сообщить("" + КолонкаПолеВыбора1.Элементы(1).Значение);

Форма1.Отображать = Истина;

Форма1.Активизировать();

Таблица1 = Ф.Таблица();

Таблица1.Колонки.Добавить(КолонкаПолеВыбора1);
