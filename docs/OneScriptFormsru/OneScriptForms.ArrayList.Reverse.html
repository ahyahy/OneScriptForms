МассивСписок.ОбратныйПорядок (ArrayList.Reverse) Метод

Синтаксис

ОбратныйПорядок()

Параметры

Возвращаемое значение

Описание

Изменяет порядок элементов во всем списке МассивСписок (ArrayList) на обратный.

Примечание

Пример

М3.ОбратныйПорядок();
