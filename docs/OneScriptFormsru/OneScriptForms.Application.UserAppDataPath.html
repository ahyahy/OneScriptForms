Приложение.ПутьДанныхПриложенияПользователя (Application.UserAppDataPath) Свойство

Использование

Только чтение.

Значение

Тип: Строка.

Возвращает путь для данных приложения перемещаемого пользователя.

Примечание

Если путь не существует, он создается в следующем формате:

Base Path \ CompanyName \ ProductName \ ProductVersion

Данные, сохраненные в этом пути являются частью перемещаемого профиля пользователя. Перемещающийся пользователь работает на нескольких компьютерах в сети. Профиль перемещающегося пользователя хранится на сервере сети и загружается в систему при входе пользователя в систему. Чтобы профиль пользователя считался перемещаемым, операционная система должна поддерживать перемещаемые профили и должна быть включена. Обычно базовый путь - C:\Documents and Settings\пользователя\Application Data.

Пример

Сообщить(Ф.Приложение().ПутьДанныхПриложенияПользователя);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;

Сообщить(Приложение1.ПутьДанныхПриложенияПользователя);
