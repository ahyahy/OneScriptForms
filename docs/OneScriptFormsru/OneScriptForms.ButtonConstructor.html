Описание

Используйте метод ФормыДляОдноСкрипта.Кнопка (OneScriptForms.Button) для создания нового экземпляра класса Кнопка (Button).

Кнопка1 = Форма1.ЭлементыУправления.Добавить(Ф.Кнопка());
