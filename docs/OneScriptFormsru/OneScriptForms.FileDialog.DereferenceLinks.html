ФайловыйДиалог.РазыменоватьСсылки (FileDialog.DereferenceLinks) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, будет ли диалоговое окно возвращать местоположение файла, на который ссылается ярлык, или вернёт местоположение ярлыка (.lnk).

Истина, если диалоговое окно возвращает расположение файла, на который ссылается контекст; в противном случае - Ложь. Значение по умолчанию - Истина.

Примечание

Установка свойства в значение Ложь ускорит работу приложения.

Пример

ДиалогОткрытияФайла1.РазыменоватьСсылки = Ложь;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ДиалогОткрытияФайла1 = Ф.ДиалогОткрытияФайла();
ДиалогОткрытияФайла1.РазыменоватьСсылки = Ложь;

Ф.ЗапуститьОбработкуСобытий();
