Разделитель.ПозицияРазделителя (Splitter.SplitPosition) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает расстояние между элементом управления Разделитель (Splitter) и краем контейнера, к которому присоединен данный элемент управления.

Примечание

Пример

Разделитель1.ПозицияРазделителя = 100;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Ширина = 300;
Форма1.Высота = 200;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Панель1 = Форма1.ЭлементыУправления.Добавить(Ф.Панель());//Панель1
Панель1.Стыковка = Ф.СтильСтыковки.Заполнение;
Панель1.Имя = "Панель1";

Разделитель1 = Ф.Разделитель();
Разделитель1.Родитель = Панель1;
Разделитель1.Стыковка = Ф.СтильСтыковки.Лево;

Панель2 = Панель1.ЭлементыУправления.Добавить(Ф.Панель());//Панель2
Панель2.СтильГраницы = Ф.СтильГраницы.Трехмерная;
Панель2.СтильГраницы = 2;
Панель2.СтильГраницы = Ф.СтильГраницы.Трехмерная;
Панель2.ЦветФона = Ф.Цвет().Белый;
Панель2.Стыковка = Ф.СтильСтыковки.Лево;
Надпись1 = Панель2.ЭлементыУправления.Добавить(Ф.Надпись());
Надпись1.Текст = "Панель2";
Надпись1.Стыковка = Ф.СтильСтыковки.Верх;

Панель3 = Панель1.ЭлементыУправления.Добавить(Ф.Панель());//Панель3
Панель3.СтильГраницы = Ф.СтильГраницы.Трехмерная;
Панель3.ЦветФона = Ф.Цвет().Белый;
Панель3.Стыковка = Ф.СтильСтыковки.Заполнение;
Надпись2 = Панель3.ЭлементыУправления.Добавить(Ф.Надпись());
Надпись2.Текст = "Панель3";
Надпись2.Стыковка = Ф.СтильСтыковки.Верх;
Надпись2.ВыравниваниеТекста = Ф.ВыравниваниеСодержимого.СерединаПраво;

Разделитель1.ПозицияРазделителя = 100;

Ф.ЗапуститьОбработкуСобытий();
