СортированныйСписок.ПолучитьПоИндексу (SortedList.GetByIndex) Метод

Синтаксис

ПолучитьПоИндексу(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.
Отсчитываемый от нуля индекс.

Возвращаемое значение

Тип: Произвольный.

Значение по указанному индексу в объекте СортированныйСписок (SortedList).

Описание

Возвращает значение по указанному индексу из объекта СортированныйСписок (SortedList).

Примечание

Пример

Сообщить("ПолучитьПоИндексу(0) = " + СортированныйСписок1.ПолучитьПоИндексу(0));

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

СортированныйСписок1 = Ф.СортированныйСписок();
