Меню (Menu) Методы

Методы класса Меню (Menu) перечислены ниже. Полный список членов класса Меню (Menu) смотрите в разделе Меню (Menu) Члены.

Методы

ПолучитьГлавноеМеню (GetMainMenu) Возвращает ГлавноеМеню (MainMenu), содержащее это меню.
СлитьМеню (MergeMenu) Объединяет объекты ЭлементМеню (MenuItem) одного меню с текущим меню.
ЭлементМеню (MenuItem) Возвращает ЭлементМеню (MenuItem) по указанному индексу в коллекции.
ЭлементыМеню (MenuItems) Извлекает ЭлементМеню (MenuItem) по указанному индексу в коллекции.
