ПолеВводаУзла.ПолучитьЗначение (NodeTextBox.GetValue) Метод

Синтаксис

ПолучитьЗначение(<УзелДереваЗначений>)

Параметры

УзелДереваЗначений (обязательный)
Тип: УзелДереваЗначений (TreeNodeAdv).
Узел дерева значений, для которого необходимо получить значение поля ввода.

Возвращаемое значение

Тип: Строка.

Описание

Возвращает значение поля ввода узла.

Примечание

Пример

Сообщить("" + ПолеВводаУзла1.ПолучитьЗначение(Нептун));
