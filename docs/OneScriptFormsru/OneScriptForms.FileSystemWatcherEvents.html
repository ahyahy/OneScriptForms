НаблюдательФайловойСистемы (FileSystemWatcher) События

События, подерживаемые библиотекой для класса НаблюдательФайловойСистемы (FileSystemWatcher), и ссылка на свойство задающее обработчик события, перечислены ниже. Полный список членов класса НаблюдательФайловойСистемы (FileSystemWatcher) смотрите в разделе НаблюдательФайловойСистемы (FileSystemWatcher) Члены.

События

ПриИзменении (Changed) Происходит при изменении файла или каталога по заданному пути.
ПриПереименовании (Renamed) Происходит при переименовании файла или каталога по заданному пути.
ПриСоздании (Created) Происходит при создании файла или каталога по заданному пути.
ПриУдалении (Deleted) Происходит при удалении файла или каталога по заданному пути.
