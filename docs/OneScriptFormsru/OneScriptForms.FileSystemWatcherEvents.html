НаблюдательФайловойСистемы (FileSystemWatcher) События

События, поддерживаемые библиотекой для класса НаблюдательФайловойСистемы (FileSystemWatcher), и ссылка на свойство задающее обработчик события, перечислены ниже. События

ПриИзменении (Changed) Происходит при изменении файла или каталога по заданному пути. Аргументы события представлены классом СобытиеФайловойСистемыАрг (FileSystemEventArgs).
ПриПереименовании (Renamed) Происходит при переименовании файла или каталога по заданному пути. Аргументы события представлены классом ПереименованиеАрг (RenamedEventArgs).
ПриСоздании (Created) Происходит при создании файла или каталога по заданному пути. Аргументы события представлены классом СобытиеФайловойСистемыАрг (FileSystemEventArgs).
ПриУдалении (Deleted) Происходит при удалении файла или каталога по заданному пути. Аргументы события представлены классом СобытиеФайловойСистемыАрг (FileSystemEventArgs).
