Обзор Заполнение (Padding)

Конструктор

Заполнение (Padding) Конструктор Создаёт новый экземпляр класса Заполнение (Padding).

Свойства

Вертикаль (Vertical) Возвращает объединенные внутренние поля для верхнего и нижнего краев.
Верх (Top) Возвращает или задает значение внутренних полей для верхнего края.
Все (All) Возвращает или задает значение для всех краев.
Горизонталь (Horizontal) Возвращает объединенные внутренние поля для правого и левого краев.
Лево (Left) Возвращает или задает значение внутренних полей для левого края.
Низ (Bottom) Возвращает или задает значение внутренних полей для нижнего края.
Право (Right) Возвращает или задает значение внутренних полей для правого края.
Размер (Size) Возвращает экземпляр класса Размер (Size), содержащий сведения о внутренних полях.
