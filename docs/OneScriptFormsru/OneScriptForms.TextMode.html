РежимТекста (TextMode) Перечисление

Описание

Указывает, как будет отображаться текст на кольцевом индикаторе.

Члены

Имя члена Описание Значение
Значение (Value) Текст в виде значения. 1
Отсутствие (None) Текст отсутствует. 0
Пользовательский (Custom) Текст заданный пользователем. 3
Проценты (Percentage) Текст в виде процентов. 2
