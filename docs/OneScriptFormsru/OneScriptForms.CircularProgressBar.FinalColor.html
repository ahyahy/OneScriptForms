КольцевойИндикатор.КонечныйЦвет (CircularProgressBar.FinalColor) Свойство

Использование

Чтение и запись.

Значение

Тип: Цвет (Color).

Возвращает или задает конечный цвет индикатора.

Примечание

Конечным считается цвет полосы индикатора при приращении по часовой стрелке в районе от 11 до 12 часов.

Пример

КольцевойИндикатор1.КонечныйЦвет = Ф.Цвет("Красный");
Полный пример кода 




Перем КольцевойИндикатор1;

Процедура Таймер1_ПриСрабатыванииТаймера() Экспорт
	Если КольцевойИндикатор1.Значение = -100 Тогда
		КольцевойИндикатор1.Значение = 100;
	КонецЕсли;
	КольцевойИндикатор1.Уменьшение(1);
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;

Форма1.Активизировать();
