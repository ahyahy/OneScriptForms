КоллекцияСсылок.Добавить (LinkCollection.Add) Метод

Синтаксис

Добавить(<Ссылка>)

Параметры

Ссылка (обязательный)
Тип: Ссылка (Link).
Объект Ссылка (Link), добавляемый в коллекцию.

Возвращаемое значение

Тип: Число.

Отсчитываемый от нуля индекс, по которому в коллекции находится добавленная ссылка.

Описание

Добавляет в коллекцию ссылку и связываемые с ней сведения.

Примечание

Пример

Индекс1 = КоллекцияСсылок1.Добавить(Ссылка1);
Полный пример кода 

Копировать     Выделить всё


ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

НадписьСсылка1 = Ф.НадписьСсылка();
Форма1.ЭлементыУправления.Добавить(НадписьСсылка1);
НадписьСсылка1.Текст = "Пример гиперссылок:" + Символы.ПС + "Первая ссылка - www.kixforms.org." + Символы.ПС + "Вторая ссылка - oscript.io.";
НадписьСсылка1.Границы = Ф.Прямоугольник(10, 10, 300, 100);
НадписьСсылка1.СтильГраницы = Ф.СтильГраницы.Одинарная;
КоллекцияСсылок1 = НадписьСсылка1.Ссылки;
Ссылка1 = Ф.Ссылка();
Ссылка1.Начало = 36;
Ссылка1.Длина = 16;
Ссылка2 = Ф.Ссылка();
Ссылка2.Начало = 70;
Ссылка2.Длина = 10;
Индекс1 = КоллекцияСсылок1.Добавить(Ссылка1);
Индекс2 = КоллекцияСсылок1.Добавить(Ссылка2);
	
Пока Ф.Продолжать Цикл
	Выполнить(Ф.ПолучитьСобытие());
КонецЦикла;
