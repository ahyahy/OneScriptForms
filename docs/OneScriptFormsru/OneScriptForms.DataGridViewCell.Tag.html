ЯчейкаТаблицы.Метка (DataGridViewCell.Tag) Свойство

Использование

Только чтение.

Значение

Тип: Произвольный.

Возвращает или задает объект, содержащий дополнительные данные о ячейке.

Значение по умолчанию - Неопределено.

Примечание

Пример

Коллекция1 = Ячейка.Метка;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Родитель = Форма1;
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;
Таблица1.АвтоНумерацияСтрок = Истина;
Таблица1.ДобавлятьСтроки = Ложь;
Таблица1.КоличествоКолонок = 2;
Таблица1.КоличествоСтрок = 3;

Ячейка = Таблица1.ТекущаяЯчейка;

Коллекция1 = Ячейка.Метка;
Коллекция1.Добавить(Форма1, "Объект");
Коллекция1.Добавить("Форма1", "Имя");
Коллекция1.Добавить(Истина, "Булево");
Коллекция1.Добавить(156.35, "Число");
Коллекция1.Добавить(Дата(1984,03,26,06,24,21), "Дата");

Сообщить("Объект = " + Ячейка.Метка.Элемент("Объект"));
Сообщить("Имя = " + Ячейка.Метка.Элемент("Имя"));
Сообщить("Булево = " + Ячейка.Метка.Элемент("Булево"));
Сообщить("Число = " + Ячейка.Метка.Элемент("Число") / 2);
Сообщить("Дата = " + Ячейка.Метка.Элемент("Дата"));
