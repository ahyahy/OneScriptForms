КолонкаТаблицы.РежимСортировки (DataGridViewColumn.SortMode) Свойство

Использование

Чтение и запись.

Значение

Тип: РежимСортировки (DataGridViewColumnSortMode).

Возвращает или задает режим сортировки для колонки.

Примечание

Режим сортировки для КолонкаПолеВвода (DataGridViewTextBoxColumn) по умолчанию - Автоматически (Automatic). Режим сортировки по умолчанию для других типов колонок - НеСортируемый (NotSortable).

Пример

КолонкаПолеВвода.РежимСортировки = Ф.РежимСортировки.Автоматически;
