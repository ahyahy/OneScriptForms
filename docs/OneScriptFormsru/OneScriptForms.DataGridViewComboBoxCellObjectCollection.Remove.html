ЭлементыПоляВыбораЯчейки.Удалить (DataGridViewComboBoxCellObjectCollection.Remove) Метод

Синтаксис

Удалить(<Значение>)

Параметры

Значение (обязательный)
Тип: ЭлементСписка (ListItem).
Объект, представляющий элемент, который удаляется из коллекции.

Возвращаемое значение

Описание

Удаляет указанный объект из коллекции.

Примечание

Пример

ЭлементыПоляВыбораЯчейки.Удалить(Чита);
