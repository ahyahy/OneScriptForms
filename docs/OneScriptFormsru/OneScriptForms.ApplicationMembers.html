Версия (Version) Возвращает объект Версия (Version), который описывает основные, второстепенные сборки и ревизии, связанные с этим приложением.
ВерсияПродукта (ProductVersion) Возвращает версию продукта для этого приложения.
ИмяПродукта (ProductName) Возвращает имя продукта для этого приложения.
ПутьДанныхПриложенияПользователя (UserAppDataPath) Возвращает путь для данных приложения перемещаемого пользователя.
