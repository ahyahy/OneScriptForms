ВыравниваниеВСпискеЭлементов (ListViewAlignment) Перечисление

Описание

Указывает, как элементы выравниваются в элементе управления СписокЭлементов (ListView).

Члены

Имя члена Описание Значение
Верх (Top) Выравнивание элементов по верхнему краю. 2
Лево (Left) Выравнивание элементов по левому краю. 1
ПоСетке (SnapToGrid) Элементы выравниваются по невидимой сетке в элементе управления. Когда пользователь перемещает элемент, он перемещается к ближайшей точке пересечения сетки. 5
ПоУмолчанию (Default) Когда пользователь перемещает элемент, он остается там, куда его положили. 0
