КольцевойИндикатор.Уменьшение (CircularProgressBar.Decrement) Метод

Синтаксис

Уменьшение(<Шаг>)

Параметры

Шаг (обязательный)
Тип: Число.

Возвращаемое значение

Описание

Задает величину уменьшения для кольцевого индикатора.

Примечание

Пример

КольцевойИндикатор1.Уменьшение(1);
