Обзор ЭлементыУзла (NodeControlsCollection)

Свойства

Количество (Count) Возвращает количество объектов в коллекции.

Методы

Вставить (Insert) Вставляет существующий элемент в коллекцию элементов по указанному индексу.
Добавить (Add) Добавляет элемент в коллекцию элементов.
Индекс (IndexOf) Возвращает отсчитываемый от нуля индекс первого вхождения, найденного элемента.
Очистить (Clear) Очищает коллекцию.
Содержит (Contains) Определяет, входит ли элемент в коллекцию.
Удалить (Remove) Удаляет элемент из коллекции.
УдалитьПоИндексу (RemoveAt) Удаляет элемент по указанному индексу из коллекции.
УстановитьЭлемент (SetControl) Устанавливает элемент по указанному индексу.
