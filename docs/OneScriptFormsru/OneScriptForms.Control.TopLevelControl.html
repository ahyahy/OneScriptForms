ЭлементУправления.ЭлементВерхнегоУровня (Control.TopLevelControl) Свойство

Использование

Только чтение.

Значение

Тип: ЭлементУправления (Control).

Возвращает родительский элемент управления, не имеющий другого родительского элемента управления Форма (Form). Как правило, им является внешний объект Форма (Form), в котором содержится элемент управления.

Примечание

Элемент управления верхнего уровня определяемый как родительский элемент управления не имеющий в качестве родителя другую Windows форму.

Пример

Сообщить("Панель3.ЭлементВерхнегоУровня.Текст = " + Панель3.ЭлементВерхнегоУровня.Текст);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Ширина = 600;
Форма1.Высота = 400;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Форма1.Текст = "Форма1";

Панель1 = Ф.Панель();
Панель1.Родитель = Форма1;
Панель1.Размер = Ф.Размер(300, 300);
Панель1.СтильГраницы = Ф.СтильГраницы.Трехмерная;
Панель1.ЦветФона = Ф.Цвет().Красный;
Панель1.Центр();

Панель2 = Ф.Панель();
Панель2.Родитель = Панель1;
Панель2.Размер = Ф.Размер(200, 200);
Панель2.СтильГраницы = Ф.СтильГраницы.Трехмерная;
Панель2.ЦветФона = Ф.Цвет().Синий;
Панель2.Центр();

Панель3 = Ф.Панель();
Панель3.Родитель = Панель2;
Панель3.Размер = Ф.Размер(100, 100);
Панель3.СтильГраницы = Ф.СтильГраницы.Трехмерная;
Панель3.ЦветФона = Ф.Цвет().Зеленый;
Панель3.Центр();

Сообщить("Панель3.ЭлементВерхнегоУровня.Текст = " + Панель3.ЭлементВерхнегоУровня.Текст);

Ф.ЗапуститьОбработкуСобытий();
