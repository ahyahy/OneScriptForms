ИнформацияЗапускаПроцесса (ProcessStartInfo) Свойства

Свойства класса ИнформацияЗапускаПроцесса (ProcessStartInfo) перечислены ниже. Полный список членов класса ИнформацияЗапускаПроцесса (ProcessStartInfo) смотрите в разделе ИнформацияЗапускаПроцесса (ProcessStartInfo) Члены .

Свойства

Аргументы (Arguments) Возвращает или задает набор аргументов командной строки, используемых при запуске приложения.
Домен (Domain) Возвращает или задает значение, позволяющее идентифицировать домен, используемый при запуске процесса.
ИмяПользователя (UserName) Возвращает или задает имя пользователя, которое следует использовать при запуске процесса.
ИмяФайла (FileName) Возвращает или задает приложение или документ для запуска.
ИспользоватьОболочку (UseShellExecute) Возвращает или задает значение, позволяющее определить, нужно ли использовать оболочку операционной системы для запуска процесса.
Пароль (Password) Возвращает или задает пароль пользователя, используемый при запуске процесса.
ПеренаправитьСтандартныйВывод (RedirectStandardOutput) Возвращает или задает значение, позволяющее определить, записываются ли текстовые выходные данные приложения в поток Процесс.СтандартныйВывод (Process.StandardOutput).
СоздатьБезОкна (CreateNoWindow) Возвращает или задает значение, позволяющее определить, нужно ли запускать процесс в новом окне.
СтильОкна (WindowStyle) Возвращает или задает состояние окна, используемое при запуске процесса.
