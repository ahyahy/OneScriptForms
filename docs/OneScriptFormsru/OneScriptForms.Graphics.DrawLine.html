Графика.РисоватьЛинию (Graphics.DrawLine) Метод

Синтаксис

РисоватьЛинию(<Перо>, <X1>, <Y1>, <X2>, <Y2>)

Параметры

Перо (обязательный)
Тип: Перо (Pen).
Объект Перо (Pen), который определяет цвет, толщину и стиль линии.
X1 (обязательный)
Тип: Число.
x координата первой точки.
Y1 (обязательный)
Тип: Число.
y координата первой точки.
X2 (обязательный)
Тип: Число.
x координата второй точки.
Y2 (обязательный)
Тип: Число.
y координата второй точки.

Возвращаемое значение

Описание

Проводит линию, соединяющую две точки, задаваемые парами координат.

Примечание

При создании объекта Графика (Graphics) с помощью метода СоздатьГрафику (CreateGraphics) какого либо элемента управления, код рисования графики необходимо поместить в процедуру, указанную в свойстве ПриПерерисовке (Paint) этого элемента управления.

Пример

Графика1 = Форма1.СоздатьГрафику();
Перо1 = Ф.Перо(Ф.Цвет().Красный, 5);
Графика1.РисоватьЛинию(Перо1, 20, 100, 120, 100);
Полный пример кода 




Перем Ф, Форма1;

Процедура Форма1_ПриПерерисовке() Экспорт
	Графика1 = Форма1.СоздатьГрафику();

	Графика1.РисоватьЛинию(Перо1, 20, 100, 120, 100);
