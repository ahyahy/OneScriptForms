IpАдрес (IpAddress) Свойства

Свойства класса IpАдрес (IpAddress) перечислены ниже. Полный список членов класса IpАдрес (IpAddress) смотрите в разделе IpАдрес (IpAddress) Члены.

Свойства

Замыкание (LoopBack) Возвращает IP-адрес замыкания на себя.
Любой (Any) Возвращает IP-адрес, указывающий, что сервер должен контролировать действия клиентов на всех сетевых интерфейсах.
Отсутствие (None) Возвращает IP-адрес, указывающий, что сетевой интерфейс не должен использоваться.
Широковещательный (BroadCast) Возвращает широковещательный IP-адрес.
