Цвет.Имя (Color.Name) Свойство

Использование

Только чтение.

Значение

Тип: Строка.

Возвращает имя цвета. Вернется имя цвета, если цвет был создан из имени. Для пользовательских цветов возвратятся значения RGB в виде форматированной строки.

Примечание

Возвращаемое значение зависит от языка синтаксиса. Если свойство запрашивается в англоязычном варианте, то имя вернется тоже в англоязычном варианте, и наоборот.

Пример

Цвет1 = Ф.Цвет("Синий");
Цвет2 = Ф.Цвет(148, 155, 15);
Сообщить("" + Цвет1.Имя);
Форма1.Текст = Цвет1.Имя;
Форма1.ЦветФона = Цвет1;
Сообщить("" + Цвет1.Name);
Сообщить("" + Цвет2.Имя);
