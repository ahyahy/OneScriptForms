СортированныйСписок.Удалить (SortedList.Remove) Метод

Синтаксис

Удалить(<Ключ>)

Параметры

Ключ (обязательный)
Тип: Произвольный.
Ключ удаляемого элемента.

Возвращаемое значение

Описание

Удаляет элемент с указанным ключом из СортированныйСписок (SortedList).

Примечание

Пример

СортированныйСписок1.Удалить(10.10);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

СортированныйСписок1 = Ф.СортированныйСписок();
СортированныйСписок1.Добавить(1, "Простой стиль");
СортированныйСписок1.Добавить(2, "Простой стиль2");
СортированныйСписок1.Добавить(3, Ложь);
СортированныйСписок1.Добавить(4, 18.245);
СортированныйСписок1.Добавить(10.10, 18.245);
СортированныйСписок1.Добавить(5, Дата(1984,03,26,06,24,21));
СортированныйСписок1.Добавить(6, Форма1);
	
СортированныйСписок1.Удалить(10.10);

МассивСписок1 = СортированныйСписок1.Ключи;
Для А = 0 По МассивСписок1.Количество - 1 Цикл
    Ключ1 = МассивСписок1.Элемент(А);
    Сообщить("" + СортированныйСписок1.Элемент(Ключ1).Ключ + " - " + СортированныйСписок1.Элемент(Ключ1).Значение);
КонецЦикла;
