Dns (Dns) Методы

Методы класса Dns (Dns) перечислены ниже. Полный список членов класса Dns (Dns) смотрите в разделе Dns (Dns) Члены.

Методы

ПолучитьИмяУзла (GetHostName) Возвращает имя узла локального компьютера.
ПолучитьУзелПоАдресу (GetHostByAddress) Возвращает сведения DNS для указанного IP-адреса.
ПолучитьУзелПоИмени (GetHostByName) Возвращает сведения DNS для указанного DNS-имени узла.
Разрешить (Resolve) Разрешает DNS-имя узла или IP-адрес в экземпляр IpУзел (IpHostEntry).
