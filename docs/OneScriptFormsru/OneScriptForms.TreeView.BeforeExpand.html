Дерево.ПередРазвертыванием (TreeView.BeforeExpand) Свойство

Использование

Чтение и запись.

Значение

Тип: Произвольный.

Возвращает или задает код для выполнения перед развертыванием узла дерева.

Примечание

Свойство может быть задано типом значения Действие из библиотеки односкрипта, или типом значения Действие (Action).

Процедура обработки события должна быть объявлена с ключевым словом Экспорт.

Пример

Дерево1.ПередРазвертыванием = Новый Действие(ЭтотОбъект, "Дерево1_ПередРазвертыванием");
