Дерево.ПередРазвертыванием (TreeView.BeforeExpand) Свойство

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код для выполнения перед развертыванием узла дерева.

Примечание

Пример

Дерево1.ПередРазвертыванием = Ф.Действие(ЭтотОбъект, "Дерево1_ПередРазвертыванием");
Полный пример кода 




Перем Ф;

Процедура Дерево1_ПередРазвертыванием() Экспорт
	АргументыСобытия1 = Ф.ДеревоОтменаАрг();
	Сообщить("АргументыСобытия1.Действие = " + АргументыСобытия1.Действие);
	Сообщить("АргументыСобытия1.Узел.Текст = " + АргументыСобытия1.Узел.Текст);
	
	Если АргументыСобытия1.Узел.Текст = "Узел1" Тогда
		АргументыСобытия1.Отмена = Истина;
	КонецЕсли;
