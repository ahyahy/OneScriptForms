СлияниеМеню (MenuMerge) Перечисление

Описание

Задает поведение ЭлементМеню (MenuItem) при слиянии с элементами в другом меню.

Члены

Имя члена Описание Значение
Добавить (Add) Объект ЭлементМеню (MenuItem) добавляется к коллекции объектов ЭлементМеню (MenuItem) в объединенном меню. 0
Заменить (Replace) Объект ЭлементМеню (MenuItem) заменяет существующий ЭлементМеню (MenuItem) в той же позиции в объединенном меню. 1
ОбъединитьМеню (MergeItems) Все элементы вложенного меню этого ЭлементМеню (MenuItem) объединяются с соответствующими элементами существующих объектов ЭлементМеню (MenuItem) в той же позиции в объединенном меню. 2
Удалить (Remove) Объект ЭлементМеню (MenuItem) удаляется из объединенного меню. 3
