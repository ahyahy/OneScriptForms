ФормыДляОдноСкрипта.[Перечисление].Name (OneScriptForms.[Перечисление].Name) Метод

Синтаксис

[Перечисление].Name(<Значение>)

Параметры

Значение (обязательный)
Тип: Число.

Возвращаемое значение

Тип: Строка.

Описание

Возвращает строковое представление значения перечисления на английском языке.

Примечание

Пример

Стр = "Перечисление СтилиПривязки через цикл Для Каждого...";
Для Каждого Эл Из Ф.СтилиПривязки Цикл
	Стр = Стр + "
	|    Значение = " + Эл + " Name = " + Ф.СтилиПривязки.Name(Эл);
КонецЦикла;
Сообщить(Стр);
