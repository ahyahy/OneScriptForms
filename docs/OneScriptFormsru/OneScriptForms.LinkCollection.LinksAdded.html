КоллекцияСсылок.СсылкиДобавлены (LinkCollection.LinksAdded) Свойство

Использование

Только чтение.

Значение

Тип: Булево.

Возвращает значение, показывающее, были ли в коллекцию добавлены ссылки.

Истина, если ссылки были добавлены в коллекцию; в противном случае — значение Ложь.

Примечание

Пример

Сообщить("КоллекцияСсылок1.СсылкиДобавлены = " + КоллекцияСсылок1.СсылкиДобавлены);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

НадписьСсылка1 = Ф.НадписьСсылка();
Форма1.ЭлементыУправления.Добавить(НадписьСсылка1);
НадписьСсылка1.Текст = "Это гиперссылка";
НадписьСсылка1.Границы = Ф.Прямоугольник(10, 10, 200, 100);
НадписьСсылка1.СтильГраницы = Ф.СтильГраницы.Одинарная;
КоллекцияСсылок1 = НадписьСсылка1.Ссылки;
Сообщить("КоллекцияСсылок1.СсылкиДобавлены = " + КоллекцияСсылок1.СсылкиДобавлены);
