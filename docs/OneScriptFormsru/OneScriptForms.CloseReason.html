ПричинаЗакрытия (CloseReason) Перечисление

Описание

Указывает причину, по которой форма была закрыта.

Члены

Имя члена Описание Значение
Владелец (FormOwnerClosing) Форма-владелец закрывается. 5
Выход (ApplicationExitCall) Был вызван метод Exit() класса Application. 6
Диспетчер (TaskManagerClosing) Диспетчер задач Microsoft Windows закрывает приложение. 4
Отсутствие (None) Причина закрытия не была определена или не может быть определена. 0
Пользователь (UserClosing) Форма закрывается программными способами или с помощью действия пользователя в пользовательском интерфейсе (например нажатием кнопки Закрыть в окне формы, выбором параметра Закрыть в системном меню окна или нажатием сочетания клавиш ALT+F4). 3
Система (WindowsShutDown) Операционная система закрывает все приложения перед завершением работы. 1
ФормаMDI (MdiFormClosing) Родительская форма этой формы многодокументного интерфейса (MDI) закрывается. 2
