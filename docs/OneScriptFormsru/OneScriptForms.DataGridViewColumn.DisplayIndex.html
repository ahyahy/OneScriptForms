КолонкаТаблицы.ИндексОтображения (DataGridViewColumn.DisplayIndex) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает расположение колонки относительно колонок, отображаемых в текущий момент.

Примечание

Индекс колонки в объекте КолонкиТаблицы (DataGridViewColumnCollection) остается неизменным. Меняется только местоположение в таблице при отображении на экране.

Важно: При запуске в Linux ИндексОтображения следует отсчитывать от второй на экране колонке считая слева на право. При этом отсчет ведётся также от нуля. Тоесть индекс установленный в ноль отобразит колонку на втором месте слева на право в таблице.

Пример

КолонкаПолеВвода1.ИндексОтображения = 0;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Ширина = 500;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Родитель = Форма1;
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;

Таблица1.КоличествоКолонок = 20;
Таблица1.КоличествоСтрок = 7;

Для А = 0 По Таблица1.Колонки.Количество - 1 Цикл
	Колонка = Таблица1.Колонки(А);
	Колонка.ТекстЗаголовка = "Кол" + А;
	Колонка.РежимСортировки = Ф.РежимСортировки.Программный;
КонецЦикла;
Для А = 0 По Таблица1.Строки.Количество - 1 Цикл
	Таблица1.Строки(А).ЗаголовокСтроки.Значение = "" + А;
КонецЦикла;

КолонкаПолеВвода1 = Таблица1.Колонки(2);
КолонкаПолеВвода1.ИндексОтображения = 0;

Ф.ЗапуститьОбработкуСобытий();
