ПолеВвода.ПринятиеВозврат (TextBox.AcceptsReturn) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Получение или установка значения, определяющего поведение элементов управления, предназначенных для редактирование текста, при нажатии клавиши ВВОД.

Истина - клавиша ENTER создает новую строку текста в многострочном элементе управления, Ложь - клавиша ENTER активирует кнопку по умолчанию для формы. Значение по умолчанию: Истина.

Примечание

Если значение этого свойства Ложь, пользователь должен нажать CTRL + ENTER, чтобы создать новую строку в многострочном элементе управления ПолеВвода (TextBox). Если кнопка по умолчанию отсутствует для формы, тогда клавиша ENTER всегда создаст новую строку текста в независимо от значения этого свойства.

Пример

ПолеВвода1.ПринятиеВозврат = Ложь;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Ширина = 750;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ПолеВвода1 = Форма1.ЭлементыУправления.Добавить(Ф.ПолеВвода());
ПолеВвода1.Текст = "";
ПолеВвода1.Ширина = 500;
ПолеВвода1.Высота = 200;
ПолеВвода1.МногострочныйРежим = Истина;
ПолеВвода1.Перенос = Ложь;
ПолеВвода1.ПолосыПрокрутки = Ф.ПолосыПрокрутки.Обе;

Кнопка1 = Форма1.ЭлементыУправления.Добавить(Ф.Кнопка());
Кнопка1.Правее(ПолеВвода1, 10);
Форма1.КнопкаПринять = Кнопка1;

ПолеВвода1.ПринятиеВозврат = Ложь;

Ф.ЗапуститьОбработкуСобытий();
