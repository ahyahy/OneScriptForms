ФормыДляОдноСкрипта (OneScriptForms) Свойства

Свойства класса ФормыДляОдноСкрипта (OneScriptForms) перечислены ниже. Полный список членов класса ФормыДляОдноСкрипта (OneScriptForms) смотрите в разделе ФормыДляОдноСкрипта (OneScriptForms) Члены.

Свойства

АргументыСобытия (EventArgs) Возвращает аргументы события, связанные с действием, которое вызвало последнее событие.
ВерсияПродукта (ProductVersion) Возвращает версию продукта, сопоставленную с данным приложением.
ИмяПродукта (ProductName) Возвращает имя продукта, связанное с этим приложением.
ИспользоватьГлавнуюФорму (UseMainForm) Дает возможность установки одной из форм главной формой.
КоллекцияФорм (FormsCollection) Возвращает коллекцию созданных в приложении форм.
Отправитель (Sender) Источник (объект) последнего события.
ПлатформаWin (WinPlatform) Возвращает значение, показывающее, используется ли операционная система семейства Windows.
Продолжать (GoOn) Возвращает значение, показывающее, следует ли продолжать цикл обработки событий.
РазрешитьСобытия (AllowEvents) Возвращает или задает значение, показывающее, разрешена ли обработка событий.
Сборка (Build) Возвращает номер текущей сборки.
ТекущаяРаскладка (CurrentLayout) Возвращает имя текущей раскладки клавиатуры в том виде, в каком она отображается в региональных параметрах операционной системы компьютера.
ТекущийЯзыкВвода (CurrentInputLanguage) Возвращает язык ввода для текущего потока.
УстановленныеЯзыкиВвода (InstalledInputLanguages) Возвращает строку со списком всех установленных языков ввода.
ЯзыкВводаПоУмолчанию (DefaultInputLanguage) Возвращает язык ввода, используемый в системе по умолчанию.
