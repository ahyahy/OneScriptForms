Прямоугольник.Низ (Rectangle.Bottom) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает координату по оси Y, являющуюся суммой значений свойств Прямоугольник.Игрек (Rectangle.Y) и Прямоугольник.Высота (Rectangle.Height) данного прямоугольника.

Примечание

Пример

Прямоугольник1 = Ф.Прямоугольник(20, 25, 200, 100);
Сообщить("Прямоугольник1.Низ = " + Прямоугольник1.Низ);
