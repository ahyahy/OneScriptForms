Таблица.ШиринаЗаголовковСтрок (DataGridView.RowHeadersWidth) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает ширину (в пикселях) колонки, содержащей заголовки строк.

Значение по умолчанию - 43.

Примечание

Значение этого свойства должно находится в диапазоне от 4 пикселей до 32768 пикселей.

Чтобы настроить ширину колонок в соответствии с содержимым ячеек заголовка строки, используйте метод Таблица.АвтоШиринаЗаголовковСтрок (DataGridView.AutoResizeRowHeadersWidth).

Пример

Таблица1.ШиринаЗаголовковСтрок = 100;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;
Таблица1.Родитель = Форма1;
Таблица1.КоличествоКолонок = 2;
Таблица1.КоличествоСтрок = 5;

Таблица1.ШиринаЗаголовковСтрок = 100;
Сообщить("" + Таблица1.ШиринаЗаголовковСтрок);

Таблица1.АвтоНумерацияСтрок = Истина;
Таблица1.Строки(0).Ячейки(0).Значение = "Ячейка1";
Таблица1.Колонки(0).ЗаголовокКолонки.Значение = "Колонка1";
Таблица1.Колонки(1).ЗаголовокКолонки.Значение = "Колонка2";
