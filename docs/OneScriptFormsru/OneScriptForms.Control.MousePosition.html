ЭлементУправления.ПозицияМыши (Control.MousePosition) Свойство

Использование

Только чтение.

Значение

Тип: Точка (Point).

Возвращает позицию курсора мыши в экранных координатах.

Примечание

Форма1.Текст = "X = " + Форма1.ПозицияМыши.Икс + " , Y = " + Форма1.ПозицияМыши.Игрек;
