ЯчейкаТаблицы.Размер (DataGridViewCell.Size) Свойство

Использование

Только чтение.

Значение

Тип: Размер (Size).

Возвращает размер ячейки.

Объект Размер (Size), для которого заданы значения высоты строки и ширины колонки, которым принадлежит ячейка.

Примечание

Если ячейка не связана с элементом управления Таблица (DataGridView), это свойство возвращает объект Размер (Size) с шириной и высотой -1.

Пример

Сообщить("" + Таблица1.Ячейка(0, 0).Размер.Высота);

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
