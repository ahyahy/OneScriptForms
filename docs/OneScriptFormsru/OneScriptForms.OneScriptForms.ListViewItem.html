ФормыДляОдноСкрипта.ЭлементСпискаЭлементов (OneScriptForms.ListViewItem) Метод

Синтаксис

ЭлементСпискаЭлементов(<Текст>, <ИндексИзображения>)

Параметры

Текст (необязательный)
Тип: Строка.
Текст, отображаемый для элемента. Значение не должно превышать 259 знаков.
ИндексИзображения (необязательный)
Тип: Число.
Начинающийся с нуля индекс изображения в объекте СписокИзображений (ImageList), связанном с элементом управления СписокЭлементов (ListView), содержащим данный элемент.

Возвращаемое значение

Тип: ЭлементСпискаЭлементов (ListViewItem).

Описание

Создаёт новый экземпляр класса ЭлементСпискаЭлементов (ListViewItem).

Примечание

Пример

ЭлементСпискаЭлементов1 = Ф.ЭлементСпискаЭлементов("Тетраэдр", 0);
