Графика.РисоватьПрямоугольник (Graphics.DrawRectangle) Метод

Синтаксис

РисоватьПрямоугольник(<Перо>, <X>, <Y>, <Ширина>, <Высота>)

Параметры

Перо (обязательный)
Тип: Перо (Pen).
Объект Перо (Pen), определяющий цвет, ширину и стиль контура прямоугольника.
X (обязательный)
Тип: Число.
x координата верхнего левого угла нарисованного прямоугольника.
Y (обязательный)
Тип: Число.
y координата верхнего левого угла нарисованного прямоугольника.
Ширина (обязательный)
Тип: Число.
Ширина нарисованного прямоугольника.
Высота (обязательный)
Тип: Число.
Высота нарисованного прямоугольника.

Возвращаемое значение

Описание

Рисует прямоугольник, заданный парой координат, а также высотой и шириной.

Примечание

При создании объекта Графика (Graphics) с помощью метода СоздатьГрафику (CreateGraphics) какого либо элемента управления, код рисования графики необходимо поместить в процедуру, указанную в свойстве ПриПерерисовке (Paint) этого элемента управления.

Пример

Графика1 = Форма1.СоздатьГрафику();
Цвет1 = Ф.Цвет().Зеленый;
Перо1 = Ф.Перо(Цвет1);
Графика1.РисоватьПрямоугольник(Перо1, 20, 20, 100, 100);
