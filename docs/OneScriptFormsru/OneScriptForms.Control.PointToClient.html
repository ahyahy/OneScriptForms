ЭлементУправления.ТочкаНаКлиенте (Control.PointToClient) Метод

Синтаксис

ТочкаНаКлиенте(<ТочкаНаЭкране>)

Параметры

ТочкаНаЭкране (обязательный)
Тип: Точка (Point).
Координата экрана Точка (Point) для преобразования.

Возвращаемое значение

Тип: Точка (Point).

Описание

Вычисляет местоположение указанной экранной точки в клиентские координаты.

Примечание

Форма не должна находиться в свернутом состоянии.

Пример

Точка = Форма1.ТочкаНаКлиенте(Ф.Точка(90, 90));
