КолонкиДанных.Элемент (DataColumnCollection.Item) Метод

Синтаксис

Элемент(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Строка; Число.
Если число, отсчитываемый от нуля индекс возвращаемой колонки. Если строка, тогда Индекс это ИмяКолонки (ColumnName) возвращаемой колонки.

Возвращаемое значение

Тип: КолонкаДанных (DataColumn).

Колонка данных в коллекции по указанному индексу или с указанным именем колонки; в противном случае значение Неопределено, если колонка не существует.

Описание

Возвращает КолонкаДанных (DataColumn) из коллекции по указанному индексу.

Примечание

Пример

Сообщить("Колонки1.Элемент(1).ИмяКолонки = " + Колонки1.Элемент(1).ИмяКолонки); // Колонки1.Элемент(1).ИмяКолонки = ФИО
Сообщить("Колонки1.Элемент(""ФИО"").ИмяКолонки = " + Колонки1.Элемент("ФИО").ИмяКолонки); // Колонки1.Элемент("ФИО").ИмяКолонки = ФИО
