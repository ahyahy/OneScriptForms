Синтаксис

Значок(<ИмяФайла>)

Значок(<ИмяФайла>, <НомерЗначка>)

Значок(<ЗакодированнаяСтрока>)

Значок(<ТочечныйРисунок>)

Параметры

ИмяФайла
Тип: Строка.
Полное имя файла из которого будет создан значок.
ЗакодированнаяСтрока
Тип: Строка.
Строка, состоящая из символов в кодировке Base64.
ТочечныйРисунок
Тип: Картинка (Bitmap).
Растровый рисунок, который будет преобразован в эквивалентный значок.
НомерЗначка
Тип: Число.
Номер значка в файле или библиотеке.

Возвращаемое значение

Тип: Значок (Icon).

Описание

Создаёт новый экземпляр класса Значок (Icon).

Примечание

Пример

Значок1 = Ф.Значок(ИмяВременногоФайла2);

Значок3 = Ф.Значок(Рисунок1);
