Словарь.Ключи (Dictionary.Keys) Свойство

Использование

Только чтение.

Значение

Тип: МассивСписок (ArrayList).

Возвращает МассивСписок (ArrayList), содержащий ключи словаря в произвольном порядке.

Примечание

Порядок ключей в МассивСписок (ArrayList) не задан, но он совпадает с порядком значений в МассивСписок (ArrayList), возвращаемых методом Словарь.Значения (Dictionary.Values). Изменения в Словарь (Dictionary) не отражаются в МассивСписок (ArrayList).

Пример

МассивСписок1 = Словарь1.Ключи;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Словарь1 = Ф.Словарь();
Словарь1.Добавить("Строка", "Простой стиль");
Словарь1.Добавить("Булево", Ложь);
Словарь1.Добавить("Число", 18.245);
Словарь1.Добавить(10.10, 18.245);
Словарь1.Добавить("Дата", Дата(1984,03,26,06,24,21));
Словарь1.Добавить("Объект", Форма1);

МассивСписок1 = Словарь1.Ключи;
Для А = 0 По МассивСписок1.Количество - 1 Цикл
    Ключ1 = МассивСписок1.Элемент(А);
    Сообщить("" + А + " Ключ = " + Ключ1);
КонецЦикла;

Ф.ЗапуститьОбработкуСобытий();
