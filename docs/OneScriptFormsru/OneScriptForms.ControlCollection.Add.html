ЭлементыУправления.Добавить (ControlCollection.Add) Метод

Синтаксис

Добавить(<ЭлементУправления>)

Параметры

ЭлементУправления (обязательный)
Тип: Произвольный.
Добавляемый в коллекцию элемент управления.

Возвращаемое значение

Тип: Произвольный.

Элемент управления, который был добавлен в коллекцию.

Описание

Добавляет элемент управления в коллекцию.

Примечание

Пример

Кнопка1 = Форма1.ЭлементыУправления.Добавить(Ф.Кнопка());
