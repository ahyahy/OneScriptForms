МассивСписок.Добавить (ArrayList.Add) Метод

Синтаксис

Добавить(<Значение>)

Параметры

Значение (обязательный)
Тип: Произвольный.
Значение элемента, который будет добавлен в конец МассивСписок (ArrayList).

Возвращаемое значение

Тип: Произвольный.

Значение элемента, который был добавлен в конец МассивСписок (ArrayList).

Описание

Добавляет элемент в конец МассивСписок (ArrayList).

М3 = Ф.МассивСписок();
М3.Добавить("Строка в массив");
