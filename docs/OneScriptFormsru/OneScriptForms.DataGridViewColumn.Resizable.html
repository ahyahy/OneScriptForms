КолонкаТаблицы.ИзменяемыйРазмер (DataGridViewColumn.Resizable) Свойство

Использование

Чтение и запись.

Значение

Тип: ТриСостояния (DataGridViewTriState).

Возвращает или задает значение, указывающее, возможно ли изменение размера колонки.

Значение по умолчанию - Истина (True).

Примечание

Установка свойства в значение НеУстановлено (NotSet) восстанавливает наследование для этого свойства и оно автоматически устанавливается в значение Истина (True). Поэтому свойство никогда не вернет значение НеУстановлено (NotSet).

Пример

КолонкаПолеВвода1.ИзменяемыйРазмер = Ф.ТриСостояния.Ложь;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Ширина = 500;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Родитель = Форма1;
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;

Таблица1.КоличествоКолонок = 20;
Таблица1.КоличествоСтрок = 7;

Для А = 0 По Таблица1.Колонки.Количество - 1 Цикл
	Колонка = Таблица1.Колонки(А);
	Колонка.ТекстЗаголовка = "Кол" + А;
	Колонка.РежимСортировки = Ф.РежимСортировки.Программный;
КонецЦикла;
Для А = 0 По Таблица1.Строки.Количество - 1 Цикл
	Таблица1.Строки(А).ЗаголовокСтроки.Значение = "" + А;
КонецЦикла;

КолонкаПолеВвода1 = Таблица1.Колонки(2);

КолонкаПолеВвода1.ИзменяемыйРазмер = Ф.ТриСостояния.Ложь;

Ф.ЗапуститьОбработкуСобытий();
