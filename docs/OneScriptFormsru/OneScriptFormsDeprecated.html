Устаревшие свойства, методы

В целях совместимости устаревшие свойства и методы будут работать как и прежде. Рекомендуется использовать новый синтаксис.

В других особых случаях будут даны разъяснения.


Устаревшее Рекомендуемое
Графика.РисоватьЭлипс (Graphics.DrawEllipse)
В русском варианте имени была неточность. Вместо него используйте метод Графика.РисоватьЭллипс (Graphics.DrawEllipse). 		Графика.РисоватьЭллипс (Graphics.DrawEllipse)
Дерево.ВыбиратьПодэлементы (TreeView.FullRowSelect)
Русский вариант имени не точно отражал назначение свойства. Вместо него используйте свойство Дерево.ПолныйВыборСтроки (TreeView.FullRowSelect). 		Дерево.ПолныйВыборСтроки (TreeView.FullRowSelect)
СписокЭлементов.ВыбиратьПодэлементы (ListView.FullRowSelect)
Русский вариант имени не точно отражал назначение свойства. Вместо него используйте свойство СписокЭлементов.ПолныйВыборСтроки (ListView.FullRowSelect). 		СписокЭлементов.ПолныйВыборСтроки (ListView.FullRowSelect)
Установка обработчика события
Начиная с версии OneScript 1.8.3 обработчик события библиотеки OneScriptForms нельзя устанавливать используя класс Действие (Action) из библиотеки OneScript.
Так не сработает: 

Форма1.Нажатие = Новый Действие(ЭтотОбъект, "Форма1_Нажатие");
Начиная с версии OneScriptForms 2.5.0.0 обработчик события библиотеки OneScriptForms нужно устанавливать используя только класс Действие (Action) из библиотеки OneScriptForms.
Используйте: 

Форма1.Нажатие = Ф.Действие(ЭтотОбъект, "Форма1_Нажатие");
В случае запуска библиотеки OneScriptForms на версии OneScript ниже версии 1.8.3 этой рекомендации можно не придерживаться.
