КнопкаЯчейки.ИспользоватьТекстКакЗначение (DataGridViewButtonCell.UseColumnTextForButtonValue) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, будет ли появляться значение свойства Текст (Text) колонки, в которой содержится ячейка, на кнопке, отображаемой посредством ячейки.

Значение Истина, если значение свойства Значение (Value) будет автоматически соответствовать значению свойства Текст (Text) колонки, в которой содержится ячейка; в противном случае - значение Ложь. Значение по умолчанию - Ложь.

Примечание

Пример

Таблица1.Ячейка(0, 0).ИспользоватьТекстКакЗначение = Истина;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Родитель = Форма1;
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;
Таблица1.АвтоНумерацияСтрок = Истина;
Таблица1.ДобавлятьСтроки = Ложь;

КолонкаКнопка1 = Ф.КолонкаКнопка();
КолонкаКнопка1.ТекстЗаголовка = "Кол0";
Таблица1.Колонки.Добавить(КолонкаКнопка1);
Таблица1.Колонки.Добавить(Ф.КолонкаКнопка());
Таблица1.КоличествоСтрок = 2;

КолонкаКнопка1.Текст = "Кнопа";
Таблица1.Ячейка(0, 0).ИспользоватьТекстКакЗначение = Истина;
