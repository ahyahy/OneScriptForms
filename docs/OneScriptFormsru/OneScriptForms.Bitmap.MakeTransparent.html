Картинка.СделатьПрозрачным (Bitmap.MakeTransparent) Метод

Синтаксис

СделатьПрозрачным(<Цвет>)

Параметры

Цвет (обязательный)
Тип: Цвет (Color).

Возвращаемое значение

Описание

Делает указанный цвет прозрачным для этого объекта Картинка (Bitmap).

Примечание

Пример

Картинка2.СделатьПрозрачным(Ф.Цвет().Красный);
