УзелДерева.НачатьРедактирование (TreeNode.BeginEdit) Метод

Синтаксис

НачатьРедактирование()

Параметры

Возвращаемое значение

Описание

Инициирует редактирование надписи узла дерева.

Примечание

Этот метод работает только в том случае, если свойство Дерево.РедактированиеНадписи (TreeView.LabelEdit) объекта Дерево (TreeView) имеет значение Истина. Если свойство Дерево.РедактированиеНадписи (TreeView.LabelEdit) имеет значение Ложь, создается исключение, и узел дерева не будет помещен в редактируемое состояние.

Пример

Узел2.НачатьРедактирование();
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Дерево1 = Форма1.ЭлементыУправления.Добавить(Ф.Дерево());
Дерево1.Ширина = 200;

Дерево1.РедактированиеНадписи = Истина;

Узел1 = Дерево1.Узлы.Добавить("Узел1");
Узел2 = Дерево1.Узлы.Добавить("Узел2");
Узлы1 = Узел1.Узлы;
ПодУзел1 = Узлы1.Добавить("ПодУзел1");
Узлы2 = ПодУзел1.Узлы;
ПодУзел2 = Узлы2.Добавить("ПодУзел2");

Узел2.ШрифтУзла = Ф.Шрифт(, 8, Ф.СтильШрифта.Подчеркнутый + Ф.СтильШрифта.Курсив);
Узел2.НачатьРедактирование();
