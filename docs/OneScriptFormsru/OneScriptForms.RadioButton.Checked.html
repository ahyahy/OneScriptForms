Переключатель.Помечен (RadioButton.Checked) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, выбран ли данный элемент управления.

Примечание

Пример

Переключатель1.Помечен = Истина;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Переключатель1 = Ф.Переключатель();
Форма1.ЭлементыУправления.Добавить(Переключатель1);
Переключатель1.Положение = Ф.Точка(20, 20);
Переключатель1.Текст = "Переключатель1";
Переключатель1.Ширина = 350;
Переключатель1.Помечен = Истина;

Переключатель2 = Форма1.ЭлементыУправления.Добавить(Ф.Переключатель());
Переключатель2.Ниже(Переключатель1, 10);
Переключатель2.Ширина = Переключатель1.Ширина;
Переключатель2.Текст = "Переключатель2";
