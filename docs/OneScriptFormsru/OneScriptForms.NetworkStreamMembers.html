Обзор ПотокСети (NetworkStream)

Свойства

ВозможностьЗаписи (CanWrite) Возвращает значение, указывающее, поддерживает ли объект запись.
ВозможностьЧтения (CanRead) Возвращает значение, указывающее, поддерживает ли объект чтение.
ДанныеДоступны (DataAvailable) Возвращает значение, указывающее, имеются ли в объекте данные, доступные для чтения.

Методы

Закрыть (Close) Закрывает текущий поток и отключает все ресурсы (например, сокеты и файловые дескрипторы), связанные с текущим потоком.
Записать (Write) Записывает данные в ПотокСети (NetworkStream) из указанного диапазона массива байтов.
Прочитать (Read) Считывает данные из потока сети и сохраняет их в массив байтов.
ЧитатьБайт (ReadByte) Считывает байт из потока и перемещает позицию в потоке на один байт или возвращает -1, если достигнут конец потока.
