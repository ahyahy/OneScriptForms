Обзор ЭлементыМеню (MenuItemCollection)

Свойства

Количество (Count) (унаследовано от КоллекцияОснова (CollectionBase)) Возвращает общее количество объектов в коллекции.

Методы

Добавить (Add) Добавляет ранее созданный ЭлементМеню (MenuItem) в конец текущего меню.
Индекс (IndexOf) Извлекает индекс заданного элемента в коллекции.
Очистить (Clear) (унаследовано от КоллекцияОснова (CollectionBase)) Удаляет все объекты из коллекции.
Содержит (Contains) Определяет, является ли указанный ЭлементМеню (MenuItem) членом коллекции.
Удалить (Remove) Удаляет элемент ЭлементМеню (MenuItem) из коллекции.
УдалитьПоИндексу (RemoveAt) (унаследовано от КоллекцияОснова (CollectionBase)) Удаляет объект по указанному индексу из коллекции.
Элемент (Item) Извлекает ЭлементМеню (MenuItem) по указанному индексу в коллекции.
