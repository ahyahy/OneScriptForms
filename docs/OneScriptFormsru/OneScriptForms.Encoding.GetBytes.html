Кодировка.ПолучитьБайты (Encoding.GetBytes) Метод

Синтаксис

ПолучитьБайты(<КодируемаяСтрока>)

Параметры

КодируемаяСтрока (обязательный)
Тип: Строка.
Строка для кодирования.

Возвращаемое значение

Тип: МассивСписок (ArrayList).

Объект МассивСписок (ArrayList), содержащий кодированное представление параметра Строка.

Описание

Кодирует указанную строку в массив байтов.

Примечание

Байтовые значения переводятся в целые числа.

Пример

МассивСписок1 = Кодировка1.ПолучитьБайты("Откуда мне знать?");
