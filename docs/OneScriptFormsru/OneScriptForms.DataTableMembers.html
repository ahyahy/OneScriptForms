Обзор ТаблицаДанных (DataTable)

Конструктор

ТаблицаДанных (DataTable) Конструктор Создаёт новый экземпляр класса ТаблицаДанных (DataTable).

Свойства

ИмяТаблицы (TableName) Возвращает или задает имя для объекта ТаблицаДанных (DataTable).
Колонки (Columns) Возвращает коллекцию колонок, принадлежащих этой таблице.
НаборДанных (DataSet) Возвращает объект НаборДанных (DataSet), которому эта таблица принадлежит.
ПредставлениеПоУмолчанию (DefaultView) Возвращает индивидуальный вид таблицы, которая может включать отфильтрованный вид или положение курсора.
Строки (Rows) Возвращает коллекцию строк, относящихся к этой таблице.

Методы

Выбрать (Select) Возвращает коллекцию всех объектов СтрокаДанных (DataRow), отвечающих условиям фильтра.
ВыгрузитьКолонку (UnloadColumn) Создает МассивСписок (ArrayList) и копирует в него значения, содержащиеся в колонке таблицы значений.
ЗагрузитьКолонку (LoadColumn) Загружает колонку таблицы данных из МассивСписок (ArrayList).
Клонировать (Clone) Клонирует структуру ТаблицаДанных (DataTable), включая все схемы и ограничения ТаблицаДанных (DataTable).
Колонка (Column) Колонка объекта ТаблицаДанных (DataTable).
Колонки (Columns) Возвращает значение по указанному индексу.
Копировать (Copy) Копирует структуру и данные для этого объекта ТаблицаДанных (DataTable).
НоваяСтрока (NewRow) Создает новый объект СтрокаДанных (DataRow) с той же схемой, что и таблица.
ОтказИзменений (RejectChanges) Откат всех изменений, которые были внесены в таблицу после её загрузки или со времени последнего вызова метода ТаблицаДанных.ПринятьИзменения (DataTable.AcceptChanges).
ПринятьИзменения (AcceptChanges) Выполняет все изменения, внесенные в эту таблицу с момента последнего вызова метода ТаблицаДанных.ПринятьИзменения (DataTable.AcceptChanges).
Строки (Rows) Возвращает значение по указанному индексу.
