Колонки.Удалить (ListViewColumnHeaderCollection.Remove) Метод

Синтаксис

Удалить(<Колонка>)

Параметры

Колонка (обязательный)
Тип: Колонка (ColumnHeader).
Удаляемый из коллекции объект Колонка (ColumnHeader).

Возвращаемое значение

Описание

Удаляет указанную колонку из коллекции.

Примечание

Пример

Колонки1.Удалить(Колонка1);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Ширина = 630;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

СписокЭлементов1 = Форма1.ЭлементыУправления.Добавить(Ф.СписокЭлементов());
СписокЭлементов1.Стыковка = Ф.СтильСтыковки.Заполнение;
СписокЭлементов1.РежимОтображения = Ф.РежимОтображения.Подробно;

Колонки1 = СписокЭлементов1.Колонки;
Колонка1 = Колонки1.Добавить(Ф.Колонка("Имя цвета", 250, Ф.ГоризонтальноеВыравнивание.Центр));
Колонка2 = Колонки1.Добавить(Ф.Колонка("Номер", 70, Ф.ГоризонтальноеВыравнивание.Лево));

Сообщить("Количество = " + Колонки1.Количество);
