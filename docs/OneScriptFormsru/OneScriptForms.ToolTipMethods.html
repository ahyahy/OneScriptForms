Подсказка (ToolTip) Методы

Методы класса ТекстПодсказки (ToolTipText) перечислены ниже. Полный список членов класса ТекстПодсказки (ToolTipText) смотрите в разделе Подсказка (ToolTip) Члены.

Методы

ПолучитьПодсказку (GetToolTip) Извлекает текст всплывающей подсказки, связанный с указанным элементом управления.
УдалитьВсе (RemoveAll) Удаляет весь текст всплывающей подсказки.
УстановитьПодсказку (SetToolTip) Связывает текст всплывающей подсказки с указанным элементом управления.
