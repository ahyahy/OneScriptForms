ПолеКалендаря.Текст (DateTimePicker.Text) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает текст, связанный с этим элементом управления.

Примечание

Строка, возвращаемая этим свойством, эквивалентна свойству Значение (Value), отформатированному согласно установкам операционной системы, или согласно примененному свойству ПользовательскийФормат (DateTimePicker.CustomFormat).

Важно: В системе отличной от семейства Windows данное свойство недоступно дла записи.

Пример

ПолеКалендаря1.Текст = Строка(Дата(1930,01,02,03,04,05));
Сообщить("Текст = " + ПолеКалендаря1.Текст); // Текст = 2 января 1930 г. 3:04:05

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");

Сообщить("Текст = " + ПолеКалендаря1.Текст); // Текст = 07.03.1998 7:11

ПолеКалендаря1.Формат = Ф.ФорматПоляКалендаря.Время;

ПолеКалендаря1.Формат = Ф.ФорматПоляКалендаря.Длинный;

ПолеКалендаря1.Текст = Строка(Дата(1930,01,02,03,04,05));
Сообщить("Текст = " + ПолеКалендаря1.Текст); // Текст = 2 января 1930 г. 3:04:05
