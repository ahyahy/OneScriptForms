Прямоугольник.Лево (Rectangle.Left) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает координату по оси X левого края прямоугольника.

Примечание

Пример

Прямоугольник1 = Ф.Прямоугольник(20, 25, 200, 100);
Сообщить("Прямоугольник1.Лево = " + Прямоугольник1.Лево);
