ВыбранныеЭлементыСпискаЭлементов.Содержит (ListViewSelectedItemCollection.Contains) Метод

Синтаксис

Содержит(<Элемент>)

Параметры

Элемент (обязательный)
Тип: ЭлементСпискаЭлементов (ListViewItem).
Объект ЭлементСпискаЭлементов (ListViewItem), представляющий элемент для поиска в коллекции.

Возвращаемое значение

Тип: Булево.

Истина - указанный элемент находится в коллекции, Ложь - в противном случае.

Описание

Определяет, находится ли указанный элемент в коллекции.

Примечание

Пример

Сообщить("Выбранные.Содержит(Элемент2) = " + Выбранные.Содержит(Элемент2)); // Выбранные.Содержит(Элемент2) = Да
