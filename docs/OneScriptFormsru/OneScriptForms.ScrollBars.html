ПолосыПрокрутки (ScrollBars) Перечисление

Описание

Указывает, какие полосы прокрутки будут видны в элементе управления.

Члены

Имя члена Описание Значение
Вертикальная (Vertical) Отображаются только вертикальные полосы прокрутки. 2
Горизонтальная (Horizontal) Отображаются только горизонтальные полосы прокрутки. 1
Обе (Both) Отображаются горизонтальная и вертикальная полосы прокрутки. 3
Отсутствие (None) Полосы прокрутки не отображаются. 0
