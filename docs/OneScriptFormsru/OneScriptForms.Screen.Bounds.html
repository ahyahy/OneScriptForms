Экран.Границы (Screen.Bounds) Свойство

Использование

Только чтение.

Значение

Тип: Прямоугольник (Rectangle).

Возвращает границы области отображения.

Примечание

Пример

Экран1 = Ф.Экран();
Сообщить("Ширина = " + Экран1.Границы.Ширина);
