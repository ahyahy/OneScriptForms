Вкладки.Элемент (TabPageCollection.Item) Метод

Синтаксис

Элемент(<Индекс>, <Вкладка>)

Параметры

Индекс (обязательный)
Тип: Число.
Отсчитываемый от нуля индекс страницы вкладки, используемый для получения или установки.
Вкладка (необязательный)
Тип: Вкладка (TabPage).
Объект Вкладка (TabPage), представляющая устанавливаемый элемент.

Возвращаемое значение

Тип: Вкладка (TabPage).

Описание

Устанавливает и/или возвращает объект Вкладка (TabPage) в коллекции.

Примечание

Пример

ПанельВкладок1.Вкладки.Элемент(1, Ф.Вкладка("Вкладка4"));

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();
