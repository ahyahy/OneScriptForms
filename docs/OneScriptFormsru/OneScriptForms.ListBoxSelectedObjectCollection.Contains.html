ВыбранныеЭлементыПоляСписка.Содержит (ListBoxSelectedObjectCollection.Contains) Метод

Синтаксис

Содержит(<Значение>)

Параметры

Значение (обязательный)
Тип: Строка.
Элемент для поиска в коллекции.

Возвращаемое значение

Тип: Булево.

Истина - указанный элемент находится в коллекции, Ложь - в противном случае.

Описание

Определяет, находится ли указанный элемент в пределах коллекции.

Примечание

Пример

Содержит = ПолеСписка1.ВыбранныеЭлементы.Содержит("Элемент2");
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ПолеСписка1 = Форма1.ЭлементыУправления.Добавить(Ф.ПолеСписка());
ПолеСписка1.Стыковка = Ф.СтильСтыковки.Заполнение;
ПолеСписка1.РежимВыбора = Ф.РежимВыбора.МножественныйПростой;
ПолеСпискаЭлементы1 = ПолеСписка1.Элементы;
Для А = 1 По 10 Цикл
	ПолеСпискаЭлементы1.Добавить("Элемент" + А);
КонецЦикла;
ПолеСписка1.ВыбранныйЭлемент = ПолеСпискаЭлементы1.Элемент(0);
ПолеСписка1.ВыбранныйЭлемент = ПолеСпискаЭлементы1.Элемент(1);
ПолеСписка1.ВыбранныйЭлемент = ПолеСпискаЭлементы1.Элемент(3);

Содержит = ПолеСписка1.ВыбранныеЭлементы.Содержит("Элемент2");

Если Содержит Тогда
    Сообщить("Значение " + "Элемент2" + " содержится среди выбранных элементов списка.");
Иначе
    Сообщить("Значение " + "Элемент2" + " не найдено среди выбранных элементов списка.");
КонецЕсли;

Ф.ЗапуститьОбработкуСобытий();
