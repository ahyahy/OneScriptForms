РисованиеАрг.ПрямоугольникДляРисования (PaintEventArgs.ClipRectangle) Свойство

Использование

Только чтение.

Значение

Тип: Прямоугольник (Rectangle).

Возвращает прямоугольник, в котором будет осуществляться рисование.

Примечание

Пример

Аргументы = Ф.РисованиеАрг();
Сообщить("Аргументы.ПрямоугольникДляРисования = " + Аргументы.ПрямоугольникДляРисования);
