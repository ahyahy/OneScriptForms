Колонки.Вставить (ListViewColumnHeaderCollection.Insert) Метод

Синтаксис

Вставить(<Индекс>, <Колонка>)

Параметры

Индекс (обязательный)
Тип: Число.
Отсчитываемый от нуля индекс по которому вставляется колонка.
Колонка (обязательный)
Тип: Колонка (ColumnHeader).
Вставляемый в коллекцию экземпляр Колонка (ColumnHeader).

Возвращаемое значение

Тип: Колонка (ColumnHeader).

Вставленный в коллекцию экземпляр Колонка (ColumnHeader).

Описание

Вставляет колонку в коллекцию по указанному индексу.

Примечание

Пример

Колонка = Колонки1.Вставить(0, Ф.Колонка("А", 40, 1));
