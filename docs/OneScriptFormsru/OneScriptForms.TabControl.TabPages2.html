ПанельВкладок.Вкладки (TabControl.TabPages) Метод

Синтаксис

Вкладки(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.
Отсчитываемый от нуля индекс страницы вкладки, используемый для получения.

Возвращаемое значение

Тип: Вкладка (TabPage).

Описание

Возвращает значение по указанному индексу.

Примечание

Пример

Вкладка = ПанельВкладок1.Вкладки(0);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ПанельВкладок1 = Ф.ПанельВкладок();
ПанельВкладок1.Родитель = Форма1;
Вкладки1 = ПанельВкладок1.Вкладки;
Вкладка1 = ПанельВкладок1.Вкладки.Добавить(Ф.Вкладка("Вкладка1"));
Вкладка2 = ПанельВкладок1.Вкладки.Добавить(Ф.Вкладка());

Вкладка = ПанельВкладок1.Вкладки(0);
Сообщить("" + Вкладка.Текст);
