ЯчейкаТаблицы.ПредпочтительныйРазмер (DataGridViewCell.PreferredSize) Свойство

Использование

Только чтение.

Значение

Тип: Размер (Size).

Возвращает размер (в пикселях) прямоугольной области, в которую может быть вписана ячейка.

Свойство возвращает объект Размер (Size) с шириной и высотой -1, если ячейка не содержится в таблице.

Если ячейка находится в элементе КолонкаТаблицы (DataGridViewColumn), свойство ИзменяемыйРазмер (Resizable) которого имеет значение Неопределено, то для ширины будет возвращено значение -1.

Примечание

Сообщить("" + Таблица1.Ячейка(0, 0).ПредпочтительныйРазмер.Высота);
