ФормыДляОдноСкрипта (OneScriptForms) События

События, поддерживаемые библиотекой для класса ФормыДляОдноСкрипта (OneScriptForms), и ссылка на свойство задающее обработчик события, перечислены ниже. Полный список членов класса ФормыДляОдноСкрипта (OneScriptForms) смотрите в разделе ФормыДляОдноСкрипта (OneScriptForms) Члены.

События

КлиентПолучилДанные (ClientReceived) Возвращает аргументы события, связанные с действием при получении клиентом данных.
ПриОтключенииКлиента (ClientDisconnected) Возвращает аргументы события, связанные с действием при отключении клиента.
ПриОшибкеСервера (ErrorServer) Возвращает аргументы события, связанные с действием при возникновении ошибки или предупреждения на сервере.

СерверПолучилДанные (ServerReceived) Возвращает аргументы события, связанные с действием при получении сервером данных.
