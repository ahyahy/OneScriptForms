Dns.Разрешить (Dns.Resolve) Метод

Синтаксис

Разрешить(<Узел>)

Параметры

Узел (обязательный)
Тип: Строка.
DNS-имя узла или IP-адрес.

Возвращаемое значение

Объект IpУзел (IpHostEntry).

Описание

Разрешает DNS-имя узла или IP-адрес в экземпляр IpУзел (IpHostEntry).

Примечание

Пример

Сообщить("" + Dns1.Разрешить(Dns1.ПолучитьИмяУзла()));
