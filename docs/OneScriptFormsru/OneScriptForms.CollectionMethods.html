Коллекция (Collection) Методы

Методы класса Коллекция (Collection) перечислены ниже. Полный список членов класса Коллекция (Collection) смотрите в разделе Коллекция (Collection) Члены.

Методы

Добавить (Add) Добавляет элемент в коллекцию.
Удалить (Remove) Удаляет указанный элемент из коллекции.
Элемент (Item) Возвращает значение элемента по указанному ключу в коллекции.
