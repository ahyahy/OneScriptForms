Математика.Меньшее (Math.Min) Метод

Синтаксис

Меньшее(<Первое>, <Второе>)

Параметры

Первое (обязательный)
Тип: Число.
Первое из двух сравниваемых чисел.
Второе (обязательный)
Тип: Число.
Второе из двух сравниваемых чисел.

Возвращаемое значение

Тип: Число.

Меньшее из значений двух параметров Первое и Второе.

Описание

Возвращает меньшее из двух сравниваемых чисел.
