СеткаДанных.ВыбратьСтроку (DataGrid.SelectRow) Метод

Синтаксис

ВыбратьСтроку(<НомерСтроки>)

Параметры

НомерСтроки (обязательный)
Тип: Число.
Номер выделяемой строки.

Возвращаемое значение

Описание

Выделяет заданную строку.

Примечание

Пример

СеткаДанных1.ВыбратьСтроку(4);
