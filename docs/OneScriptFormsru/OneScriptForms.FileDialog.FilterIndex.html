ФайловыйДиалог.ИндексФильтра (FileDialog.FilterIndex) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает индекс фильтра, выбранного в настоящее время в окне ФайловыйДиалог (FileDialog). Первый фильтр имеет индекс 1. Значение по умолчанию это один.

Примечание

Пример

ДиалогОткрытияФайла1.ИндексФильтра = 2;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ДиалогОткрытияФайла1 = Ф.ДиалогОткрытияФайла();
ДиалогОткрытияФайла1.Фильтр = "Файлы изображений (*.BMP;*.jpg;*.GIF)|*.BMP;*.jpg;*.GIF|All files (*.*)|*.*|Текстовые файлы (*.txt)|*.txt|Все файлы (*.*)|*.*";
ДиалогОткрытияФайла1.ИндексФильтра = 2;
Если ДиалогОткрытияФайла1.ПоказатьДиалог() = Ф.РезультатДиалога.ОК  Тогда
	Сообщить("ДиалогОткрытияФайла1.ИмяФайла = " + ДиалогОткрытияФайла1.ИмяФайла);
	Сообщить("ДиалогОткрытияФайла1.ИндексФильтра = " + ДиалогОткрытияФайла1.ИндексФильтра);
КонецЕсли;

Ф.ЗапуститьОбработкуСобытий();
