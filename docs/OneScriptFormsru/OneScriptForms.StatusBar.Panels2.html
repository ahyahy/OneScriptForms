СтрокаСостояния.Панели (StatusBar.Panels) Метод

Синтаксис

Панели(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.
Индекс получаемого элемента в коллекции.

Возвращаемое значение

Тип: ПанельСтрокиСостояния (StatusBarPanel).

Объект ПанельСтрокиСостояния (StatusBarPanel), представляющий панель, расположенную по указанному индексу.

Описание

Возвращает значение по указанному индексу.

Примечание

Пример

Панель = СтрокаСостояния1.Панели(0);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Ширина = 900;
Форма1.Высота = 400;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

СтрокаСостояния1 = Ф.СтрокаСостояния();
СтрокаСостояния1.Родитель = Форма1;

СтрокаСостояния1.ПоказатьПанели = Истина;
Панели1 = СтрокаСостояния1.Панели;

Панель1 = Панели1.Добавить(Ф.ПанельСтрокиСостояния());
Панель1.Текст = "Панель1";
Панель1.СтильГраницы = Ф.СтильГраницыПанелиСтрокиСостояния.Утопленная;
Панель1.АвтоРазмер = Ф.АвтоРазмерПанелиСтрокиСостояния.Растяжимое;
Панель1.МинимальнаяШирина = 100;

Панель2 = Панели1.Добавить(Ф.ПанельСтрокиСостояния());
Панель2.Текст = "Панель2";
Панель2.СтильГраницы = Ф.СтильГраницыПанелиСтрокиСостояния.Утопленная;
Панель2.АвтоРазмер = Ф.АвтоРазмерПанелиСтрокиСостояния.Растяжимое;
Панель2.МинимальнаяШирина = 100;

Панель = СтрокаСостояния1.Панели(0);
Сообщить("" + Панель.Текст);

Ф.ЗапуститьОбработкуСобытий();
