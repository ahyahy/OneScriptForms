Конструктор

Процесс (Process) Конструктор Создаёт новый экземпляр класса Процесс (Process).

Свойства

Завершен (HasExited) Возвращает значение, определяющее завершение связанного процесса.
НачальнаяИнформация (StartInfo) Возвращает или задает набор значений для передачи их объекту Процесс (Process).
СтандартныйВывод (StandardOutput) Возвращает поток, используемый для чтения текстовых выходных данных приложения.

Методы

Закрыть (Close) Освобождает все ресурсы, связанные с этим компонентом.
Начать (Start) Запускает (или повторно использует) ресурс процесса, определенный свойством Процесс.НачальнаяИнформация (Process.StartInfo) этого компонента Процесс (Process), и связывает его с компонентом.
