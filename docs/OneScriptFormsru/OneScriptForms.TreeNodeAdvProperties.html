УзелДереваЗначений (TreeNodeAdv) Свойства

Свойства класса УзелДереваЗначений (TreeNodeAdv) перечислены ниже. Полный список членов класса УзелДереваЗначений (TreeNodeAdv) смотрите в разделе УзелДереваЗначений (TreeNodeAdv) Члены.

Свойства

Выбран (IsSelected) Возвращает или задает значение, показывающее, выбран ли узел дерева значений.
ДеревоЗначений (Tree) Возвращает объект ДеревоЗначений (TreeViewAdv), которое содержит данный узел.
ИмяУзла (NodeName) Возвращает или задает имя узла дерева значений.
Индекс (Index) Возвращает позицию узла коллекции.
Метка (Tag) Возвращает объект, содержащий дополнительные данные.
ПолныйПуть (FullPath) Возвращает путь из корневого узла дерева значений к текущему узлу дерева.
ПредыдущийУзел (PreviousNode) Возвращает объект УзелДереваЗначений (TreeNodeAdv) являющийся предыдущим узлом в структуре дерева значений.
Развернут (IsExpanded) Возвращает или задает значение, показывающее развернут ли узел.
Родитель (Parent) Возвращает или задает родительский узел дерева значений для текущего узла дерева значений.
СледующийУзел (NextNode) Возвращает объект УзелДереваЗначений (TreeNodeAdv) являющийся следующим узлом в структуре дерева значений.
Строка (Row) Возвращает номер строки для этого узла дерева значений.
Узлы (Nodes) Возвращает коллекцию объектов УзелДереваЗначений (TreeNodeAdv), назначенных текущему узлу дерева значений.
Уровень (Level) Возвращает уровень, на котором находится узел дерева значений.
