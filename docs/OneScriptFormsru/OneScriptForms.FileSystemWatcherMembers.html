Обзор НаблюдательФайловойСистемы (FileSystemWatcher)

Конструктор

НаблюдательФайловойСистемы (FileSystemWatcher) Конструктор Используйте метод ФормыДляОдноСкрипта.НаблюдательФайловойСистемы (OneScriptForms.FileSystemWatcher) для создания нового экземпляра класса НаблюдательФайловойСистемы (FileSystemWatcher).

Свойства

ВключаяПодкаталоги (IncludeSubDirectories) Возвращает или задает значение, указывающее, следует ли отслеживать подкаталоги в указанном пути.
КомпонентДоступен (EnableRaisingEvents) Возвращает или задает значение, определяющее, доступен ли данный компонент.
ПриИзменении (Changed) Возвращает или задает код для выполнения при изменении файла или каталога по заданному пути.
ПриПереименовании (Renamed) Возвращает или задает код для выполнения при переименовании файла или каталога по заданному пути.
ПриСоздании (Created) Возвращает или задает код для выполнения при создании файла или каталога по заданному пути.
ПриУдалении (Deleted) Возвращает или задает код для выполнения при удалении файла или каталога по заданному пути.
Путь (Path) Возвращает или задает путь отслеживаемого каталога.
РазмерВнутреннегоБуфера (InternalBufferSize) Возвращает или задает размер (в байтах) внутреннего буфера.
Тип (Type) (унаследовано от Компонент (Component)) Возвращает тип текущего экземпляра.
Фильтр (Filter) Возвращае или задает строку фильтра, используемую для определения файлов, контролируемых в каталоге.
ФильтрУведомлений (NotifyFilter) Возвращает или задает тип отслеживаемых изменений.

Методы

Освободить (Dispose) (унаследовано от Компонент (Component)) Освобождает ресурсы, используемые Компонент (Component).
