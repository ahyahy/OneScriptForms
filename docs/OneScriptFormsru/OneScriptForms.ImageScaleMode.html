Описание

Указывает масштабирование изображения для объекта ЗначокУзла (NodeStateIcon).

Члены

Имя члена Описание Значение
Вписать (Fit) Масштабирует изображение так, чтобы оно соответствовало прямоугольнику дисплея, соотношение сторон не фиксировано. 1
Клип (Clip) Не масштабировать 0
Масштабировать (AlwaysScale) Масштабирование изображения в соответствии с отображаемым прямоугольником с учетом соотношения сторон. 2
