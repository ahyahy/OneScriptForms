ОкноСообщений.Показать (MessageBox.Show) Метод

Синтаксис

Показать(<Текст>, <Заголовок>, <Кнопки>, <Значок>)

Параметры

Текст (необязательный)
Тип: Строка.
Текст, отображаемый в окне сообщения.
Заголовок (необязательный)
Тип: Строка.
Текст, отображаемый в строке заголовка окна сообщения.
Кнопки (необязательный)
Тип: КнопкиОкнаСообщений (MessageBoxButtons).
Одно из значений перечисления КнопкиОкнаСообщений (MessageBoxButtons), указывающее, какие кнопки должны отображаться в окне сообщения.
Значок (необязательный)
Тип: ЗначокОкнаСообщений (MessageBoxIcon).
Одно из значений перечисления ЗначокОкнаСообщений (MessageBoxIcon), указывающее, какой значок будет отображен в окне сообщения.

Возвращаемое значение

Тип: РезультатДиалога (DialogResult).

Одно из значений перечисления РезультатДиалога (DialogResult).

Описание

Отображает окно сообщения.

Примечание

Пример

Ф.ОкноСообщений().Показать("Текст", "Заголовок", Ф.КнопкиОкнаСообщений.ОКОтмена, Ф.ЗначокОкнаСообщений.Восклицание);
