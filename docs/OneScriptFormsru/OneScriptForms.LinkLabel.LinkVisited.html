НадписьСсылка.СсылкаПосещена (LinkLabel.LinkVisited) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, следует ли отображать ссылку как выбранную ранее.

Истина, если ссылка должна отображаться, как выбранная ранее; в противном случае — Ложь. Значение по умолчанию — Ложь.

Примечание

Элемент управления НадписьСсылка (LinkLabel) не отмечает автоматически ссылку просмотренной. Чтобы отобразить ссылку просмотренной, можно установить значение этого свойства в Истина в обработчике события НадписьСсылка.СсылкаНажата (LinkLabel.LinkClicked). Просмотренная ссылка отображается с использованием цвета, указанного в свойстве НадписьСсылка.ЦветПосещеннойСсылки (LinkLabel.VisitedLinkColor). После закрытия формы, содержащей элемент управления НадписьСсылка (LinkLabel), состояние отображения ссылки удаляется. Чтобы сохранить состояние отображения ссылки, необходимо сохранить состояние отображения ссылки в параметре реестра, связанном с приложением.

Это свойство влияет только на первую ссылку, определенную в элементе управления НадписьСсылка (LinkLabel). Если в элементе управления указано более одной ссылки, используйте свойство Ссылка.Посещена (Link.Visited). Доступ ко всем ссылкам, определенным в, элементе управления НадписьСсылка (LinkLabel) можно получить с помощью свойства НадписьСсылка.Ссылки (LinkLabel.Links).

Пример

НадписьСсылка1.СсылкаПосещена = Истина;

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Форма1.ЭлементыУправления.Добавить(НадписьСсылка1);
НадписьСсылка1.Текст = "Пример гиперссылок:" + Символы.ПС + "Первая ссылка - www.kixforms.org." + Символы.ПС + "Вторая ссылка - oscript.io.";
НадписьСсылка1.Границы = Ф.Прямоугольник(10, 10, 260, 100);
НадписьСсылка1.СтильГраницы = Ф.СтильГраницы.Одинарная;

НадписьСсылка1.СсылкаНажата = Новый Действие(ЭтотОбъект, "НадписьСсылка1_СсылкаНажата");

КоллекцияСсылок1 = НадписьСсылка1.Ссылки;
Ссылка1 = Ф.Ссылка();
Ссылка1.Начало = 36;
Ссылка1.Длина = 16;
Ссылка2 = Ф.Ссылка();
Ссылка2.Начало = 70;
Ссылка2.Длина = 10;
Ссылка2.Имя = "Ссылка2Имя";
Индекс1 = КоллекцияСсылок1.Добавить(Ссылка1);
Индекс2 = КоллекцияСсылок1.Добавить(Ссылка2);

НадписьСсылка1.СсылкаПосещена = Истина;
