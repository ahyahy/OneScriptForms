ПолеВыбора.Элементы (ComboBox.Items) Свойство

Использование

Только чтение.

Значение

Тип: ЭлементыПоляВыбора (ComboBoxObjectCollection).

Возвращает объект, представляющий коллекцию элементов содержащихся в ПолеВыбора (ComboBox).

Примечание

Пример

ЭлементыПоляВыбора1 = ПолеВыбора1.Элементы;

Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Ширина = 310;
Форма1.Отображать = Истина;
