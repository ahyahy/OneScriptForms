ЭлементУправления.Якорь (Control.Anchor) Свойство

Использование

Чтение и запись.

Значение

Тип: СтилиПривязки (AnchorStyles).

Возвращает или задает границы контейнера, с которым связан элемент управления, и определяет способ изменения размеров элемента управления при изменении размеров его родительского элемента.

Побитовая комбинация значений СтилиПривязки (AnchorStyles). Значение по умолчанию: Верх (Top) и Лево (Left).

Примечание

Пример

Панель1.Якорь = Ф.СтилиПривязки.Право + Ф.СтилиПривязки.Низ;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Панель1 = Ф.Панель();
Панель1.Родитель = Форма1;
Панель1.Текст = "Панель1";
Панель1.СтильГраницы = Ф.СтильГраницы.Трехмерная;
Панель1.Якорь = Ф.СтилиПривязки.Право + Ф.СтилиПривязки.Низ;
