Словарь.Значения (Dictionary.Values) Свойство

Использование

Только чтение.

Значение

Тип: МассивСписок (ArrayList).

Возвращает МассивСписок (ArrayList), содержащий значения словаря в произвольном порядке.

Примечание

Порядок значений в МассивСписок (ArrayList) не задан, но он совпадает с порядком связанных ключей в МассивСписок (ArrayList), возвращаемых методом Словарь.Ключи (Dictionary.Keys). Изменения в Словарь (Dictionary) не отражаются в МассивСписок (ArrayList).

Пример

МассивСписок1 = Словарь1.Значения;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Словарь1 = Ф.Словарь();
Словарь1.Добавить("Строка", "Простой стиль");
Словарь1.Добавить("Булево", Ложь);
Словарь1.Добавить("Число", 18.245);
Словарь1.Добавить(10.10, 18.245);
Словарь1.Добавить("Дата", Дата(1984,03,26,06,24,21));
Словарь1.Добавить("Объект", Форма1);

МассивСписок1 = Словарь1.Значения;
Для А = 0 По МассивСписок1.Количество - 1 Цикл
    Сообщить("Словарь1-Значение = " + МассивСписок1.Элемент(А));
КонецЦикла;
