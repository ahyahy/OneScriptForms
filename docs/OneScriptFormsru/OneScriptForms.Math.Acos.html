Математика.АКосинус (Math.Acos) Метод

Синтаксис

АКосинус(<Косинус>)

Параметры

Косинус (обязательный)
Тип: Число.
Число, представляющее косинус.

Возвращаемое значение

Тип: Число.

Угол, измеренный в радианах.

Описание

Возвращает числовое значение между 0 и Пи радианами для значения x, лежащего в диапазоне от -1 до 1.
