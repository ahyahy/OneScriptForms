Дерево.Отсортирован (TreeView.Sorted) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, производится ли сортировка узлов дерева в Дерево (TreeView).

Примечание

Если параметр Отсортирован (Sorted) имеет значение Истина, объекты дерева сортируются в алфавитном порядке по Текст (Text) значениям их свойств. Следует всегда использовать НачатьОбновление (BeginUpdate) и ЗавершитьОбновление (EndUpdate) для поддержания производительности при добавлении большого количества элементов в сортировку Дерево (TreeView) .

Пример

Дерево1.Отсортирован = Истина,
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Дерево1 = Форма1.ЭлементыУправления.Добавить(Ф.Дерево());
Узел1 = Дерево1.Узлы.Добавить("Узел2");
Узел2 = Дерево1.Узлы.Добавить("Узел1");
Дерево1.Отсортирован = Истина;
Сообщить("Дерево1.Отсортирован = " + Дерево1.Отсортирован); // Дерево1.Отсортирован = Да

Ф.ЗапуститьОбработкуСобытий();
