Таблица.РежимВыбора (DataGridView.SelectionMode) Свойство

Использование

Чтение и запись.

Значение

Тип: РежимВыбораТаблицы (DataGridViewSelectionMode).

Возвращает или задает значение, указывающее, каким образом могут быть выбраны ячейки объекта Таблица (DataGridView).

Одно из значений перечисления РежимВыбораТаблицы (DataGridViewSelectionMode). Значение по умолчанию - ЗаголовокСтроки (RowHeaderSelect).

Примечание

Пример

Таблица1.РежимВыбора = Ф.РежимВыбораТаблицы.Колонка;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Ширина = 500;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Родитель = Форма1;
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;

Таблица1.ИсточникДанных = Неопределено; // Если до этого была привязка к источнику данных.
Таблица1.КоличествоКолонок = 10;
Таблица1.КоличествоСтрок = 10;

Для А = 0 По Таблица1.Колонки.Количество - 1 Цикл
	Таблица1.Колонки(А).ТекстЗаголовка = "Кол" + А;
	Таблица1.Колонки(А).РежимСортировки = Ф.РежимСортировки.Программный;
КонецЦикла;
Для А = 0 По Таблица1.Строки.Количество - 1 Цикл
	Таблица1.Строки(А).ЗаголовокСтроки.Значение = "Стр №" + А;
КонецЦикла;

Таблица1.РежимВыбора = Ф.РежимВыбораТаблицы.Колонка;

Ф.ЗапуститьОбработкуСобытий();
