Ссылка.Данные (Link.LinkData) Свойство

Использование

Чтение и запись.

Значение

Тип: Произвольный.

Возвращает или задает данные, сопоставленные со ссылкой.

Примечание

Это свойство можно использовать для предоставления сведений, относящихся к ссылке. Сведения, предоставляемые этим свойством, можно использовать в обработке события НадписьСсылка.СсылкаНажата (LinkLabel.LinkClicked) для предоставления сведений о ссылке которую вы щелкнули. Например, можно указать URL-адрес, отображаемый в Internet Explorer при щелчке ссылки в качестве значения этого свойства. Кроме того, можно использовать это свойство для задания диалогового окна, отображаемого при щелчке ссылки пользователем.

Пример

Ссылка1.Данные = Дата(1984,03,26,06,24,21);

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Ширина = 350;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

НадписьСсылка1 = Ф.НадписьСсылка();
Форма1.ЭлементыУправления.Добавить(НадписьСсылка1);
НадписьСсылка1.Текст = "Пример гиперссылки:" + Символы.ПС + "scriptcoding.ru.";
НадписьСсылка1.Границы = Ф.Прямоугольник(10, 10, 300, 100);
НадписьСсылка1.СтильГраницы = Ф.СтильГраницы.Одинарная;

КоллекцияСсылок1 = НадписьСсылка1.Ссылки;
Ссылка1 = Ф.Ссылка();
Индекс1 = КоллекцияСсылок1.Добавить(Ссылка1); // Новая ссылка должна быть добавлена в коллекцию ссылок до установки каких либо её свойств.
Ссылка1.Начало = 20;
Ссылка1.Длина = 15;
Ссылка1.Посещена = Ложь;
Ссылка1.Данные = Дата(1984,03,26,06,24,21);
