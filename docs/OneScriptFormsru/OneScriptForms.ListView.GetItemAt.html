СписокЭлементов.ПолучитьЭлемент (ListView.GetItemAt) Метод

Синтаксис

ПолучитьЭлемент(<X>, <Y>)

Параметры

X (обязательный)
Тип: Число.
X-координата местоположения для поиска элемента (выражается в клиентских координатах).
Y (обязательный)
Тип: Число.
Y-координата местоположения для поиска элемента (выражается в клиентских координатах).

Возвращаемое значение

Тип: ЭлементСпискаЭлементов (ListViewItem).

Экземпляр ЭлементСпискаЭлементов (ListViewItem), который представляет элемент в указанной позиции. Если элемента в указанном месте нет, метод возвращает Неопределено.

Описание

Возвращает элемент, находящийся в указанном местоположении.

Примечание

Пример

ЭлементСпискаЭлементов = СписокЭлементов1.ПолучитьЭлемент(50, 50);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

СписокЭлементов1 = Форма1.ЭлементыУправления.Добавить(Ф.СписокЭлементов());
СписокЭлементов1.Стыковка = Ф.СтильСтыковки.Заполнение;
СписокЭлементов1.РежимОтображения = Ф.РежимОтображения.Список;
СписокЭлементов1.Флажки = Истина;

СписокЭлементов1.ПриостановитьРазмещение();
Элементы = СписокЭлементов1.Элементы;
М3 = СтрРазделить(Ф.СвойстваОбъекта(Ф.Цвет()), Символы.ПС);
Для А = 0 По М3.ВГраница() Цикл
	М4 = СтрРазделить(М3[А], " ");
	ИмяЦвета = М4[0];
	Попытка
		Цв = Ф.Цвет(ИмяЦвета);
	Исключение
		Продолжить;
	КонецПопытки;
	Элементы.Добавить(Ф.ЭлементСпискаЭлементов(ИмяЦвета));
КонецЦикла;
СписокЭлементов1.ВозобновитьРазмещение();

Элементы.Элемент(50).Помечен = Истина;
СписокЭлементов1.ОбеспечитьОтображение(50);
ЭлементСпискаЭлементов = СписокЭлементов1.ПолучитьЭлемент(50, 50);
Если ЭлементСпискаЭлементов = Неопределено Тогда
	Сообщить("По этим координатам элемента нет");
Иначе
	Сообщить("ЭлементСпискаЭлементов.Текст = " + ЭлементСпискаЭлементов.Текст);
КонецЕсли;

Ф.ЗапуститьОбработкуСобытий();
