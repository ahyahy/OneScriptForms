Графика.РисоватьСтроку (Graphics.DrawString) Метод

Синтаксис

РисоватьСтроку(<СтрокаТекста>, <Шрифт>, <Кисть>, <X>, <Y>)

Параметры

СтрокаТекста (обязательный)
Тип: Строка.
Строка для рисования.
Шрифт (обязательный)
Тип: Шрифт (Font).
Объект Шрифт (Font), определяющий формат текстовой строки.
Кисть (обязательный)
Тип: Произвольный.
Объект, производный от Кисть (Brush), определяющий цвет и текстуру рисуемого текста.
X (обязательный)
Тип: Число.
x координата верхнего левого угла нарисованного текста.
Y (обязательный)
Тип: Число.
y координата верхнего левого угла нарисованного текста.

Возвращаемое значение

Описание

Рисует указанную текстовую строку в указанном месте с указанными объектами СплошнаяКисть (SolidBrush) и Шрифт (Font).

Примечание

При создании объекта Графика (Graphics) с помощью метода СоздатьГрафику (CreateGraphics) какого либо элемента управления, код рисования графики необходимо поместить в процедуру, указанную в свойстве ПриПерерисовке (Paint) этого элемента управления.

Пример

Графика1 = Форма1.СоздатьГрафику();
Цвет1 = Ф.Цвет().Зеленый;
СплошнаяКисть1 = Ф.СплошнаяКисть(Цвет1);
Шрифт1 = Ф.Шрифт(, 12, Ф.СтильШрифта.Подчеркнутый);
Графика1.РисоватьСтроку("Строка графики." + Символы.ПС + "Вторая строка" + Символы.ПС + "Третья строка", Шрифт1, СплошнаяКисть1, 5, 5);
