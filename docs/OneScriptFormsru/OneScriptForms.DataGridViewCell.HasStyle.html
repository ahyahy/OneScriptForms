ЯчейкаТаблицы.СтильУстановлен (DataGridViewCell.HasStyle) Свойство

Использование

Только чтение.

Значение

Тип: Булево.

Возвращает значение, показывающее, было ли установлено свойство ЯчейкаТаблицы.Стиль (DataGridViewCell.Style).

Примечание

Пример

Сообщить("" + Таблица1.Ячейка(0, 0).СтильУстановлен);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Родитель = Форма1;
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;
Таблица1.КоличествоКолонок = 20;
Таблица1.КоличествоСтрок = 7;

Ячейка = Таблица1.Ячейка(0, 0);

Сообщить("" + Ячейка.СтильУстановлен);

Ф.ЗапуститьОбработкуСобытий();

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");

СтильЯчейки1 = Ф.СтильЯчейки();

Ячейка.Значение = "Буратно";
Ячейка.Стиль = СтильЯчейки1;

Таблица1.ТекущаяЯчейка = Таблица1.Ячейка(0, 1);
