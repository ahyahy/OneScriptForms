ДиалогВыбораКаталога.КорневойКаталог (FolderBrowserDialog.RootFolder) Свойство

Использование

Чтение и запись.

Значение

Тип: ОсобаяПапка (SpecialFolder).

Возвращает или задает корневой каталог, с которой начинается просмотр.

Примечание

Пример

ДиалогВыбораКаталога1.КорневойКаталог = Ф.ОсобаяПапка.МойКомпьютер;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ДиалогВыбораКаталога1 = Ф.ДиалогВыбораКаталога();
ДиалогВыбораКаталога1.КорневойКаталог = Ф.ОсобаяПапка.МойКомпьютер;
ДиалогВыбораКаталога1.КорневойКаталог = 17;
ДиалогВыбораКаталога1.КорневойКаталог = Ф.ОсобаяПапка.МойКомпьютер;
Сообщить("КорневойКаталог = " + ДиалогВыбораКаталога1.КорневойКаталог); // КорневойКаталог = 17

Ф.ЗапуститьОбработкуСобытий();
