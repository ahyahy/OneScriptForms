ВыравниваниеТекстаВПанелиИнструментов (ToolBarTextAlign) Перечисление

Описание

Указывает способ выравнивания текста на кнопке панели инструментов.

Члены

Имя члена Описание Значение
Понизу (Underneath) Текст выравнивается по нижней границе изображения кнопки панели инструментов. 0
Право (Right) Текст выравнивается по правому краю изображения кнопки панели инструментов. 1
