ПолеСписка.ГоризонтальнаяПрокрутка (ListBox.HorizontalScrollbar) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, отображается ли горизонтальная полоса прокрутки в элементе управления.

Истина означает, что горизонтальная полоса прокрутки отображается; в противном случае Ложь. Значение по умолчанию - Ложь.

Примечание

Для желаемого отображения длины строк можно, но не обязательно, задать свойство ПолеСписка.ГоризонтальнаяМера (ListBox.HorizontalExtent).

Свойство ПолеСписка.ГоризонтальнаяПрокрутка (ListBox.HorizontalScrollbar) нужно установить после заполнения списка. После установки свойства ПолеСписка.ГоризонтальнаяПрокрутка (ListBox.HorizontalScrollbar), добавленые в список новые строки не будут отображаться.

Пример

ПолеСписка1.ГоризонтальнаяПрокрутка = Истина;
Полный пример кода 

Копировать     Выделить всё


ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Ширина = 310;
Форма1.Высота = 325;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ПолеСписка1 = Ф.ПолеСписка();
Форма1.ЭлементыУправления.Добавить(ПолеСписка1);
ПолеСписка1.Ширина = 200;
ПолеСписка1.Высота = 300;

ПолеСпискаЭлементы1 = ПолеСписка1.Элементы;
Для А = 1 По 300 Цикл
	ПолеСпискаЭлементы1.Добавить(Ф.ЭлементСписка("Строка для поля списка " + А, А));
КонецЦикла;
ПолеСписка1.ГоризонтальнаяПрокрутка = Истина;
ПолеСписка1.ГоризонтальнаяМера = 225;
