ЯчейкаТаблицы.ГраницыСодержимого (DataGridViewCell.ContentBounds) Свойство

Использование

Только чтение.

Значение

Тип: Прямоугольник (Rectangle).

Возвращает прямоугольник, ограничивающий область содержимого ячейки.

Примечание

Пример

Сообщить("Ширина = " + Таблица1.Ячейка(0, 1).ГраницыСодержимого.Ширина);
