КолонкаФлажок.ТриСостояния (DataGridViewCheckBoxColumn.ThreeState) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, допускают ли размещенные ячейки флажков три контрольных состояния, а не два.

Значение Истина, если размещенные объекты ФлажокЯчейки (DataGridViewCheckBoxCell) могут иметь третье, неопределенное состояние; в противном случае - значение Ложь. Значение по умолчанию - Ложь.

Примечание

Пример

КолонкаФлажок1.ТриСостояния = Истина;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Родитель = Форма1;
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;
Таблица1.АвтоНумерацияСтрок = Истина;

КолонкаФлажок1 = Ф.КолонкаФлажок();
Таблица1.Колонки.Добавить(КолонкаФлажок1);
Таблица1.Колонки.Добавить(Ф.КолонкаФлажок());
Таблица1.КоличествоСтрок = 3;

КолонкаФлажок1.ТриСостояния = Истина;

Ф.ЗапуститьОбработкуСобытий();
