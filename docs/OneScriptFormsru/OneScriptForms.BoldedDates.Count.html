ВыделенныеДаты.Количество (BoldedDates.Count) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает количество элементов, содержащихся в ВыделенныеДаты (BoldedDates).

Примечание

Пример

Количество = Календарь1.ВыделенныеДаты.Количество;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Календарь1 = Ф.Календарь();
Календарь1.Родитель = Форма1;

МассивВыделенныхДат = Календарь1.ВыделенныеДаты;
Сообщить("Количество = " + МассивВыделенныхДат.Количество); // Количество = 0
МассивВыделенныхДат.Добавить(Дата(2019,02,16,01,01,01));
МассивВыделенныхДат.Добавить(Дата(2019,02,18,01,01,01));
Календарь1.ВыделенныеДаты.Добавить(Дата(2019,02,19,01,01,01));
Количество = Календарь1.ВыделенныеДаты.Количество;
Сообщить("Количество = " + Количество); // Количество = 3
