Картинка.УстановитьПиксель (Bitmap.SetPixel) Метод

Синтаксис

УстановитьПиксель(<X>, <Y>, <Цвет>)

Параметры

X (обязательный)
Тип: Число.
Координата X пикселя, который необходимо задать.
Y (обязательный)
Тип: Число.
Координата Y пикселя, который необходимо задать.
Цвет (обязательный)
Тип: Цвет (Color).
Экземпляр класса Цвет (Color), представляющий цвет, который нужно назначить указанному пикселю.

Возвращаемое значение

Описание

Задает цвет указанного пикселя в объекте Картинка (Bitmap).

Примечание

Пример

Картинка1.УстановитьПиксель(10, 10, Ф.Цвет("Аквамариновый"));
