ЭлементыПоляВыбораЯчейки.Элемент (DataGridViewComboBoxCellObjectCollection.Item) Метод

Синтаксис

Элемент(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.
Отсчитываемый от нуля индекс получаемого элемента.

Возвращаемое значение

Тип: ЭлементСписка (ListItem).

Значение по указанному индексу.

Описание

Возвращает значение по указанному индексу.

Пример
