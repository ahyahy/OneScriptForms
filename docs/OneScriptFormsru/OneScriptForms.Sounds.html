Звуки (Sounds) Перечисление

Описание

Указывает звуки, связанные с набором типов звуковых событий операционной системы Windows.

Члены

Имя члена Описание Значение
Вопрос (Question) Звуковой файл, связанный с событием программы Вопрос (Question) в текущей звуковой схеме Windows. 0
Восклицание (Exclamation) Звуковой файл, связанный с событием программы Восклицание (Exclamation) в текущей звуковой схеме Windows. 1
Гудок (Beep) Звуковой файл, связанный с событием программы Гудок (Beep) в текущей звуковой схеме Windows. 2
Звездочка (Asterisk) Звуковой файл, связанный с событием программы Звездочка (Asterisk) в текущей звуковой схеме Windows. 3
Рука (Hand) Звуковой файл, связанный с событием программы Рука (Hand) в текущей звуковой схеме Windows. 4
