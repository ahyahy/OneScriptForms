СтрокиТаблицы.Добавить (DataGridViewRowCollection.Add) Метод

Синтаксис

Добавить(<СтрокаТаблицы>)

Параметры

СтрокаТаблицы (необязательный)
Тип: СтрокаТаблицы (DataGridViewRow).
Объект СтрокаТаблицы (DataGridViewRow), добавляемый в список СтрокиТаблицы (DataGridViewRowCollection).

Возвращаемое значение

Тип: Число.

Индекс новой строки.

Описание

Добавляет новую строку или указанный параметр СтрокаТаблицы в коллекцию.

Примечание

Пример

Таблица1.Строки.Добавить();
