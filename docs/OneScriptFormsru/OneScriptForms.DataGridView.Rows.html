Таблица.Строки (DataGridView.Rows) Свойство

Использование

Только чтение.

Значение

Тип: СтрокиТаблицы (DataGridViewRowCollection).

Возвращает коллекцию, содержащую все строки в элементе управления Таблица (DataGridView).

Примечание

Пример

Строки1 = Таблица1.Строки;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Родитель = Форма1;
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;

Таблица1.Колонки.Добавить(Ф.КолонкаПолеВвода());
Таблица1.Колонки.Элемент(0).ТекстЗаголовка = "КолонкаПолеВвода";
Таблица1.Колонки.Элемент(0).Ширина = 150;
Таблица1.ДобавлятьСтроки = Ложь;

Строки1 = Таблица1.Строки;

Строки1.Добавить();

Ф.ЗапуститьОбработкуСобытий();
