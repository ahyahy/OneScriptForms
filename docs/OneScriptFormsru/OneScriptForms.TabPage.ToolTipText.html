Вкладка.ТекстПодсказки (TabPage.ToolTipText) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает текст всплывающей подсказки для этой вкладки.

Примечание

Свойство ПанельВкладок.ПоказатьПодсказку (TabControl.ShowToolTips) должно быть установлено в Истина.

Пример

Вкладка1.ТекстПодсказки = "Это подсказка для Вкладка1";
Полный пример кода 

Копировать     Выделить всё


ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ПанельВкладок1 = Форма1.ЭлементыУправления.Добавить(Ф.ПанельВкладок());
Вкладка1 = Ф.Вкладка();
Вкладка1.Родитель = ПанельВкладок1;
Вкладка1.Текст = "Вкладка1";
Вкладка2 = ПанельВкладок1.Вкладки.Добавить(Ф.Вкладка("Вкладка2"));

ПанельВкладок1.ПоказатьПодсказку = Истина;
Вкладка1.ТекстПодсказки = "Это подсказка для Вкладка1";

Ф.ЗапуститьОбработкуСобытий();
