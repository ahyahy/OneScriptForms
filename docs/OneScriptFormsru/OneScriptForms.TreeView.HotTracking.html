Дерево.Гиперссылка (TreeView.HotTracking) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, отображается ли метка узла дерева в виде гиперссылки, когда указатель мыши помещается над данным узлом дерева.

Примечание

Если свойство Флажки (CheckBoxes) имеет значение Истина , свойство Гиперссылка (HotTracking) не действует.

Пример

Дерево1.Гиперссылка = Истина;
Полный пример кода 

Копировать     Выделить всё


ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Дерево1 = Форма1.ЭлементыУправления.Добавить(Ф.Дерево());
Дерево1.Высота = 250;
Узел1 = Дерево1.Узлы.Добавить("Узел1");
Узел2 = Дерево1.Узлы.Добавить("Узел2");
Узлы1 = Узел1.Узлы;
ПодУзел1 = Узлы1.Добавить("ПодУзел1");
ПодУзел2 = Узлы1.Добавить("ПодУзел2");
Узел1.Развернуть();
Дерево1.Гиперссылка = Истина;
Сообщить("Дерево1.Гиперссылка = " + Дерево1.Гиперссылка); // Дерево1.Гиперссылка = Да

Ф.ЗапуститьОбработкуСобытий();
