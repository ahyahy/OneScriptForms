ЭлементыСпискаЭлементов.Удалить (ListViewItemCollection.Remove) Метод

Синтаксис

Удалить(<Элемент>)

Параметры

Элемент (обязательный)
Тип: ЭлементСпискаЭлементов (ListViewItem).
Объект ЭлементСпискаЭлементов (ListViewItem), представляющий элемент для удаления из коллекции.

Возвращаемое значение

Описание

Удаляет указанный элемент из коллекции.

Примечание

Пример

Элементы.Удалить(Элемент1);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

СписокЭлементов1 = Форма1.ЭлементыУправления.Добавить(Ф.СписокЭлементов());
СписокЭлементов1.Стыковка = Ф.СтильСтыковки.Заполнение;
СписокЭлементов1.РежимОтображения = Ф.РежимОтображения.Список;
Элемент1 = Ф.ЭлементСпискаЭлементов();
Элемент1.Текст = "Элемент1";
Элементы = СписокЭлементов1.Элементы;
Элементы.Добавить(Элемент1);

Элементы.Добавить("Элемент2");
Эл = Элементы.Вставить(0, Ф.ЭлементСпискаЭлементов("Элемент0"));

Элементы.Удалить(Элемент1);

Ф.ЗапуститьОбработкуСобытий();
