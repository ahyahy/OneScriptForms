Форма.ПриАктивизации (Form.Activated) Свойство

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код для выполнения, при активизации формы, определенной в коде или заданной пользователем.

Примечание

Пример

Форма1.ПриАктивизации = Ф.Действие(ЭтотОбъект, "Форма1_ПриАктивизации");
