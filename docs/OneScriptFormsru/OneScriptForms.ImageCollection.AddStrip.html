Изображения.ДобавитьПолосу (ImageCollection.AddStrip) Метод

Синтаксис

ДобавитьПолосу(<Изображение>)

Параметры

Изображение (обязательный)
Тип: Картинка (Bitmap).
Объект Картинка (Bitmap) с добавляемым изображением (изображениями).

Возвращаемое значение

Тип: Число.

Индекс добавленного изображения или -1, если изображение не может быть добавлено.

Описание

Добавляет полосу изображений заданного изображения для СписокИзображений (ImageList).

Примечание

Пример

Изображения1.ДобавитьПолосу(Изображение);
