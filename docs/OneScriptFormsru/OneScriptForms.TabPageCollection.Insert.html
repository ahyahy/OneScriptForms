Вкладки.Вставить (TabPageCollection.Insert) Метод

Синтаксис

Вставить(<Индекс>, <Страница>)

Параметры

Индекс (обязательный)
Тип: Число.
Отсчитываемый от нуля индекс, по которому вставляется страница вкладки.
Страница (обязательный)
Тип: Строка; Вкладка (TabPage).
Если значение строковое, тогда текст для отображения на странице вкладки. Иначе будет вставлена Вкладка (TabPage).

Возвращаемое значение

Тип: Вкладка (TabPage).

Объект Вкладка (TabPage), который был вставлен.

Описание

Создает новую страницу вкладки с заданным текстом и вставляет его в коллекцию по указанному индексу.

Примечание

Пример

Вкладки1.Вставить(0, Ф.Вкладка("Вставленная1"));
Вкладки1.Вставить(0, "Вставленная2");
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Ширина = 500;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ПанельВкладок1 = Форма1.ЭлементыУправления.Добавить(Ф.ПанельВкладок());
ПанельВкладок1.Ширина = Форма1.Ширина - 20;
Вкладка1 = ПанельВкладок1.Вкладки.Добавить(Ф.Вкладка("Вкладка1"));
Вкладка2 = ПанельВкладок1.Вкладки.Добавить(Ф.Вкладка("Вкладка2"));
Вкладка3 = ПанельВкладок1.Вкладки.Добавить(Ф.Вкладка("Вкладка3"));

Вкладки1 = ПанельВкладок1.Вкладки;
Вкладки1.Вставить(0, Ф.Вкладка("Вставленная1"));
