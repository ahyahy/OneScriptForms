TCPКлиентSSL (TCPClientSSL) Методы

Методы класса TCPКлиентSSL (TCPClientSSL) перечислены ниже. Полный список членов класса TCPКлиентSSL (TCPClientSSL) смотрите в разделе TCPКлиентSSL (TCPClientSSL) Члены.

Методы

Закрыть (Close) Удаляет данный экземпляр TCPClientSSL и запрашивает закрытие базового подключения TCP.
ОбработатьКлиентПолучилДанные (ProcessingClientReceived) Метод создает событие ФормыДляОдноСкрипта.КлиентПолучилДанные (OneScriptForms.ClientReceived).
Отправить (Send) Отправляет сообщение.
Подключить (Connect) Подключает клиента к удаленному TCP-узлу, используя указанный IP-адрес и номер порта.
ПолучитьПотокSSL (GetStreamSSL) Возвращает объект SSLПоток (SslStreamWrapper), используемый для отправки и получения данных.
