ЕжегодныеДаты (AnnuallyBoldedDates) Класс

Описание

Представляет массив объектов типа Дата, который определяет, какие даты ежегодно будут выводиться полужирным шрифтом в объекте Календарь (MonthCalendar).

Список всех членов этого типа смотрите в разделе ЕжегодныеДаты (AnnuallyBoldedDates) Члены.
