Календарь.ВыделенныеДаты (MonthCalendar.BoldedDates) Метод

Синтаксис

ВыделенныеДаты(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.
Отсчитываемый от нуля индекс получаемого элемента.

Возвращаемое значение

Тип: Дата.

Значение по указанному индексу.

Описание

Возвращает значение по указанному индексу.

Примечание

Пример

Дата1 = Календарь1.ВыделенныеДаты(0);
