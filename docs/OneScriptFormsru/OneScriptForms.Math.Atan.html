Математика.АТангенс (Math.Atan) Метод

Синтаксис

АТангенс(<Тангенс>)

Параметры

Тангенс (обязательный)
Тип: Число.
Число, представляющее тангенс.

Возвращаемое значение

Тип: Число.

Угол, q, измеренный в радианах.

Описание

Возвращает числовое значение между -Пи/2 и Пи/2 радианами.
