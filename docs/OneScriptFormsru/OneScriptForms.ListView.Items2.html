СписокЭлементов.Элементы (ListView.Items) Метод

Синтаксис

Элементы(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.
Индекс получаемого элемента в коллекции.

Возвращаемое значение

Тип: ЭлементСпискаЭлементов (ListViewItem).

Описание

Возвращает значение по указанному индексу.

Примечание

Пример

Эл = СписокЭлементов1.Элементы(0);
Полный пример кода 




Перем Ф, Аргументы1_Индекс;

Процедура СписокЭлементов1_ЭлементПомечен()
	Аргументы1 = Ф.ЭлементПомеченАрг();
	Аргументы1_Индекс = Аргументы1.Индекс;
	Сообщить("Аргументы1.Индекс = " + Аргументы1.Индекс);
	Сообщить("Аргументы1.НовоеЗначение = " + Аргументы1.НовоеЗначение);
	Сообщить("Аргументы1.ТекущееЗначение  = " + Аргументы1.ТекущееЗначение);
	Сообщить("============================");
