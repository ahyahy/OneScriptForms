Поток (Stream) Свойства

Свойства класса Поток (Stream) перечислены ниже. Полный список членов класса Поток (Stream) смотрите в разделе Поток (Stream) Члены.

Свойства

ВозможностьЗаписи (CanWrite) Возвращает значение, которое показывает, поддерживает ли текущий поток возможность записи.
ВозможностьПоиска (CanSeek) Возвращает значение, которое показывает, поддерживается ли в текущем потоке возможность поиска.
ВозможностьЧтения (CanRead) Возвращает значение, показывающее, поддерживает ли текущий поток возможность чтения.
Длина (Length) Возвращает длину потока в байтах.
Позиция (Position) Возвращает или задает позицию в текущем потоке.
