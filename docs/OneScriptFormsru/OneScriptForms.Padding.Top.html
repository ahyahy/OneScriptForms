Заполнение.Верх (Padding.Top) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает значение внутренних полей для верхнего края.

Задание этого значения также может изменить свойство Заполнение.Все (Padding.All).

Примечание

Пример

Заполнение1.Верх = 10;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;
Таблица1.Родитель = Форма1;
Таблица1.КоличествоКолонок = 2;
Таблица1.КоличествоСтрок = 5;

Таблица1.АвтоНумерацияСтрок = Истина;

Заполнение1 = Ф.Заполнение();
Заполнение1.Верх = 10;

Сообщить("" + Заполнение1.Верх);

СтильЯчейки1 = Ф.СтильЯчейки();
СтильЯчейки1.Заполнение = Заполнение1;
СтильЯчейки1.Выравнивание = Ф.ВыравниваниеСодержимогоЯчейки.СерединаЦентр;
СтильЯчейки1.ЦветФона = Таблица1.ЦветФона;
Таблица1.СтильЗаголовковСтрок = СтильЯчейки1;

Ф.ЗапуститьОбработкуСобытий();
