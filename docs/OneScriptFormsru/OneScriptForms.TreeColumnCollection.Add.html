КолонкиДереваЗначений.Добавить (TreeColumnCollection.Add) Метод

Синтаксис

Добавить(<Колонка>)

Параметры

Колонка (обязательный)
Тип: КолонкаДереваЗначений (TreeColumn).
Добавляемый объект КолонкаДереваЗначений (TreeColumn).

Возвращаемое значение

Описание

Добавляет колонку в коллекцию.

Примечание

Пример

ДеревоЗначений1.Колонки.Добавить(КолонкаДереваЗначений3);
