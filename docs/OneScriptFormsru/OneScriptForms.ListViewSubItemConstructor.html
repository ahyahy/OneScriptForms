ПодэлементСпискаЭлементов (ListViewSubItem) Конструктор

Синтаксис

ПодэлементСпискаЭлементов(<Текст>)

Параметры

Текст (необязательный)
Тип: Строка.

Описание

Используйте метод ФормыДляОдноСкрипта.ПодэлементСпискаЭлементов (OneScriptForms.ListViewSubItem) для создания нового экземпляра класса ПодэлементСпискаЭлементов (ListViewSubItem).

Примечание

Пример

ПодэлементСпискаЭлементов1 = Ф.ПодэлементСпискаЭлементов("3");
