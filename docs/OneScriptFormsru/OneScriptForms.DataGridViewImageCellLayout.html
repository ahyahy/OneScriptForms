РазмещениеИзображенияЯчейки (DataGridViewImageCellLayout) Перечисление

Описание

Указывает расположение для изображения, содержащегося в объекте КартинкаЯчейки (DataGridViewImageCell).

Члены

Имя члена Описание Значение
Масштабировать (Zoom) Рисунок растягивается равномерно, пока он не достигнет ширины и высоты содержащей его ячейки. 3
НеУстановлено (NotSet) Спецификация расположения не установлена. 0
Растянуть (Stretch) Рисунок растягивается до такой ширины и высоты, чтобы заполнить содержащую его ячейку. 2
Стандартное (Normal) Рисунок отображается по центру с использованием его собственного разрешения. 1
