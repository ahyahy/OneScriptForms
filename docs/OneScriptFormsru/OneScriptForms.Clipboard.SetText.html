БуферОбмена.УстановитьТекст (Clipboard.SetText) Метод

Синтаксис

УстановитьТекст(<Текст>)

Параметры

Текст (обязательный)
Тип: Строка.

Возвращаемое значение

Описание

Очищает буфер обмена и добавляет текстовые данные в формате UnicodeText.

Примечание

Пример

Буфер1.УстановитьТекст("Текст для буфера обмена.");
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Буфер1 = Ф.БуферОбмена();

Если Ф.ПлатформаWin Тогда
	Буфер1.УстановитьТекст("Текст для буфера обмена.");
Иначе
	Буфер1.УстановитьТекст("Text for the clipboard.");
КонецЕсли;

Стр1 = Буфер1.ПолучитьТекст();
Если Стр1 = Неопределено Тогда
	Сообщить("Неопределено"); // Неопределено
Иначе
	Сообщить("Стр1 = " + Стр1);
КонецЕсли;

Ф.ЗапуститьОбработкуСобытий();
