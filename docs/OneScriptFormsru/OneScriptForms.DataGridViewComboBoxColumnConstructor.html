КолонкаПолеВыбора (DataGridViewComboBoxColumn) Конструктор

Синтаксис

КолонкаПолеВыбора()

Параметры

Описание

Создаёт новый экземпляр класса КолонкаПолеВыбора (DataGridViewComboBoxColumn).

Примечание

Для начала выбора значения можно использовать клавиши F2 или F4.

  • Заполнение поля выбора с помощью таблицы данных как источника данных:
