Курсоры.Имя (Cursors.Name) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает имя элемента управления.

Примечание

Пример

Курсоры1 = Ф.Курсоры();
Курсоры1.Имя = "Курсоры1";
Сообщить("Курсоры1.Имя = " + Курсоры1.Имя);
Полный пример кода 



Перем Ф,  ТекущийКурсор, Кнопка1;

Процедура Кнопка1_Нажатие() Экспорт
    Если ТекущийКурсор = "Ладонь" Тогда
        Кнопка1.Курсор = Ф.Курсоры().Перекрестие;
        ТекущийКурсор = "Перекрестие";
    ИначеЕсли ТекущийКурсор = "Перекрестие" Тогда
        Кнопка1.Курсор = Ф.Курсоры().Ладонь;
        ТекущийКурсор = "Ладонь";
    КонецЕсли;
