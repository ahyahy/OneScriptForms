Надпись.СтильГраницы (Label.BorderStyle) Свойство

Использование

Чтение и запись.

Значение

Тип: СтильГраницы (BorderStyle).

Указывает стиль рамки для элемента управления.

Примечание

Пример

Надпись1.СтильГраницы = Ф.СтильГраницы.Одинарная;
Надпись1.СтильГраницы = 1;
