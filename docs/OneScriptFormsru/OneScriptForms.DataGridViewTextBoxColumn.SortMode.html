КолонкаПолеВвода.РежимСортировки (DataGridViewTextBoxColumn.SortMode) Свойство

Использование

Чтение и запись.

Значение

Тип: РежимСортировки (DataGridViewColumnSortMode).

Возвращает или задает режим сортировки для колонки.

Объект РежимСортировки (DataGridViewColumnSortMode) задает критерии, используемые для сортировки строк по значениям, которые содержатся в ячейках колонок.

Примечание

Режим сортировки для КолонкаПолеВвода (DataGridViewTextBoxColumn) по умолчанию - Автоматически (Automatic). Режим сортировки по умолчанию для других типов колонок - НеСортируемый (NotSortable).

Пример

КолонкаПолеВвода1.РежимСортировки = Ф.РежимСортировки.Автоматически;
