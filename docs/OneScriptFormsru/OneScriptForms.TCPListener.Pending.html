TCPСлушатель.Ожидающие (TCPListener.Pending) Метод

Синтаксис

Ожидающие()

Параметры

Возвращаемое значение

Тип: Булево.

Описание

Определяет, имеются ли ожидающие запросы на подключение.

Примечание

Пример

Ожидающие = TCPСлушатель1.Ожидающие();
