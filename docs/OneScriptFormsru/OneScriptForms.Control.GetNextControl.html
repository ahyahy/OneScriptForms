ЭлементУправления.СледующийЭлемент (Control.GetNextControl) Метод

Синтаксис

СледующийЭлемент(<ЭлементУправления>, <Вперед>)

Параметры

ЭлементУправления (обязательный)
Тип: Произвольный.
Элемент управления, с которого следует начать поиск.
Вперед (обязательный)
Тип: Булево.
Значение Истина, для поиска в прямом направлении в последовательности табуляции; Ложь - для поиска в обратном направлении.

Возвращаемое значение

Тип: Произвольный.

Описание

Возвращает следующий или предыдущий элемент среди дочерних элементов управления в последовательности клавиши TAB.

Примечание

Пример

Сообщить("Следующий элемент = " + Форма1.СледующийЭлемент(Кнопка3, Истина).Текст);
Сообщить("Предыдущий элемент = " + Форма1.СледующийЭлемент(Кнопка3, Ложь).Текст);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;

Форма1.Активизировать();

Кнопка2.Ниже(Кнопка1, 5);
Кнопка2.ТабФокус = Истина;
Кнопка2.ПорядокОбхода = 6;
