СортированныйСписок.Добавить (SortedList.Add) Метод

Синтаксис

Добавить(<Ключ>, <Значение>)

Параметры

Ключ (обязательный)
Тип: Произвольный.
Ключ добавляемого элемента.
Значение (обязательный)
Тип: Произвольный.
Значение добавляемого элемента.

Возвращаемое значение

Описание

Добавляет элемент с указанным ключом и значением в СортированныйСписок (SortedList).

Примечание

Тип ключа должен быть одинаковым для всей коллекции.

Пример

СортированныйСписок1.Добавить(1.10, "Простой стиль");
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

СортированныйСписок1 = Ф.СортированныйСписок();
СортированныйСписок1.Добавить(1.10, "Простой стиль");
СортированныйСписок1.Добавить(2.11, "Простой стиль2");
СортированныйСписок1.Добавить(3.12, Ложь);
СортированныйСписок1.Добавить(4.13, 18.245);
СортированныйСписок1.Добавить(3, 18.246);
СортированныйСписок1.Добавить(5.15, Дата(1984,03,26,06,24,21));
СортированныйСписок1.Добавить(6.16, Форма1);
