СтильЯчейки.Шрифт (DataGridViewCellStyle.Font) Свойство

Использование

Чтение и запись.

Значение

Тип: Шрифт (Font).

Возвращает или задает шрифт, применимый к текстовому содержимому ячейки объекта Таблица (DataGridView).

Примечание

Пример

Шрифт1 = Ф.Шрифт(, 10, Ф.СтильШрифта.Подчеркнутый + Ф.СтильШрифта.Курсив);
СтильЯчейки1.Шрифт = Шрифт1;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Родитель = Форма1;
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;
Таблица1.КоличествоКолонок = 2;
Таблица1.КоличествоСтрок = 5;

СтильЯчейки1 = Ф.СтильЯчейки();
Шрифт1 = Ф.Шрифт(, 10, Ф.СтильШрифта.Подчеркнутый + Ф.СтильШрифта.Курсив);
СтильЯчейки1.Шрифт = Шрифт1;

Ячейка = Таблица1.Ячейка(0, 0);
Ячейка.Значение = "Буратно";
Ячейка.Стиль = СтильЯчейки1;

Таблица1.ТекущаяЯчейка = Таблица1.Ячейка(0, 1);
