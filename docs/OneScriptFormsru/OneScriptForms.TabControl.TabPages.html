ПанельВкладок.Вкладки (TabControl.TabPages) Свойство

Использование

Только чтение.

Значение

Тип: Вкладки (TabPageCollection).

Возвращает набор вкладок в этом элементе управления ПанельВкладок (TabControl).

Примечание

Пример

Вкладки1 = ПанельВкладок1.Вкладки;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ПанельВкладок1 = Ф.ПанельВкладок();
ПанельВкладок1.Родитель = Форма1;
Вкладки1 = ПанельВкладок1.Вкладки;

Сообщить("Вкладки1.Количество = " + Вкладки1.Количество); // Вкладки1.Количество = 0

Ф.ЗапуститьОбработкуСобытий();
