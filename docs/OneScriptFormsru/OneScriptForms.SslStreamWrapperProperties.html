SSLПоток (SslStreamWrapper) Свойства

Свойства класса SSLПоток (SslStreamWrapper) перечислены ниже. Полный список членов класса SSLПоток (SslStreamWrapper) смотрите в разделе SSLПоток (SslStreamWrapper) Члены.

Свойства

ВозможностьЗаписи (CanWrite) Возвращает значение, указывающее, поддерживает ли объект запись. Значение Истина, если данные могут быть записаны в поток, в противном случае - Ложь.
ВозможностьЧтения (CanRead) Возвращает значение, указывающее, поддерживает ли объект чтение. Значение Истина, если данные могут быть прочитаны из потока, в противном случае - Ложь.
