Заполнение (Padding) Конструктор

Синтаксис

Заполнение()

Заполнение(<Все>)

Заполнение(<Лево>, <Верх>, <Право>, <Низ>)

Параметры

Все
Тип: Число.
Размер внутренних полей (в пикселях) на всех краях.
Лево
Тип: Число.
Размер внутренних полей (в пикселях) для левого края.
Верх
Тип: Число.
Размер внутренних полей (в пикселях) для верхнего края.
Право
Тип: Число.
Размер внутренних полей (в пикселях) для правого края.
Низ
Тип: Число.
Размер внутренних полей (в пикселях) для нижнего края.

Описание

Создаёт новый экземпляр класса Заполнение (Padding).

Примечание

При некоторых значениях объекта Заполнение (Padding) изображение стрелки в заголовках строк объекта Таблица (DataGridView) может не отображаться.

Пример

Заполнение1 = Ф.Заполнение();
Заполнение2 = Ф.Заполнение(4);
Заполнение3 = Ф.Заполнение(4, 3, 4, 3);
