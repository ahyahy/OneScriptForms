ПлоскийСтиль (FlatStyle) Перечисление

Описание

Задает внешний вид элемента управления.

Члены

Имя члена Описание Значение
Всплывающий (Popup) Элемент управления выглядит плоским, пока указатель мыши перемещается над ним, после чего становится объемным. 1
Плоский (Flat) Элемент управления выглядит плоским. 0
Система (System) Внешний вид элемента управления определяется операционной системой пользователя. 3
Стандартный (Standard) Элемент управления выглядит объемным. 2
