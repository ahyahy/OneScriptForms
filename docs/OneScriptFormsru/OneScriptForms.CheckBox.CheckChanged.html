Флажок.ПометкаИзменена (CheckBox.CheckChanged) Свойство

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код для выполнения, когда значение свойства Помечен (Checked) изменилось.

Примечание

Пример

Флажок1.ПометкаИзменена = Ф.Действие(ЭтотОбъект, "Флажок1_ПометкаИзменена");
Полный пример кода 




Перем Ф, Флажок1;

Процедура Флажок1_ПометкаИзменена() Экспорт
	Если Флажок1.Помечен = Истина Тогда
		Флажок1.ЦветФона = Ф.Цвет("Васильковый");
	Иначе
		Флажок1.ЦветФона = Ф.Цвет("Фиолетовый");
	КонецЕсли;
КонецПроцедуры
