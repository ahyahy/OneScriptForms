УзелДереваЗначенийАрг.ЭлементУзла (TreeNodeAdvMouseEventArgs.Control) Свойство

Использование

Только чтение.

Значение

Тип: Произвольный.

Возвращает объект одного из классов ФлажокУзла (NodeCheckBox), ЗначокУзла (NodeStateIcon), ПолеВыбораУзла (NodeComboBox), ЧисловоеПолеУзла (NodeDecimalTextBox), РегуляторВверхВнизУзла (NodeNumericUpDown), ПолеВводаУзла (NodeTextBox).

Примечание

Пример

УзелДереваЗначенийАрг = Ф.АргументыСобытия;
ЭлементУзла = УзелДереваЗначенийАрг.ЭлементУзла;
