ПолеВыбораЯчейки.СтильОтображения (DataGridViewComboBoxCell.DisplayStyle) Свойство

Использование

Чтение и запись.

Значение

Тип: СтильПоляВыбораЯчейки (DataGridViewComboBoxDisplayStyle).

Возвращает или задает значение, которое определяет способ отображения поля со списком, когда оно не в режиме редактирования.

Примечание

Пример

ПолеВыбораЯчейки = Таблица1.Ячейка(0, 0);
Сообщить("" + ПолеВыбораЯчейки.ПлоскийСтиль);
ПолеВыбораЯчейки.ПлоскийСтиль = Ф.СтильПоляВыбораЯчейки.ПолеВыбора;
