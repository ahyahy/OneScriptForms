УзелДереваЗначений.Узлы (TreeNodeAdv.Nodes) Свойство

Использование

Только чтение.

Значение

Тип: УзлыДереваЗначений (TreeNodeAdvCollection).

Возвращает коллекцию объектов УзелДереваЗначений (TreeNodeAdv), назначенных текущему узлу дерева значений.

Примечание

Пример

Сообщить("" + Солнце.Узлы.Количество);

Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Ширина = 500;
Форма1.Высота = 250;
Форма1.Отображать = Истина;
Форма1.Показать();

ДеревоЗначений1 = Ф.ДеревоЗначений();

КолонкаДереваЗначений1 = Ф.КолонкаДереваЗначений("Небесные тела", 220);

Сатурн.УстановитьЗначение(ЧисловоеПолеУзла1, 9.54);
