Компонент.Тип (Component.Type) Свойство

Использование

Только чтение.

Значение

Тип: Тип (Type).

Возвращает тип текущего экземпляра.

Примечание

Свойство не используется непосредственно.

Пример

Таймер1 = Ф.Таймер();
Сообщить("Таймер1.Тип.ВСтроку() = " + Таймер1.Тип.ВСтроку());
Полный пример кода
