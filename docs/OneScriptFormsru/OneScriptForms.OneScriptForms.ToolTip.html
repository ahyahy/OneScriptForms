Синтаксис

Подсказка()

Параметры

Возвращаемое значение

Тип: Подсказка (ToolTip).

Описание

Создаёт новый экземпляр класса Подсказка (ToolTip).

Примечание

Пример

Подсказка1 = Ф.Подсказка();
