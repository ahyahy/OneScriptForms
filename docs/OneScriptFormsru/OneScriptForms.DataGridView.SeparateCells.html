Таблица.РазъединитьЯчейки (DataGridView.SeparateCells) Метод

Синтаксис

РазъединитьЯчейки(<НомерКолонки>, <НомерСтроки>)

Параметры

НомерКолонки (обязательный)
Тип: Число.
Отсчитываемый от нуля индекс колонки.
НомерСтроки (обязательный)
Тип: Число.
Отсчитываемый от нуля индекс строки.

Возвращаемое значение

Описание

Выполняет разъединение ячеек таблицы.

Примечание

Метод применим к таблице, не использующей свойство Таблица.ИсточникДанных (DataGridView.DataSource).

Важно: В системе отличной от семейства Windows данный метод неприменим.

Пример

Таблица1.РазъединитьЯчейки(1, 1);

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
