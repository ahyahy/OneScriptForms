СортировкаСвойств (PropertySort) Перечисление

Описание

Указывает, как свойства сортируются в СеткаСвойств (PropertyGrid).

Члены

Имя члена Описание Значение
БезСортировки (NoSort) Свойства не сортируются. 0
ПоАлфавиту (Alphabetical) Свойства сортируются в алфавитном порядке. 1
ПоКатегориям (Categorized) Свойства сортируются в соответствии с их категорией в группе. Категории определяются непосредственно из свойств. 2
ПоКатегориямПоАлфавиту (CategorizedAlphabetical) Свойства сортируются в соответствии с их категорией в группе. Дополнительные свойства сортируются в алфавитном порядке в пределах группы. Категории определяются непосредственно из свойств. 3
