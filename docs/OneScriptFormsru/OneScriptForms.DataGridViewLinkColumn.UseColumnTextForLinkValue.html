КолонкаСсылка.ИспользоватьТекстКакСсылку (DataGridViewLinkColumn.UseColumnTextForLinkValue) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, отображается ли значение свойства КолонкаСсылка.Текст (DataGridViewLinkColumn.Text) в качестве текста ссылки.

Значение Истина, если в качестве текста ссылки отображается значение КолонкаСсылка.Текст (DataGridViewLinkColumn.Text). Значение по умолчанию - Ложь.

Примечание

Пример

КолонкаСсылка1.ИспользоватьТекстКакСсылку = Истина;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Родитель = Форма1;
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;
Таблица1.АвтоНумерацияСтрок = Истина;

КолонкаСсылка1 = Ф.КолонкаСсылка();
Таблица1.Колонки.Добавить(КолонкаСсылка1);
Таблица1.Колонки.Добавить(Ф.КолонкаСсылка());
Таблица1.КоличествоСтрок = 3;

КолонкаСсылка1.Текст = "infostart.ru";
КолонкаСсылка1.ИспользоватьТекстКакСсылку = Истина;

Ф.ЗапуститьОбработкуСобытий();
