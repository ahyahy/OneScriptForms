КнопкаЯчейки.ПлоскийСтиль (DataGridViewButtonCell.FlatStyle) Свойство

Использование

Чтение и запись.

Значение

Тип: ПлоскийСтиль (FlatStyle).

Возвращает или задает стиль, определяющий представление кнопки.

Примечание

Важно: В системе отличной от семейства Windows данное свойство неприменимо.

Пример

КнопкаЯчейки.ПлоскийСтиль = Ф.ПлоскийСтиль.Всплывающий;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Родитель = Форма1;
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;
Таблица1.АвтоНумерацияСтрок = Истина;
Таблица1.ДобавлятьСтроки = Ложь;

КолонкаКнопка1 = Ф.КолонкаКнопка();
КолонкаКнопка1.ТекстЗаголовка = "Кол0";
Таблица1.Колонки.Добавить(КолонкаКнопка1);
Таблица1.Колонки.Добавить(Ф.КолонкаКнопка());
Таблица1.КоличествоСтрок = 2;
КолонкаКнопка1.Текст = "Кнопа";
КнопкаЯчейки = Таблица1.Ячейка(0, 0);

КнопкаЯчейки.ПлоскийСтиль = Ф.ПлоскийСтиль.Всплывающий;
