Форма.Владелец (Form.Owner) Свойство

Использование

Чтение и запись.

Значение

Тип: Форма (Form).

Возвращает или задает форму, владеющую этой формой.

Примечание

Если форма принадлежит другой форме, она закрывается или скрывается вместе с формой-владельцем.

Пример

Форма2.Владелец = Форма1;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Текст = "Форма примеров";
Форма1.Ширина = 400;
Форма1.Высота = 200;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Форма2 = Ф.Форма();
Форма2.Текст = "Вторая форма";
Форма2.Ширина = 300;
Форма2.Высота = 100;

Форма2.Владелец = Форма1;

Форма2.Отображать = Истина;
Форма2.Показать();
Форма2.Активизировать();

Сообщить("Форма2.Владелец.Текст = " + Форма2.Владелец.Текст); // Форма2.Владелец.Текст = Форма примеров

Ф.ЗапуститьОбработкуСобытий();
