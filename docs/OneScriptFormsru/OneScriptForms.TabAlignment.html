ВыравниваниеВкладок (TabAlignment) Перечисление

Описание

Задает расположение вкладок в наборе вкладок.

Члены

Имя члена Описание Значение
Верх (Top) Вкладки расположены по верхнему краю элемента управления. 0
Лево (Left) Вкладки расположены вдоль левого края элемента управления. 2
Низ (Bottom) Вкладки расположены по нижнему краю элемента управления. 1
Право (Right) Вкладки расположены по правому краю элемента управления. 3
