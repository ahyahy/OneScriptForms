Вкладки.Количество (TabPageCollection.Count) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает число страниц вкладок в коллекции.

Примечание

Пример

Сообщить("Вкладки1.Количество = " + Вкладки1.Количество);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ПанельВкладок1 = Форма1.ЭлементыУправления.Добавить(Ф.ПанельВкладок());
Вкладка1 = ПанельВкладок1.Вкладки.Добавить(Ф.Вкладка("Вкладка1"));
Вкладка2 = ПанельВкладок1.Вкладки.Добавить(Ф.Вкладка("Вкладка2"));
Вкладка3 = ПанельВкладок1.Вкладки.Добавить(Ф.Вкладка("Вкладка3"));

Вкладки1 = ПанельВкладок1.Вкладки;
Сообщить("Вкладки1.Количество = " + Вкладки1.Количество); // Вкладки1.Количество = 3

Ф.ЗапуститьОбработкуСобытий();
