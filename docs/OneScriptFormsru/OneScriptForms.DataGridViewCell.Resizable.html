ЯчейкаТаблицы.ИзменяемыйРазмер (DataGridViewCell.Resizable) Свойство

Использование

Только чтение.

Значение

Тип: Булево.

Возвращает значение, указывающее возможность изменения размеров ячейки.

Значение Истина, если размеры ячейки можно изменять; в противном случае - значение Ложь.

Значение свойства задается совместно родительской строкой и колонкой ячейки как результирующее значений их свойств ИзменяемыйРазмер (Resizable).

Примечание

Пример

Сообщить("" + Таблица1.Ячейка(0, 1).ИзменяемыйРазмер);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Родитель = Форма1;
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;
Таблица1.АвтоНумерацияСтрок = Истина;
Таблица1.ДобавлятьСтроки = Ложь;
Таблица1.КоличествоКолонок = 20;
Таблица1.КоличествоСтрок = 20;
Таблица1.АвтоШиринаЗаголовковСтрок = Ф.РежимШириныЗаголовковСтрок.ДляОтображаемых;

Сообщить("" + Таблица1.Ячейка(0, 1).ИзменяемыйРазмер);
Таблица1.Колонки(0).ИзменяемыйРазмер = Ф.ТриСостояния.Ложь;
Таблица1.Строки(1).ИзменяемыйРазмер = Ф.ТриСостояния.Ложь;
Сообщить("" + Таблица1.Ячейка(0, 1).ИзменяемыйРазмер);

Ф.ЗапуститьОбработкуСобытий();
