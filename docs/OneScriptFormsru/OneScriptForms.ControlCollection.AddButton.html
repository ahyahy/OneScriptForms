ЭлементыУправления.ДобавитьКнопку (ControlCollection.AddButton) Метод

Синтаксис

ДобавитьКнопку(<Текст>, <Лево>, <Верх>, <Ширина>, <Высота>)

Параметры

Текст (необязательный)
Тип: Строка.
Текст, связанный с элементом управления.
Лево (необязательный)
Тип: Число.
Верх (необязательный)
Тип: Число.
Ширина (необязательный)
Тип: Число.
Высота (необязательный)
Тип: Число.

Возвращаемое значение

Тип: Кнопка (Button).

Возвращает объект Кнопка (Button), представляющий новую кнопку.

Описание

Добавляет новый элемент управления Кнопка (Button) с указанным текстом, размером и местоположением.

Примечание

Пример

Кнопка1 = Форма1.ЭлементыУправления.ДобавитьКнопку("Сохранить файл", 0, 0, 100, 50);
