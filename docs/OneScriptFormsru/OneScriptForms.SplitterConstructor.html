Синтаксис

Разделитель()

Параметры

Описание

Создаёт новый экземпляр класса Разделитель (Splitter).

Примечание

При добавлении разделителя на форму необходимо соблюдать определенную последовательность добавления объектов и правильно указывать их свойство СтильГраницы (BorderStyle).

Если у нас есть панель на форме и мы хотим разбить её на три области с помощью вертикально расположенных разделителей, тогда последовательность действий должна быть такой:

  1. Добавить разделитель, задав свойству Стыковка значение Ф.СтильСтыковки.Лево.

  2. Добавить панель, задав свойству Стыковка значение Ф.СтильСтыковки.Лево.

  3. Добавить разделитель, задав свойству Стыковка значение Ф.СтильСтыковки.Лево.

  4. Добавить панель, задав свойству Стыковка значение Ф.СтильСтыковки.Лево.

  5. Добавить панель, задав свойству Стыковка значение Ф.СтильСтыковки.Заполнение.

Разберем подробнее. Подготовим форму с панелью, которую будем делить разделителями на три части.

Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Размер = Ф.Размер(300, 150);
Форма1.Отображать = Истина;

Панель0 = Форма1.ЭлементыУправления.Добавить(Ф.Панель());
Панель0.Размер = Ф.Размер(Форма1.КлиентШирина, Форма1.КлиентВысота);
Панель0.СтильГраницы = Ф.СтильГраницы.Трехмерная;

Добавим разделитель, задав свойству Стыковка значение Ф.СтильСтыковки.Лево.

Разделитель1 = Панель0.ЭлементыУправления.Добавить(Ф.Разделитель());
Разделитель1.Стыковка = Ф.СтильСтыковки.Лево;

Разделитель добавлен. Об этом свидетельствует изменение вида курсора при приближении к левому краю панели.

Добавим панель, задав свойству Стыковка значение Ф.СтильСтыковки.Лево. Для наглядности окрасим её в синий цвет.

Панель1 = Панель0.ЭлементыУправления.Добавить(Ф.Панель());
Панель1.СтильГраницы = Ф.СтильГраницы.Трехмерная;
Панель1.ЦветФона = Ф.Цвет().Синий;
Панель1.Стыковка = Ф.СтильСтыковки.Лево;

У нас есть синяя панель и справа к ней прикреплен разделитель. Об этом нам опять же говорит изменение вида курсора на правой границе синей панели. И ещё правее остается незаполненная пока область.

Добавим ещё один разделитель, задав свойству Стыковка значение Ф.СтильСтыковки.Лево.

Разделитель2 = Панель0.ЭлементыУправления.Добавить(Ф.Разделитель());
Разделитель2.Стыковка = Ф.СтильСтыковки.Лево;

Добавим ещё одну панель, задав свойству Стыковка значение Ф.СтильСтыковки.Лево. Окрасим её в красный цвет.

Панель2 = Панель0.ЭлементыУправления.Добавить(Ф.Панель());
Панель2.Ширина = 50;
Панель2.СтильГраницы = Ф.СтильГраницы.Трехмерная;
Панель2.ЦветФона = Ф.Цвет().Красный;
Панель2.Стыковка = Ф.СтильСтыковки.Лево;

Мы всё время добавляли объекты пристыковывая их слева. Они прикреплялись к левому краю вытесняя уже присутствующие объекты вправо. При этом справа оставалось незаполненное пространство. Заполнить его должна панель, добавленная последней и свойство Стыковка у неё должно быть установлено в значение Ф.СтильСтыковки.Заполнение. Добавим последнюю панель с зеленым цветом.

Панель3 = Панель0.ЭлементыУправления.Добавить(Ф.Панель());
Панель3.СтильГраницы = Ф.СтильГраницы.Трехмерная;
Панель3.ЦветФона = Ф.Цвет().Зеленый;
Панель3.Стыковка = Ф.СтильСтыковки.Заполнение;

В итоге на нашей форме мы получили панель-контейнер (Панель0), на которой находятся три панели (Панель1, Панель2, Панель3) "сшитые" разделителями.

Для областей расположенных сверху вниз, необходимо последовательно пристыковывать объекты к верхнему краю объекта-контейнера.

Комбинируя панели-контейнеры и помещая на них горизонтальные или вертикальные разделители и элементы управления (Форма, Дерево, СписокЭлементов, Панель и др.), можно получить большое количество вариантов оформления формы.

  • Есть ещё один способ добавления объектов, в том числе и разделителей, интуитивно более понятный, более удобный.

    • Можно добавлять объекты без эффекта "вытеснения", с пристыковкой к краю не родительского контейнера, а к краю уже добавленного на форму другого объекта. Для этого применяйте к объекту метод ЭлементУправления.НаПереднийПлан (Control.BringToFront) сразу после установки свойства Стыковка (Control.Dock).

Процедура УдалитьЭлементыУправления(ЭлементУправления)
	ЭлУпр = ЭлементУправления;
	Пока ЭлУпр.ЭлементыУправления.Количество > 0 Цикл
		Эл = ЭлУпр.ЭлементыУправления.Элемент(0);
		Если Эл.ЭлементыУправления.Количество > 0 Тогда
			УдалитьЭлементыУправления(Эл);
		КонецЕсли;
		ЭлУпр.ЭлементыУправления.Удалить(Эл);
		Эл.Освободить();
	КонецЦикла;
