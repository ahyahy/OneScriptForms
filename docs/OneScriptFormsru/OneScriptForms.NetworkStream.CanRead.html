ПотокСети.ВозможностьЧтения (NetworkStream.CanRead) Свойство

Использование

Только чтение.

Значение

Тип: Булево.

Возвращает значение, указывающее, поддерживает ли объект чтение.

Значение Истина, если данные могут быть прочитаны из потока, в противном случае - Ложь. Значение по умолчанию: Истина.

Примечание

Пример

Сообщить("ВозможностьЧтения = " + ПотокСети1.ВозможностьЧтения);
