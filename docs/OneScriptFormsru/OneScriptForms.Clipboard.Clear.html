БуферОбмена.Очистить (Clipboard.Clear) Метод

Синтаксис

Очистить()

Параметры

Возвращаемое значение

Описание

Удаляет все данные из буфера обмена.

Примечание

Пример

Буфер1.Очистить();
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Буфер1 = Ф.БуферОбмена();
Если Ф.ПлатформаWin Тогда
	Буфер1.УстановитьТекст("Текст для буфера обмена.");
Иначе
	Буфер1.УстановитьТекст("Text for the clipboard.");
КонецЕсли;
Буфер1.Очистить();
