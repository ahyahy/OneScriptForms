СтрокаТаблицы.Ячейки (DataGridViewRow.Cells) Метод

Синтаксис

Ячейки(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.
Отсчитываемый от нуля индекс ячейки.

Возвращаемое значение

Описание

Возвращает объект ЯчейкаТаблицы (DataGridViewCell) по указанному индексу.

Примечание

Пример

Таблица1.Строки(1).Ячейки(0).Значение = "Буратино";

Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Ширина = 500;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Родитель = Форма1;
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;

Таблица1.КоличествоКолонок = 3;
