СтильТаблицыСеткиДанных (DataGridTableStyle) Члены

Обзор СтильТаблицыСеткиДанных (DataGridTableStyle)

Конструктор

СтильТаблицыСеткиДанных (DataGridTableStyle) Конструктор Создаёт новый экземпляр класса СтильТаблицыСеткиДанных (DataGridTableStyle).

Свойства

ИмяОтображаемого (MappingName) Возвращает или задает имя, используемое для сопоставления этой таблицы с конкретным источником данных.
ОсновнойЦвет (ForeColor) Возвращает или задает основной цвет таблицы.
ОсновнойЦветЗаголовков (HeaderForeColor) Возвращает или задает основной цвет заголовков.
ОтображатьЗаголовкиСтолбцов (ColumnHeadersVisible) Возвращает или задает значение, показывающее, являются ли заголовки колонок видимыми.
ОтображатьЗаголовкиСтрок (RowHeadersVisible) Возвращает или задает значение, показывающее, являются ли заголовки строк видимыми.
ПредпочтительнаяВысотаСтрок (PreferredRowHeight) Возвращает или задает высоту, используемую для создания строк при отображении новой сетки.
ПредпочтительнаяШиринаСтолбцов (PreferredColumnWidth) Возвращает или задает ширину, используемую для создания колонок при отображении новой сетки.
РазрешитьСортировку (AllowSorting) Возвращает или задает значение, показывающее, разрешена ли сортировка в таблице.
СеткаДанных (DataGrid) Возвращает или задает элемент управления СеткаДанных (DataGrid) для нарисованной таблицы.
СтилиКолонкиСеткиДанных (GridColumnStyles) Возвращает коллекцию колонок, нарисованных для этой таблицы.
Тип (Type) (унаследовано от Компонент (Component)) Возвращает тип текущего экземпляра.
ТолькоЧтение (ReadOnly) Возвращает или задает значение, показывающее, разрешено ли редактирование колонок.
ЦветСетки (GridLineColor) Возвращает или задает цвет линий сетки.
ЦветФона (BackColor) Возвращает или задает цвет фона строк сетки с четными номерами.
ЦветФонаЗаголовков (HeaderBackColor) Возвращает или задает цвет фона заголовков.
ЦветФонаНечетныхСтрок (AlternatingBackColor) Возвращает или задает цвет фона строк сетки с нечетными номерами.
ШиринаЗаголовковСтрок (RowHeaderWidth) Возвращает или задает ширину заголовков строк.
ШрифтЗаголовков (HeaderFont) Возвращает или задает шрифт, используемый для названий заголовков.

Методы

Освободить (Dispose) (унаследовано от Компонент (Component)) Освобождает ресурсы, используемые Компонент (Component).
СтилиКолонкиСеткиДанных (GridColumnStyles) Возвращает значение по указанному индексу.
