ФорматированноеПолеВвода.Rtf (RichTextBox.Rtf) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает текст элемента управления ФорматированноеПолеВвода (RichTextBox), включая все коды в формате RTF.

Примечание

Важно: В системе отличной от семейства Windows данное свойство неприменимо.

Пример

ФорматированноеПолеВвода1.Rtf = Текст1; // Вернули исходный текст.
