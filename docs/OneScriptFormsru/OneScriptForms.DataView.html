ПредставлениеДанных (DataView) Класс

Описание

Представляет допускающее привязку данных, настраиваемое представление ТаблицаДанных (DataTable) для сортировки, фильтрации, поиска, изменения и навигации. ПредставлениеДанных (DataView) не сохраняет данные, а представляет связанное представление соответствующей ТаблицаДанных (DataTable). Изменения данных ПредставлениеДанных (DataView) повлияют на ТаблицаДанных (DataTable). Изменения данных ТаблицаДанных (DataTable) повлияют на все связанные с ним ПредставлениеДанных (DataView).

Список всех членов этого типа смотрите в разделе ПредставлениеДанных (DataView) Члены.
