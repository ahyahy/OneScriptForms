КоллекцияФорм.Количество (FormsCollection.Count) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает общее количество объектов в коллекции.

Примечание

Пример

Сообщить("Ф.КоллекцияФорм.Количество = " + Ф.КоллекцияФорм.Количество);
