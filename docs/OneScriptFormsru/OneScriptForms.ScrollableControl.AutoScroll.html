ПрокручиваемыйЭлементУправления.АвтоПрокрутка (ScrollableControl.AutoScroll) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, позволит ли контейнер пользователю прокручивать любые элементы управления, размещенные за пределами его видимых границ.

Примечание

Пример

Форма1.АвтоПрокрутка = Истина;

Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Сообщить("Форма1.АвтоПрокрутка = " + Форма1.АвтоПрокрутка); // Форма1.АвтоПрокрутка = Нет
