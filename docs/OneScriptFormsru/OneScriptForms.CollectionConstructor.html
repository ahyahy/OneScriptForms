Коллекция()

Параметры

Описание

Используйте метод ФормыДляОдноСкрипта.Коллекция (OneScriptForms.Collection) для создания нового экземпляра класса Коллекция (Collection).

Примечание

Пример

Коллекция1 = Ф.Коллекция();
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Коллекция1 = Ф.Коллекция();
Коллекция1.Добавить(Ф.Кнопка(), "Объект");
Коллекция1.Добавить("Подэлемент1", "Имя");
Коллекция1.Добавить(Истина, "Булево");
Коллекция1.Добавить(156.35, "Число");
Коллекция1.Добавить(Дата(1984,03,26,06,24,21), "Дата");
Сообщить("Коллекция1.Количество = " + Коллекция1.Количество); // Коллекция1.Количество = 5
