Словарь.Установить (Dictionary.Set) Метод

Синтаксис

Установить(<Ключ>, <Значение>)

Параметры

Ключ (обязательный)
Тип: Произвольный.
Ключ устанавливаемого нового значения.
Значение (обязательный)
Тип: Произвольный.
Устанавливаемое значение.

Возвращаемое значение

Описание

Задает значение записи в словаре.

Примечание

Пример

Словарь1.Установить("Число", 25.33);
