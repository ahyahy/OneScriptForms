Дерево.Прокручиваемый (TreeView.Scrollable) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, отображаются ли при необходимости полосы прокрутки в элементе управления Дерево (TreeView).

Примечание

Пример

Дерево1.Прокручиваемый = Истина;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Дерево1 = Форма1.ЭлементыУправления.Добавить(Ф.Дерево());
Для А = 1 По 25 Цикл
	Дерево1.Узлы.Добавить("Узел дерева №" + А);
КонецЦикла;

Дерево1.Прокручиваемый = Истина;
Сообщить("Дерево1.Прокручиваемый = " + Дерево1.Прокручиваемый); // Дерево1.Прокручиваемый = Да

Ф.ЗапуститьОбработкуСобытий();
