Обзор ХэшТаблица (HashTable)

Конструктор

ХэшТаблица (HashTable) Конструктор Используйте метод ФормыДляОдноСкрипта.ХэшТаблица (OneScriptForms.HashTable) для создания нового экземпляра класса ХэшТаблица (HashTable).

Свойства

Значения (Values) Возвращает МассивСписок (ArrayList), содержащий значения из коллекции ХэшТаблица (HashTable).
Ключи (Keys) Возвращает МассивСписок (ArrayList), содержащий ключи из коллекции ХэшТаблица (HashTable).
Количество (Count) Возвращает количество пар ключ-значение, содержащихся в хэш-таблице.

Методы

Добавить (Add) Добавляет элемент с указанным ключом и значением в хэш-таблицу.
Очистить (Clear) Удаляет все элементы из хэш-таблицы.
Удалить (Remove) Удаляет элемент с указанным ключом из хэш-таблицы.
Установить (Set) Задает значение записи в хэш-таблице.
Элемент (Item) Возвращает значение, связанное с указанным ключом.
