ГруппировкаТаблицы (DataGridViewGrouper) Класс

Описание

Представляет компонент, который добавляет функциональность группировки к любому существующему объекту Таблица (DataGridView).

Компонент применим, если таблица использует источник данных, то есть если значение свойства Таблица.ИсточникДанных (DataGridView.DataSource) задано.

Список всех членов этого типа смотрите в разделе ГруппировкаТаблицы (DataGridViewGrouper) Члены.
