Таблица.ИзменятьРазмерСтрок (DataGridView.AllowUserToResizeRows) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, может ли пользователь изменять размер строк.

Значение Истина, если могут изменяться размеры всех строк; в противном случае - значение Ложь. Значение по умолчанию - Истина.

Примечание

Пример

Таблица1.ИзменятьРазмерСтрок = Ложь;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Родитель = Форма1;
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;
Таблица1.ИсточникДанных = Неопределено; // Если до этого была привязка к источнику данных.
Таблица1.КоличествоКолонок = 2;
Таблица1.КоличествоСтрок = 2;

Сообщить("Таблица1.ИзменятьРазмерСтрок = " + Таблица1.ИзменятьРазмерСтрок);
Таблица1.ИзменятьРазмерСтрок = Ложь;
Сообщить("Таблица1.ИзменятьРазмерСтрок = " + Таблица1.ИзменятьРазмерСтрок);

Ф.ЗапуститьОбработкуСобытий();
