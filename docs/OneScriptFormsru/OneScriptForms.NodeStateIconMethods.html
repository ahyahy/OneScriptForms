ЗначокУзла (NodeStateIcon) Методы

Методы класса ЗначокУзла (NodeStateIcon) перечислены ниже. Полный список членов класса ЗначокУзла (NodeStateIcon) смотрите в разделе ЗначокУзла (NodeStateIcon) Члены.

Методы

ПолучитьЗначение (GetValue) Возвращает объект Картинка (Bitmap), который отображается для значка узла.
СброситьЗначение (ResetValue) Сбрасывает значение для значка узла, установленное методом ЗначокУзла.УстановитьЗначение (NodeStateIcon.SetValue) до значения, установленного свойством ЗначокУзла.Изображение (NodeStateIcon.Image) и ЗначокУзла.ИзображениеВыбранного (NodeStateIcon.SelectedImage).
УстановитьЗначение (SetValue) Задает значение для значка узла дерева значений.
