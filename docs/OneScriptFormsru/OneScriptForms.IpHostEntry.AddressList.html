IpУзел.Адреса (IpHostEntry.AddressList) Свойство

Использование

Чтение и запись.

Значение

Тип: МассивСписок (ArrayList).

Возвращает массив IP-адресов, связанных с узлом.

Примечание

Пример

МассивСписок1 = IpУзел1.Адреса;
МассивСписок1.Добавить(Ф.IpАдрес("127.0.0.1"));
МассивСписок1.Добавить(Ф.IpАдрес("127.0.0.2"));

IpУзел1.Адреса = МассивСписок1;
