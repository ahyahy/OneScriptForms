Курсор.Текущий (Cursor.Current) Свойство

Использование

Только чтение.

Значение

Тип: Курсор (Cursor).

Возвращает курсор, который представляет собой текущий курсор мыши.

Примечание

Пример

Курсор1 = Ф.Курсор();
Сообщить("Курсор1.Текущий.Позиция.Икс = " + Курсор1.Текущий.Позиция.Икс);
