Форма.Родитель (Form.Parent) Свойство

Использование

Чтение и запись.

Значение

Тип: Произвольный.

Возвращает или задает родительский контейнер элемента управления.

Примечание

Свойство можно использовать, если задано значение свойства Форма.Владелец (Form.Owner).

Важно: В системе отличной от семейства Windows данное свойство неприменимо.

Пример

Форма2.Родитель = Форма1;
Полный пример кода 

Копировать     Выделить всё


ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Текст = "Форма примеров";
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Кнопка1 = Ф.Кнопка();
Кнопка1.Родитель = Форма1;
Кнопка1.Текст = "Да";

Сообщить("Кнопка1.Родитель.Текст = " + Кнопка1.Родитель.Текст); // Кнопка1.Родитель.Текст = Форма примеров

Форма2 = Ф.Форма();
Форма2.Текст = "Вторая форма";
Форма2.Ширина = 300;
Форма2.Высота = 100;

Форма2.Владелец = Форма1;
Форма2.Родитель = Форма1;

Форма2.Отображать = Истина;
Форма2.Показать();
Форма2.Активизировать();

Сообщить("Форма2.Владелец.Текст = " + Форма2.Владелец.Текст); // Форма2.Владелец.Текст = Форма примеров
Сообщить("Форма2.Родитель.Текст = " + Форма2.Родитель.Текст); // Форма2.Родитель.Текст = Форма примеров
