Коллекция.Элемент (Collection.Item) Метод

Синтаксис

Элемент(<Ключ>)

Параметры

Ключ (обязательный)
Тип: Строка.
Значение ключа, указанное при добавлении в коллекцию члена, на который ссылается Ключ.

Возвращаемое значение

Тип: Произвольный.

Описание

Возвращает значение элемента по указанному ключу в коллекции.

Примечание

Пример

Сообщить("Элемент = " + Коллекция1.Элемент("Объект").Текст);
