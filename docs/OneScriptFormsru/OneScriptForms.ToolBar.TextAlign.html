ПанельИнструментов.ВыравниваниеТекста (ToolBar.TextAlign) Свойство

Использование

Чтение и запись.

Значение

Тип: ВыравниваниеТекстаВПанелиИнструментов (ToolBarTextAlign).

Возвращает или задает выравнивание текста относительно изображений, отображаемых на кнопках панели инструментов.

Примечание

Пример

ПанельИнструментов1.ВыравниваниеТекста = Ф.ВыравниваниеТекстаВПанелиИнструментов.Понизу;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ПанельИнструментов1 = Форма1.ЭлементыУправления.Добавить(Ф.ПанельИнструментов());
ПанельИнструментов1.АвтоРазмер = Истина;
ПанельИнструментов1.ВыравниваниеТекста = Ф.ВыравниваниеТекстаВПанелиИнструментов.Понизу;

Кнопки = ПанельИнструментов1.Кнопки;
Кнопка1 = Кнопки.Добавить(Ф.КнопкаПанелиИнструментов("Открыть"));
Кнопка2 = Кнопки.Добавить(Ф.КнопкаПанелиИнструментов("Сохранить"));
Кнопка3 = Кнопки.Добавить(Ф.КнопкаПанелиИнструментов("Справка"));
Кнопка4 = Кнопки.Добавить(Ф.КнопкаПанелиИнструментов("Выход"));

Ф.ЗапуститьОбработкуСобытий();
