ПолеКалендаря.ПоказатьФлажок (DateTimePicker.ShowCheckBox) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, отображается ли флажок слева от выбранной даты.

Истина - флажок отображается слева от выбранной даты, Ложь - в противном случае. Значение по умолчанию: Ложь.

Примечание

Если флажок установлен, можно обновить значения даты и времени. Если флажок снят, значения даты и времени изменить нельзя.

Пример

ПолеКалендаря1.ПоказатьФлажок = Истина;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ПолеКалендаря1 = Ф.ПолеКалендаря();
ПолеКалендаря1.Родитель = Форма1;

ПолеКалендаря1.ПоказатьРегулятор = Истина;
ПолеКалендаря1.ПоказатьФлажок = Истина;

Ф.ЗапуститьОбработкуСобытий();
