ЭлементУправления.ОбновитьСтили (Control.UpdateStyles) Метод

Синтаксис

ОбновитьСтили()

Параметры

Возвращаемое значение

Описание

Вызывает принудительное повторное применение назначенных стилей к элементу управления.

Примечание

Пример

Форма1.ОбновитьСтили();
