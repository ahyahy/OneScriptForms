СтрокиДанных.Количество (DataRowCollection.Count) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает общее число объектов СтрокаДанных (DataRow) в этой коллекции.

Примечание

Пример

Строки = ТаблицаДанных1.Строки;
