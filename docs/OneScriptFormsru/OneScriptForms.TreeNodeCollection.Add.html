УзлыДерева.Добавить (TreeNodeCollection.Add) Метод

Синтаксис

Добавить(<УзелДерева>)

Параметры

УзелДерева (обязательный)
Тип: Строка; УзелДерева (TreeNode).
Если строка, текст объекта УзелДерева (TreeNode), добавляемого в коллекцию. Иначе объект УзелДерева (TreeNode), добавляемый в коллекцию.

Возвращаемое значение

Тип: УзелДерева (TreeNode).

Объект УзелДерева (TreeNode), который был добавлен в коллекция.

Описание

Добавляет новый узел дерева с заданным текстом надписи в конец текущей коллекции узлов дерева.

Примечание

Пример

Узел1 = Дерево1.Узлы.Добавить(Ф.УзелДерева("Узел1"));
