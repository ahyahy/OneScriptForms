Обзор TCPКлиент (TCPClient)

Конструктор

TCPКлиент (TCPClient) Конструктор Создаёт новый экземпляр класса TCPКлиент (TCPClient).

Свойства

Кодировка (Encoding) Возвращает или задает кодировку текста. Значение по умолчанию Кодировка.UTF8 (Encoding.UTF8).
Подключен (Connected) Возвращает значение, указывающее, установлено ли подключение.

Методы

Закрыть (Close) Удаляет данный экземпляр TcpClient и запрашивает закрытие базового подключения TCP.
ОбработатьКлиентПолучилДанные (ProcessingClientReceived) Метод создает событие ФормыДляОдноСкрипта.КлиентПолучилДанные (OneScriptForms.ClientReceived).
Отправить (Send) Отправляет сообщение.
Подключить (Connect) Подключает клиента к удаленному TCP-узлу, используя указанный IP-адрес и номер порта.
ПолучитьПоток (GetStream) Возвращает объект ПотокСети (NetworkStream), используемый для отправки и получения данных.
