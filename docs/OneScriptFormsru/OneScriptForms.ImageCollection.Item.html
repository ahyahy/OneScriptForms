Изображения.Элемент (ImageCollection.Item) Метод

Синтаксис

Элемент(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.
Индекс изображения.

Возвращаемое значение

Тип: Картинка (Bitmap).

Изображение в списке, по указанному индексу.

Описание

Возвращает изображение в существующем СписокИзображений (ImageList).

Примечание

Пример

Сообщить("Изображения1.Элемент(0).Ширина = " + Изображения1.Элемент(0).Ширина);
