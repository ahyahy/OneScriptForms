СписокЭлементов.Выравнивание (ListView.Alignment) Свойство

Использование

Чтение и запись.

Значение

Тип: ВыравниваниеВСпискеЭлементов (ListViewAlignment).

Возвращает или задает выравнивание элементов в элементе управления.

Примечание

Свойство влияет на выравнивание только когда свойство СписокЭлементов.РежимОтображения (ListView.View) имеет значение БольшойЗначок (LargeIcon) или МаленькийЗначок (SmallIcon).

Пример

СписокЭлементов1.Выравнивание = Ф.ВыравниваниеВСпискеЭлементов.ПоСетке;

	Если Картинка1.ФорматПикселей = Ф.ФорматПикселей.Бит32АКЗС Тогда
		А5 = 0;

			М1.Установить(А5 + 0, Цв.ЗначениеСиний);
			М1.Установить(А5 + 1, Цв.ЗначениеЗеленый);
			М1.Установить(А5 + 2, Цв.ЗначениеКрасный);
			М1.Установить(А5 + 3, Цв.ЗначениеАльфа);
			А5 = А5 + 4;
		КонецЦикла;
	КонецЕсли;
	Картинка1.УстановитьБайты(АтрибутыКартинки1, М1);
	Картинка1.Разблокировать(АтрибутыКартинки1);
	СписокИзображений1.Изображения.Добавить(Картинка1);
	
	Элементы1.Добавить(Ф.ЭлементСпискаЭлементов(ИмяЦвета, (СписокИзображений1.Изображения.Количество - 1)));
КонецЦикла;
СписокЭлементов1.ВозобновитьРазмещение();
