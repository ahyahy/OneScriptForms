Обзор КолонкаДанных (DataColumn)

Конструктор

КолонкаДанных (DataColumn) Конструктор Создаёт новый экземпляр класса КолонкаДанных (DataColumn).

Свойства

АвтоПриращение (AutoIncrement) Возвращает или задает значение, указывающее, увеличивать ли автоматически значение колонки для новых строк, добавляемых в таблицу.
Заголовок (Caption) Возвращает или задает заголовок для колонки.
ЗначениеПоУмолчанию (DefaultValue) Возвращает или задает значение по умолчанию для колонки при создании новых строк.
ИмяКолонки (ColumnName) Возвращает или задает имя колонки в КолонкиДанных (DataColumnCollection).
НачальноеЧисло (AutoIncrementSeed) Возвращает или задает начальное значение для столбца, свойству которого АвтоПриращение (AutoIncrement) задано значение Истина.
Позиция (Ordinal) Возвращает позицию (начиная с нуля) колонки в коллекции КолонкиДанных (DataColumnCollection).
Таблица (Table) Возвращает ТаблицаДанных (DataTable), которой принадлежит колонка.
ТипДанных (DataType) Возвращает или задает тип данных, хранящихся в колонке.
ТолькоЧтение (ReadOnly) Возвращает или задает значение, указывающее, разрешено ли изменение колонки после добавления строки в таблицу.
ШагПриращения (AutoIncrementStep) Возвращает или задает шаг приращения для столбца, свойству которого АвтоПриращение (AutoIncrement) задано значение Истина.
