ЭлементыМеню.Элемент (MenuItemCollection.Item) Метод

Синтаксис

Элемент(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.
Индексированное местоположение ЭлементМеню (MenuItem) в коллекции.

Возвращаемое значение

Тип: ЭлементМеню (MenuItem).

Описание

Извлекает ЭлементМеню (MenuItem) по указанному индексу в коллекции.

Примечание

Пример

ЭлементМеню = Подменю_Файл.ЭлементМеню(0);
