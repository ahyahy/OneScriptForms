ЭлементСпискаЭлементов.ИндексИзображения (ListViewItem.ImageIndex) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает индекс изображения, отображаемого для данного элемента.

Примечание

Пример

Второй.ИндексИзображения = 0;
