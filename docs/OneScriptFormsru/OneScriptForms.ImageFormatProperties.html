ФорматИзображения (ImageFormat) Свойства

Свойства класса ФорматИзображения (ImageFormat) перечислены ниже. Полный список членов класса ФорматИзображения (ImageFormat) смотрите в разделе ФорматИзображения (ImageFormat) Члены.

Свойства

Bmp (Bmp) Возвращает формат изображения точечного рисунка (BMP).
Gif (Gif) Возвращает формат изображения графического обмена (GIF).
Icon (Icon) Возвращает формат изображения значков Windows.
Jpeg (Jpeg) Возвращает формат изображения jpeg (JPEG).
Png (Png) Возвращает формат изображения png (PNG).
