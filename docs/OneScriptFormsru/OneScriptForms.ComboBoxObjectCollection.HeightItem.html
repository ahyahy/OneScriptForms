ЭлементыПоляВыбора.ВысотаЭлемента (ComboBoxObjectCollection.HeightItem) Метод

Синтаксис

ВысотаЭлемента(<Индекс>, <Значение>)

Параметры

Индекс (обязательный)
Тип: Число.
Индекс элемента в коллекции, для которого возвращается или устанавливается высота в пикселях.
Значение (необязательный)
Тип: Число.
Высота в пикселях для элемента в коллекции.

Возвращаемое значение

Тип: Число.

Описание

Устанавливает и/или возвращает высоту элемента находящегося по указанному индексу в коллекции.

Примечание

Свойство ПолеВыбора.РежимРисования (ComboBox.DrawMode) должно быть установлено в значение ВручнуюПеременный (OwnerDrawVariable).

Пример

ПолеВыбораЭлементы1.ВысотаЭлемента(0, 30);
