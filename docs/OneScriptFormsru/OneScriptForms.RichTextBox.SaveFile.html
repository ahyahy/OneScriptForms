ФорматированноеПолеВвода.СохранитьФайл (RichTextBox.SaveFile) Метод

Синтаксис

СохранитьФайл(<Путь>, <ТипФайла>)

Параметры

Путь (обязательный)
Тип: Строка.
ТипФайла (необязательный)
Тип: ФорматированноеПолеВводаТипыПотоков (RichTextBoxStreamType).

Возвращаемое значение

Описание

Сохраняет содержимое ФорматированноеПолеВвода (RichTextBox) в файл определенного типа.

Примечание

Пример

ИмяВременногоФайла = СтрЗаменить(ПолучитьИмяВременногоФайла(), ".", "") + ".Rtf";
ФорматированноеПолеВвода1.СохранитьФайл(ИмяВременногоФайла, Ф.ФорматированноеПолеВводаТипыПотоков.Форматированный);
