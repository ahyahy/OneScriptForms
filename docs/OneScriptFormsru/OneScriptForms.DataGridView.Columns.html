Таблица.Колонки (DataGridView.Columns) Свойство

Использование

Только чтение.

Значение

Тип: КолонкиТаблицы (DataGridViewColumnCollection).

Возвращает коллекцию, содержащую все колонки элемента управления.

Примечание

Пример

Колонки1 = Таблица1.Колонки;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Родитель = Форма1;
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;

Колонки1 = Таблица1.Колонки;

КолонкаПолеВвода1 = Ф.КолонкаПолеВвода();
Колонки1.Добавить(КолонкаПолеВвода1);

КолонкаКартинка1 = Ф.КолонкаКартинка();
Колонки1.Добавить(КолонкаКартинка1);

КолонкаКнопка1 = Ф.КолонкаКнопка();
Колонки1.Добавить(КолонкаКнопка1);

КолонкаПолеВыбора1 = Ф.КолонкаПолеВыбора();
Колонки1.Добавить(КолонкаПолеВыбора1);

КолонкаСсылка1 = Ф.КолонкаСсылка();
Колонки1.Добавить(КолонкаСсылка1);

КолонкаФлажок1 = Ф.КолонкаФлажок();
