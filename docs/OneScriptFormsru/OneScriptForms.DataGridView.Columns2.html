Таблица.Колонки (DataGridView.Columns) Метод

Синтаксис

Колонки(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.
Отсчитываемый от нуля индекс колонки.

Возвращаемое значение

Тип: КолонкаТаблицы (DataGridViewColumn).

Описание

Возвращает объект КолонкаТаблицы (DataGridViewColumn) по указанному индексу.

Примечание

Пример

Таблица1.Колонки(0).ЗаголовокКолонки.Значение = "Колонка1";
