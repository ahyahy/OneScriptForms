Обзор ЭлементСпискаЭлементов (ListViewItem)

Конструктор

ЭлементСпискаЭлементов (ListViewItem) Конструктор Используйте метод ФормыДляОдноСкрипта.ЭлементСпискаЭлементов (OneScriptForms.ListViewItem) для создания нового экземпляра класса ЭлементСпискаЭлементов (ListViewItem).

Свойства

Выбран (Selected) Возвращает или задает значение, указывающее, является ли элемент выбраным.
Границы (Bounds) Возвращает прямоугольник, ограничивающий элемент, включая его подэлементы.
Индекс (Index) Возвращает отсчитываемый от нуля индекс элемента в пределах элемента управления СписокЭлементов (ListView).
ИндексИзображения (ImageIndex) Возвращает или задает индекс изображения, отображаемого для данного элемента.
ИспользоватьСтильДляПодэлементов (UseItemStyleForSubItems) Возвращает или задает значение, указывающее, будут ли шрифты, свойства ОсновнойЦвет (ForeColor) и ЦветФона (BackColor) для элемента использоваться для всех его подэлементов.
Метка (Tag) Возвращает объект, содержащий дополнительные данные.
ОсновнойЦвет (ForeColor) Возвращает или задает цвет текста элемента.
Подэлементы (SubItems) Возвращает коллекцию, содержащую все подэлементы элемента.
Помечен (Checked) Возвращает или задает значение, указывающее, является ли элемент помеченным.
СписокИзображений (ImageList) Возвращает СписокИзображений (ImageList) содержащий изображение, отображаемое с элементом.
Сфокусирован (Focused) Возвращает или задает значение, указывающее, имеет ли объект фокус в элементе управления СписокЭлементов (ListView).
Текст (Text) Возвращает или задает текст элемента.
ЦветФона (BackColor) Возвращает или задает цвет фона текста элемента.
Шрифт (Font) Возвращает или задает шрифт текста, отображаемого элементом.

Методы

НачатьРедактирование (BeginEdit) Помещает текст элемента в режим редактирования.
ОбеспечитьОтображение (EnsureVisible) Гарантирует видимость элемента списка элементов выполнением прокрутки содержимого элемента управления.
Подэлементы (SubItems) Возвращает значение по указанному индексу.
Удалить (Remove) Удаляет элемент из связанного с ним элемента управления СписокЭлементов (ListView).
