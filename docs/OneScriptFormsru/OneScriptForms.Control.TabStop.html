ЭлементУправления.ТабФокус (Control.TabStop) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, может ли пользователь перевести фокус на данный элемент управления при помощи клавиши TAB.

Примечание

Пример

Кнопка4.ТабФокус = Истина;
Полный пример кода 

Копировать     Выделить всё


Перем Ф, Кнопка1;
