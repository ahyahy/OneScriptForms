КолонкиДанных.Удалить (DataColumnCollection.Remove) Метод

Синтаксис

Удалить(<КолонкаДанных>)

Параметры

КолонкаДанных (обязательный)
Тип: КолонкаДанных (DataColumn).

Возвращаемое значение

Описание

Удаляет объект КолонкаДанных (DataColumn) из коллекции.

Примечание

Пример

Колонки1.Удалить(Колонка2);

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ТаблицаДанных1 = Ф.ТаблицаДанных();

Колонка1.ИмяКолонки = "№ н/п";
