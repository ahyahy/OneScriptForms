Календарь.ЕжемесячныеДаты (MonthCalendar.MonthlyBoldedDates) Метод

Синтаксис

ЕжемесячныеДаты(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.
Отсчитываемый от нуля индекс получаемого элемента.

Возвращаемое значение

Тип: Дата.

Значение по указанному индексу.

Описание

Возвращает значение по указанному индексу.

Примечание

Пример

Дата1 = Календарь1.ЕжемесячныеДаты(0);
