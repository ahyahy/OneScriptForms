ФлагиМыши (MouseFlags) Перечисление

Описание

Задает константы, перечисляющие действия мыши.

В системе отличной от семейства Windows в одном методе применимы только комбинации значений перечисленния 1, 2, 4 ,6, 8, 16, 24, 32, 64, 96.

Так же можно достичь желаемого результата задавая значения перечисления в нескольких следующих друг за другом методах. Имеется в виду применение несколько раз метода ФормыДляОдноСкрипта.НажатьКнопкуМыши (OneScriptForms.MouseKeyPress).

Члены

Имя члена Описание Значение
Абсолютно (Absolute) Если задано значение Абсолютно (Absolute), то координаты x и y содержат нормализованные абсолютные координаты в диапазоне от 0 до 65535. Процедура события отображает эти координаты на поверхность дисплея. Координаты (0,0) отображаются в верхнем левом углу поверхности дисплея, (65535,65535) - в нижнем правом углу. Если значение Абсолютно (Absolute) не задано, то координаты x и y задают относительные движения с момента создания последнего события мыши (последняя сообщенная позиция). Положительные значения означают, что мышь переместилась вправо (или вниз); отрицательные значения означают, что мышь переместилась влево (или вверх). В системе отличной от семейства Windows данное значение не применяется. 32768
ЛеваяВверх (LeftUp) Отпустить левую кнопку мыши. 4
ЛеваяВниз (LeftDown) Нажать левую кнопку мыши. 2
Переместить (Move) Переместить мышь. 1
ПраваяВверх (RightUp) Отпустить правую кнопку мыши. 16
ПраваяВниз (RightDown) Нажать правую кнопку мыши. 8
СредняяВверх (MiddleUp) Отпустить среднюю кнопку мыши. 64
СредняяВниз (MiddleDown) Нажать среднюю кнопку мыши. 32
