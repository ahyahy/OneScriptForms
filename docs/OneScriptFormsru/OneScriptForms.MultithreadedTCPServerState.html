СостояниеМногопоточногоTCPСервера (MultithreadedTCPServerState) Перечисление

Описание

Указывает состояние TCP сервера.

Члены

Имя члена Описание Значение
Запускается (Starting) Сервер запускается. 1
Запущен (Running) Сервер запущен. 2
Останавливается (Stopping) Сервер останавливается. 3
Остановлен (Stopped) Сервер остановлен пользователем. 0
ОстановленСОшибкой (StoppedWithError) Сервер остановлен из за ошибки. 4
