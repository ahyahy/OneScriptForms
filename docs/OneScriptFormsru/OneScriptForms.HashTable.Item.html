ХэшТаблица.Элемент (HashTable.Item) Метод

Синтаксис

Элемент(<Ключ>)

Параметры

Ключ (обязательный)
Тип: Произвольный.
Ключ, значение которого нужно получить.

Возвращаемое значение

Тип: Произвольный.

Значение, связанное с указанным ключом. Если указанный ключ не найден, возвращается Неопределено.

Описание

Возвращает значение, связанное с указанным ключом.

Примечание

Пример

Сообщить("ХэшТаблица.Элемент = " + ХэшТаблица.Элемент("Дата"));
