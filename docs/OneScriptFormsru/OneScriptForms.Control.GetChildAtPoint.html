ЭлементУправления.ДочернийПоКоординатам (Control.GetChildAtPoint) Метод

Синтаксис

ДочернийПоКоординатам(<ТочкаПоиска>)

Параметры

ТочкаПоиска (обязательный)
Тип: Точка (Point).

Возвращаемое значение

Тип: Произвольный.

Описание

Возвращает дочерний элемент управления, имеющий указанные координаты.

Примечание

Если в указанной точке нет дочернего элемента управления, вернется Неопределено.

Пример

ТочкаПоиска = Ф.Точка(11, 11);
Объект = Форма1.ДочернийПоКоординатам(ТочкаПоиска);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Кнопка1 = Ф.Кнопка();
Кнопка1.Родитель = Форма1;
Кнопка1.Текст = "Кнопка1";
Кнопка1.Положение = Ф.Точка(10, 10);
ТочкаПоиска = Ф.Точка(11, 11);

Объект = Форма1.ДочернийПоКоординатам(ТочкаПоиска);
Сообщить("Объект.Текст = " + Объект.Текст); // Объект.Текст = Кнопка1
