Поток.ВозможностьЗаписи (Stream.CanWrite) Свойство

Использование

Только чтение.

Значение

Тип: Булево.

Возвращает значение, которое показывает, поддерживает ли текущий поток возможность записи.

Истина - поток поддерживает запись, Ложь - в противном случае.

Примечание

Пример

Поток1 = Ф.Поток();
Сообщить("Поток1.ВозможностьЗаписи = " + Поток1.ВозможностьЗаписи); // Поток1.ВозможностьЗаписи = Да
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Поток1 = Ф.Поток();
Сообщить("Поток1.ВозможностьЗаписи = " + Поток1.ВозможностьЗаписи); // Поток1.ВозможностьЗаписи = Да

Ф.ЗапуститьОбработкуСобытий();
