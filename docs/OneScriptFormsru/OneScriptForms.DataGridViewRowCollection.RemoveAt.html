СтрокиТаблицы.УдалитьПоИндексу (DataGridViewRowCollection.RemoveAt) Метод

Синтаксис

УдалитьПоИндексу(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.
Позиция удаляемой строки.

Возвращаемое значение

Описание

Удаляет строку в указанной позиции из коллекции.

Примечание

Пример

Таблица1.Строки.УдалитьПоИндексу(0);
