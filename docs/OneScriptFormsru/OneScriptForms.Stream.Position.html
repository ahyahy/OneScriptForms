Поток.Позиция (Stream.Position) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает позицию в текущем потоке.

Примечание

Позиция не должна ни на байт превышать величину потока.

Пример

Поток1.Позиция = 3;
Сообщить("Поток1.Позиция = " + Поток1.Позиция);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Поток1 = Ф.Поток();
СтрБуфер = "Съешь [же] ещё этих мягких французских булок да выпей чаю.";
Буфер = Ф.Кодировка().ПолучитьБайты(СтрБуфер);

Поток1.Записать(Буфер, 20, 18);
Поток1.Позиция = 3;
Сообщить("Поток1.Позиция = " + Поток1.Позиция); // Поток1.Позиция = 3
