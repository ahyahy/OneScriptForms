КлавишаАрг.Ctrl (KeyEventArgs.Control) Свойство

Использование

Только чтение.

Значение

Тип: Булево.

Возвращает значение, указывающее, была ли нажата клавиша Ctrl.

Истина - нажата клавиша Ctrl, Ложь - в противном случае.
