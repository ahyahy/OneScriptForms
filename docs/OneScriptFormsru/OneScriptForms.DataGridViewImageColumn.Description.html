КолонкаКартинка.Описание (DataGridViewImageColumn.Description) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает строку, описывающую изображение колонки.

Задание этого свойства также задает свойство КартинкаЯчейки.Описание (DataGridViewImageCell.Description) для каждой ячейки в колонке. Чтобы переопределить указанное значение для отдельных ячеек, задайте значения ячеек после установки значения колонки.

Примечание

Пример

КолонкаКартинка1.Описание = "Масленица";
