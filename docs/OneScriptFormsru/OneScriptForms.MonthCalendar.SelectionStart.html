Календарь.НачальнаяДата (MonthCalendar.SelectionStart) Свойство

Использование

Чтение и запись.

Значение

Тип: Дата.

Возвращает или задает начальную дату для выбранного диапазона дат.

Примечание

Пример

НачальнаяДата1 = Календарь1.НачальнаяДата;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Календарь1 = Ф.Календарь();
Календарь1.Родитель = Форма1;
НачальнаяДата1 = Календарь1.НачальнаяДата;
Сообщить("Старая НачальнаяДата = " + НачальнаяДата1);
НачальнаяДата1 = НачальнаяДата1 - 86400; //отняли один день в секундах
Сообщить("Новая НачальнаяДата = " + НачальнаяДата1);

Ф.ЗапуститьОбработкуСобытий();
