СтрокаТаблицы.Выбран (DataGridViewRow.Selected) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, выбрана ли строка.

Значение Истина, если строка выбрана; в противном случае - значение Ложь.

Примечание

Пример

Таблица1.Строки(5).Выбран = Истина;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Ширина = 500;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Родитель = Форма1;
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;
Таблица1.АвтоШиринаЗаголовковСтрок = Ф.РежимШириныЗаголовковСтрок.ДляОтображаемых;

Для А = 0 По 20 Цикл
	Таблица1.Колонки.Добавить(Ф.КолонкаПолеВвода());
	Таблица1.Колонки(А).ТекстЗаголовка = "Кол" + А;
КонецЦикла;

Для А = 0 По 20 Цикл
	Таблица1.Строки.Добавить();
	Таблица1.Строки(А).ЗаголовокСтроки.Значение = "Стр" + А;
КонецЦикла;

Таблица1.Строки(5).Выбран = Истина;
Таблица1.Строки(6).Выбран = Истина;

Ф.ЗапуститьОбработкуСобытий();
