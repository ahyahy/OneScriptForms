СтрокиТаблицы (DataGridViewRowCollection) Члены

Обзор СтрокиТаблицы (DataGridViewRowCollection)

Свойства

Количество (Count) Возвращает количество строк в коллекции.

Методы

Вставить (Insert) Вставляет заданный объект СтрокаТаблицы (DataGridViewRow) в коллекцию.
Добавить (Add) Добавляет новую строку или указанный параметр СтрокаТаблицы в коллекцию.
ДобавитьКопию (AddCopy) Добавляет новую строку, основанную на строке по указанному индексу.
Индекс (IndexOf) Возвращает индекс указанного элемента в коллекции.
Очистить (Clear) Очищает коллекцию.
Содержит (Contains) Определяет, присутствует ли в коллекции указанный объект СтрокаТаблицы (DataGridViewRow).
Удалить (Remove) Удаляет строку из коллекции.
УдалитьПоИндексу (RemoveAt) Удаляет строку в указанной позиции из коллекции.
Элемент (Item) Возвращает объект СтрокаТаблицы (DataGridViewRow) по указанному индексу.
