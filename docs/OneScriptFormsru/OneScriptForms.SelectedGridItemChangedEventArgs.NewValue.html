ВыбранныйЭлементСеткиИзмененАрг.НовоеЗначение (SelectedGridItemChangedEventArgs.NewValue) Свойство

Использование

Только чтение.

Значение

Тип: Произвольный.

Возвращает значение вновь выбранного объекта ЭлементСетки (GridItem).

Примечание

Важно: В системе отличной от семейства Windows данное свойство неприменимо.

Пример

Аргументы = Ф.ВыбранныйЭлементСеткиИзмененАрг();
Сообщить("Аргументы.НовоеЗначение = " + Аргументы.НовоеЗначение);

Процедура СеткаСвойств1_ВыбранныйЭлементСеткиИзменен() Экспорт
	Аргументы = Ф.ВыбранныйЭлементСеткиИзмененАрг();
	Сообщить("Аргументы.НоваяНадпись = " + Аргументы.НоваяНадпись);
