Меню.ЭлементМеню (Menu.MenuItem) Метод

Синтаксис

ЭлементМеню(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.
Отсчитываемый от нуля индекс извлекаемого объекта ЭлементМеню (MenuItem).

Возвращаемое значение

Тип: ЭлементМеню (MenuItem).

Описание

Возвращает ЭлементМеню (MenuItem) по указанному индексу в коллекции.

Примечание

Пример

Сообщить("ГлавноеМеню1.ЭлементМеню(0).Текст = " + ГлавноеМеню1.ЭлементМеню(0).Текст); // ГлавноеМеню1.ЭлементМеню(0).Текст = Файл
