Описание

Определяет значения для указания способа, которым регулируется высота заголовков колонок в элементе управления Таблица (DataGridView).

Члены

Имя члена Описание Значение
АвтоРазмер (AutoSize) Высота заголовка колонки настраивается так, чтобы она соответствовала содержимому всех ячеек заголовков колонок. 2
Включить (EnableResizing) Пользователи могут настраивать высоту заголовка колонки с помощью мыши. 0
Отключить (DisableResizing) Пользователи не могут изменить высоту заголовка колонки с помощью мыши. 1
