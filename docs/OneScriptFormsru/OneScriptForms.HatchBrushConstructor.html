ПрямоугольнаяКисть (HatchBrush) Конструктор

Синтаксис

ПрямоугольнаяКисть(<СтильШтриховки>, <ОсновнойЦвет>, <ЦветФона>)

Параметры

СтильШтриховки (обязательный)
Тип: СтильШтриховки (HatchStyle).
ОсновнойЦвет (обязательный)
Тип: Цвет (Color).
ЦветФона (необязательный)
Тип: Цвет (Color).

Описание

Создаёт новый экземпляр класса ПрямоугольнаяКисть (HatchBrush).

Примечание

Цвет фона инициализируется черным цветом.

Пример

ПрямоугольнаяКисть1 = Ф.ПрямоугольнаяКисть(Ф.СтильШтриховки.Мешковина, Ф.Цвет().Белый, Ф.Цвет().Красный);

	Графика1.ЗаполнитьПрямоугольник(ПрямоугольнаяКисть1, 10, 10, 100, 100);
