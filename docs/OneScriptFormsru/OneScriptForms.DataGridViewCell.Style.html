ЯчейкаТаблицы.Стиль (DataGridViewCell.Style) Свойство

Использование

Чтение и запись.

Значение

Тип: СтильЯчейки (DataGridViewCellStyle).

Возвращает или задает стиль ячейки.

Примечание

Пример

Ячейка.Стиль = СтильЯчейки1;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Родитель = Форма1;
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;
Таблица1.КоличествоКолонок = 20;
Таблица1.КоличествоСтрок = 7;

Ячейка = Таблица1.Ячейка(0, 0);
СтильЯчейки1 = Ф.СтильЯчейки();
СтильЯчейки1.Выравнивание = Ф.ВыравниваниеСодержимогоЯчейки.ВерхПраво;
СтильЯчейки1.ЦветФона = Ф.Цвет().Бирюзовый;

Ячейка.Значение = "Буратно";
Ячейка.Стиль = СтильЯчейки1;

Таблица1.ТекущаяЯчейка = Таблица1.Ячейка(0, 1);

Ф.ЗапуститьОбработкуСобытий();
