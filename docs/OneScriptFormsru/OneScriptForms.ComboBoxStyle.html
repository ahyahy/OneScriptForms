СтильПоляВыбора (ComboBoxStyle) Перечисление

Описание

Указывает стиль элемента управления ПолеВыбора (ComboBox).

Члены

Имя члена Описание Значение
НеРедактируемый (DropDownList) Нажав кнопку со стрелкой вниз получаем выпадающий список, текстовая часть которого не является редактируемой. Это означает, что пользователь не может ввести новое значение. Можно выбрать только значение из списка. 2
Простой (Simple) Выпадающий список отображается всегда, а текстовая часть является редактируемой. Это означает, что пользователь может ввести новое значение, а не ограничиваться выбором из существующих значений в списке. 0
Редактируемый (DropDown) Нажав кнопку со стрелкой вниз получаем выпадающий список, текстовая часть которого является редактируемой. Это означает, что пользователь может ввести новое значение, а не ограничиваться выбором из существующих значений в списке. Это стиль по умолчанию. 1
