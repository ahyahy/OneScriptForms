ВводОтклоненАрг.Причина (MaskInputRejectedEventArgs.RejectionHint) Свойство

Использование

Только чтение.

Значение

Тип: РезультатМаски (MaskedTextResultHint).

Возвращает значение перечисления с описанием причины, по которой введенный символ считается недопустимым.

	Сообщить("ВводОтклоненАрг1.Причина = " + ВводОтклоненАрг1.Причина);
