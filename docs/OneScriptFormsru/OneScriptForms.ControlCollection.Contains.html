ЭлементыУправления.Содержит (ControlCollection.Contains) Метод

Синтаксис

Содержит(<ЭлементУправления>)

Параметры

ЭлементУправления (обязательный)
Тип: Произвольный.
Элемент управления для поиска в коллекции.

Возвращаемое значение

Тип: Булево.

Истина - указанный элемент управления существует в коллекции, Ложь - в противном случае.

Описание

Определяет, является ли указанный элемент управления членом коллекции.

Примечание

Пример

Сообщить("Содержит = " + Форма1.ЭлементыУправления.Содержит(Кнопка2));
