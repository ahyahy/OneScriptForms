ХэшТаблица.Установить (HashTable.Set) Метод

Синтаксис

Установить(<Ключ>, <Значение>)

Параметры

Ключ (обязательный)
Тип: Произвольный.
Ключ устанавливаемого нового значения.
Значение (обязательный)
Тип: Произвольный.
Устанавливаемое значение.

Возвращаемое значение

Описание

Задает значение записи в хэш-таблице.

Примечание

Пример

ХэшТаблица.Установить (10.10, 218.6);
ХэшТаблица.Установить ("Строка2", "Строка в ХэшТаблица");
