ФормыДляОдноСкрипта.СеткаСвойств (OneScriptForms.PropertyGrid) Метод

Синтаксис

СеткаСвойств()

Параметры

Возвращаемое значение

Тип: СеткаСвойств (PropertyGrid).

Описание

Создаёт новый экземпляр класса СеткаСвойств (PropertyGrid).

Примечание

Важно: В системе отличной от семейства Windows данный метод неприменим.

Пример

СеткаСвойств1 = Ф.СеткаСвойств();
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Ширина = 900;
Форма1.Высота = 400;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

СеткаСвойств1 = Ф.СеткаСвойств();
