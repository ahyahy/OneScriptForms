СеткаСвойств.ВыбранныйОбъект (PropertyGrid.SelectedObject) Свойство

Использование

Чтение и запись.

Значение

Тип: Произвольный.

Возвращает или задает объект, свойства которого отображает данная сетка.

Первый объект в списке объектов. Если текущий объект не выбран возвращается Неопределено.

Примечание

Свойство ВыбранныйОбъект устанавливает один объект в сетке для просмотра. При нескольких выбранных, это свойство возвращает первый из них. Если объекты не выбраны, возвращается Неопределено.

Важно: В системе отличной от семейства Windows данное свойство неприменимо.

Пример

СеткаСвойств1.ВыбранныйОбъект = Форма1;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Ширина = 900;
Форма1.Высота = 400;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

СеткаСвойств1 = Ф.СеткаСвойств();
СеткаСвойств1.Родитель = Форма1;
СеткаСвойств1.ВыбранныйОбъект = Форма1;
СеткаСвойств1.Стыковка = Ф.СтильСтыковки.Заполнение;

Ф.ЗапуститьОбработкуСобытий();
