Конструктор

БуферОбмена (Clipboard) Конструктор Создаёт новый экземпляр класса БуферОбмена (Clipboard).

Методы

Очистить (Clear) Удаляет все данные из буфера обмена.
ПолучитьИзображение (GetImage) Возвращает изображение из буфера обмена.
ПолучитьТекст (GetText) Извлекает текстовые данные из буфера обмена в формате UnicodeText.
СодержитДанные (ContainsData) Возвращает значение, указывающее, содержатся ли в данный момент в системном буфере обмена данные.
СодержитЮникод (ContainsUnicode) Возвращает значение, указывающее, имеются ли в буфере обмена текстовые данные в формате Юникод.
УстановитьИзображение (SetImage) Очищает буфер обмена и добавляет изображение в формате Картинка (Bitmap).
УстановитьТекст (SetText) Очищает буфер обмена и добавляет текстовые данные в формате UnicodeText.
