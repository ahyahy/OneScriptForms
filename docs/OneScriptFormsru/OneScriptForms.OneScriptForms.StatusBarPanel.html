ФормыДляОдноСкрипта.ПанельСтрокиСостояния (OneScriptForms.StatusBarPanel) Метод

Синтаксис

ПанельСтрокиСостояния()

Параметры

Возвращаемое значение

Тип: ПанельСтрокиСостояния (StatusBarPanel).

Описание

Создаёт новый экземпляр класса ПанельСтрокиСостояния (StatusBarPanel).

Примечание

Пример

Панель1 = Ф.ПанельСтрокиСостояния();
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

СтрокаСостояния1 = Ф.СтрокаСостояния();
СтрокаСостояния1.Родитель = Форма1;
СтрокаСостояния1.ПоказатьПанели = Истина;
Панели1 = СтрокаСостояния1.Панели;

Панель1 = Ф.ПанельСтрокиСостояния();
Панели1.Добавить(Панель1);
Панель1.Текст = "Панель1";
Панель1.СтильГраницы = Ф.СтильГраницыПанелиСтрокиСостояния.Рельефная;
Панель1.АвтоРазмер = Ф.АвтоРазмерПанелиСтрокиСостояния.Растяжимое;
Панель1.МинимальнаяШирина = 100;
