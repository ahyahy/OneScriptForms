Обзор ЗначокУзла (NodeStateIcon)

Конструктор

ЗначокУзла (NodeStateIcon) Конструктор Создаёт новый экземпляр класса ЗначокУзла (NodeStateIcon).

Свойства

Изображение (Image) Возвращает или задает изображение, отображаемое для объекта ЗначокУзла (NodeStateIcon).
ИзображениеВыбранного (SelectedImage) Возвращает или задает изображение, отображаемое для выбранного объекта ЗначокУзла (NodeStateIcon).
Колонка (ParentColumn) Объект КолонкаДереваЗначений (TreeColumn), которому принадлежит значок узла.
ЛевыйОтступ (LeftMargin) Возвращает или задает размер левого отступа.
РежимМасштабирования (ScaleMode) Указывает способ отображения изображения.

Методы

ПолучитьЗначение (GetValue) Возвращает объект Картинка (Bitmap), который отображается для значка узла.
СброситьЗначение (ResetValue) Сбрасывает значение для значка узла, установленное методом ЗначокУзла.УстановитьЗначение (NodeStateIcon.SetValue) до значения, установленного свойством ЗначокУзла.Изображение (NodeStateIcon.Image) и ЗначокУзла.ИзображениеВыбранного (NodeStateIcon.SelectedImage).
УстановитьЗначение (SetValue) Задает значение для значка узла дерева значений.
