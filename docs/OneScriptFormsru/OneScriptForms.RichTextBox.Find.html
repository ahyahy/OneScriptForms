ФорматированноеПолеВвода.Поиск (RichTextBox.Find) Метод

Синтаксис

Поиск(<ИскомаяСтрока>, <Начало>, <СпособПоиска>)

Параметры

ИскомаяСтрока (обязательный)
Тип: Строка.
Текст для поиска.
Начало (необязательный)
Тип: Число.
Расположение в тексте элемента управления, с которого начинается поиск.
СпособПоиска (необязательный)
Тип: ФорматированноеПолеВводаПоиск (RichTextBoxFinds).
Поразрядное сочетание значений перечисления ФорматированноеПолеВводаПоиск (RichTextBoxFinds).

Возвращаемое значение

Тип: Число.

Местоположение внутри элемента управления, в котором был найден текст поиска, или -1, если строка поиска не найдена или в параметре Строка указана пустая строка.

Описание

Выполняет поиск строки в тексте элемента управления ФорматированноеПолеВвода (RichTextBox).

Примечание

Пример

Позиция = ФорматированноеПолеВвода1.Поиск(Стр2, 0, Ф.ФорматированноеПолеВводаПоиск.УчитыватьРегистр + Ф.ФорматированноеПолеВводаПоиск.НеВыделять);
