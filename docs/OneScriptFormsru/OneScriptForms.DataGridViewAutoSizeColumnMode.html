РежимАвтоРазмераКолонки (DataGridViewAutoSizeColumnMode) Перечисление

Описание

Определяет значения для указания способа, которым регулируется ширина колонки.

Члены

Имя члена Описание Значение
ВсеЯчейки (AllCells) Ширина колонки изменяется так, чтобы вместить содержимое всех ячеек колонки, включая ячейку заголовка. 6
ВсеЯчейкиБезЗаголовков (AllCellsExceptHeader) Ширина колонки изменяется так, чтобы вместить содержимое всех ячеек колонки, за исключением ячейки заголовка. 4
ЗаголовокКолонки (ColumnHeader) Ширина колонки изменяется так, чтобы вместить содержимое ячейки заголовка для колонки. 2
Заполнение (Fill) Ширина колонки подбирается таким образом, чтобы суммарная ширина всех колонок в точности заполняла отображаемую область элемента управления, а прокрутка по горизонтали требовалась только для колонок, ширина которых превышает значение свойства КолонкаТаблицы.МинимальнаяШирина (DataGridViewColumn.MinimumWidth). 16
НеУстановлено (NotSet) Режим изменения размеров колонки наследуется из свойства AutoSizeColumnsMode. 0
ОтобразритьЯчейки (DisplayedCells) Ширина колонки изменяется так, чтобы вместить содержимое всех ячеек колонки, которые находятся в строках, отображающихся на экране в настоящий момент, включая строку заголовка. 10
ОтобразритьЯчейкиБезЗаголовков (DisplayedCellsExceptHeader) Ширина колонки изменяется так, чтобы вместить содержимое всех ячеек колонки, которые находятся в строках, отображающихся на экране в настоящий момент, за исключением строки заголовка. 8
Отсутствие (None) Значения ширины колонок не изменяются автоматически. 1
