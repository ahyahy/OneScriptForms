ВертикальноеВыравнивание (VerticalAlign) Перечисление

Описание

Указывает выравнивание текста по вертикали в элементе управления.

Члены

Имя члена Описание Значение
Верх (Top) Выравнивание по верхнему краю. 0
Низ (Bottom) Выравнивание по нижнему краю. 1
Центр (Center) Выравнивание по центру. 2
