ЭлементыПоляВыбораЯчейки.Добавить (DataGridViewComboBoxCellObjectCollection.Add) Метод

Синтаксис

Добавить(<Значение>)

Параметры

Значение (обязательный)
Тип: Произвольный.
Объект для добавления в коллекцию.

Возвращаемое значение

Тип: ЭлементСписка (ListItem).

Элемент списка, который был добавлен в коллекцию. Если параметр Значение не является экземпляром класса ЭлементСписка (ListItem), то будет создан объект ЭлементСписка (ListItem) со свойством Текст (Text) равным параметру Значение, приведенному к строке и свойством Значение (Value) равным параметру Значение.

Описание

Добавляет элемент в список позиций ПолеВыбораЯчейки (DataGridViewComboBoxCell).

Примечание

Пример

Челябинск = ЭлементыПоляВыбораЯчейки.Добавить("Челябинск");
