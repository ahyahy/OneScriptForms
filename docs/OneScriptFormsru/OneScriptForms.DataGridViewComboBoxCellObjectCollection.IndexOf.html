ЭлементыПоляВыбораЯчейки.Индекс (DataGridViewComboBoxCellObjectCollection.IndexOf) Метод

Синтаксис

Индекс(<Значение>)

Параметры

Значение (обязательный)
Тип: ЭлементСписка (ListItem).
Значение элемента для поиска в коллекции.

Возвращаемое значение

Тип: Число.

Отсчитываемый от нуля индекс элемента в коллекции; в противном случае -1.

Описание

Объект, представляющий элемент для поиска в коллекции.

Примечание

Пример

Сообщить("Индекс = " + ЭлементыПоляВыбораЯчейки.Индекс(Чита));
