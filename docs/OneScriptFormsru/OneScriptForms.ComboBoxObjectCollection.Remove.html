ЭлементыПоляВыбора.Удалить (ComboBoxObjectCollection.Remove) Метод

Синтаксис

Удалить(<Элемент>)

Параметры

Элемент (обязательный)
Тип: ЭлементСписка (ListItem).
Удаляемый из списка элемент.

Возвращаемое значение

Описание

Удаляет указанный элемент из объекта ПолеВыбора (ComboBox).

Примечание

Пример

ЭлементыПоляВыбора1.Удалить(Эл2);
Полный пример кода 

Копировать     Выделить всё


ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ПолеВыбора1 = Ф.ПолеВыбора();
ПолеВыбора1.Родитель = Форма1;
ПолеВыбора1.Ширина = Форма1.Ширина - 10;

ПолеВыбора1.СтильВыпадающегоСписка = Ф.СтильПоляВыбора.Редактируемый;
ПолеВыбора1.РежимРисования = Ф.РежимРисования.ВручнуюПеременный;

ЭлементыПоляВыбора1 = ПолеВыбора1.Элементы;
ЭлементыПоляВыбора1.Добавить("Просто строка");
ЭлементыПоляВыбора1.Добавить(Ложь);
Эл2 = ЭлементыПоляВыбора1.Добавить(18.245);
ЭлементыПоляВыбора1.Добавить(Дата(1984,03,26,06,24,21));
ЭлементыПоляВыбора1.Добавить(Ф.Кнопка()); 
Эл3 = ЭлементыПоляВыбора1.Добавить(33.542);

ЭлементыПоляВыбора1.ВысотаЭлемента(0, 50); // У первого элемента высота 50
ЭлементыПоляВыбора1.ВысотаЭлемента(2, 50); // У третьего элемента высота 50
ПолеВыбора1.ВысотаЭлемента = 40; // У поля выбора высота 40

ЭлементыПоляВыбора1.Удалить(Эл2);
