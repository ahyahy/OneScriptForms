Обзор КонтекстноеМеню (ContextMenu)

Конструктор

КонтекстноеМеню (ContextMenu) Конструктор Создаёт новый экземпляр класса КонтекстноеМеню (ContextMenu).

Свойства

Источник (SourceControl) Возвращает элемент управления, отображающий контекстное меню.
ЭлементыМеню (MenuItems) (унаследовано от Меню (Menu)) Возвращает значение, указывающее на коллекцию объектов ЭлементМеню (MenuItem), связанных с меню.

Методы

КлонироватьМеню (CloneMenu) Создает новый объект КонтекстноеМеню (ContextMenu), который является дубликатом текущего объекта КонтекстноеМеню (ContextMenu).
Показать (Show) Отображает контекстное меню в указанной позиции.
ПолучитьГлавноеМеню (GetMainMenu) (унаследовано от Меню (Menu)) Возвращает ГлавноеМеню (MainMenu), содержащее это меню.
СлитьМеню (MergeMenu) (унаследовано от Меню (Menu)) Объединяет объекты ЭлементМеню (MenuItem) одного меню с текущим меню.
ЭлементМеню (MenuItem) (унаследовано от Меню (Menu)) Возвращает ЭлементМеню (MenuItem) по указанному индексу в коллекции.
