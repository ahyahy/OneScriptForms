КольцевойИндикатор.ПоЧасовой (CircularProgressBar.Clockwise) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает направление продвижения индикатора.

Примечание

Направление приращения задаваемое этим свойством носит несколько условный характер. Оно действительно будет соответствовать вращению часовой стрелки при использовании метода Увеличение (Increment) и значении Истина.

Пример

КольцевойИндикатор1.ПоЧасовой = Истина;
Полный пример кода 




Перем КольцевойИндикатор1;

Процедура Таймер1_ПриСрабатыванииТаймера() Экспорт
	Если КольцевойИндикатор1.Значение = 100 Тогда
		КольцевойИндикатор1.Значение = 0;
	КонецЕсли;
	КольцевойИндикатор1.Увеличение(1);
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

КольцевойИндикатор1 = Ф.КольцевойИндикатор();
КольцевойИндикатор1.Родитель = Форма1;
КольцевойИндикатор1.Размер = Ф.Размер(200, 25);
КольцевойИндикатор1.ПоЧасовой = Ложь;
