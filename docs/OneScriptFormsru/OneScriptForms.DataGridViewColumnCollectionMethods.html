КолонкиТаблицы (DataGridViewColumnCollection) Методы

Методы класса КолонкиТаблицы (DataGridViewColumnCollection) перечислены ниже. Полный список членов класса КолонкиТаблицы (DataGridViewColumnCollection) смотрите в разделе КолонкиТаблицы (DataGridViewColumnCollection) Члены.

Методы

Вставить (Insert) Вставляет в коллекцию колонку по указанному индексу.
Добавить (Add) Добавляет заданную колонку в коллекцию.
Очистить (Clear) (унаследовано от КоллекцияОснова (CollectionBase)) Удаляет все объекты из коллекции.
Удалить (Remove) Удаляет указанную колонку из коллекции.
УдалитьПоИндексу (RemoveAt) (унаследовано от КоллекцияОснова (CollectionBase)) Удаляет объект по указанному индексу из коллекции.
Элемент (Item) Возвращает значение элемента по указанному индексу в коллекции.
