ЗначокУзла.ИзображениеВыбранного (NodeStateIcon.SelectedImage) Свойство

Использование

Чтение и запись.

Значение

Тип: Картинка (Bitmap).

Возвращает или задает изображение, отображаемое для выбранного объекта ЗначокУзла (NodeStateIcon).

Объект Картинка (Bitmap), который отображается для выбранного объекта ЗначокУзла (NodeStateIcon). Значение по умолчанию: Неопределено.

Примечание

Пример

ЗначокУзла1.ИзображениеВыбранного = Картинка2;
