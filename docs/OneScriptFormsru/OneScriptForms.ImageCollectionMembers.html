Свойства

Количество (Count) (унаследовано от КоллекцияОснова (CollectionBase)) Возвращает общее количество объектов в коллекции.

Методы

Добавить (Add) Добавляет указанное изображение в СписокИзображений (ImageList), используя указанный цвет для создания маски.
ДобавитьПолосу (AddStrip) Добавляет полосу изображений заданного изображения для СписокИзображений (ImageList).
Очистить (Clear) (унаследовано от КоллекцияОснова (CollectionBase)) Удаляет все объекты из коллекции.
УдалитьПоИндексу (RemoveAt) (унаследовано от КоллекцияОснова (CollectionBase)) Удаляет объект по указанному индексу из коллекции.
Элемент (Item) Возвращает изображение в существующем СписокИзображений (ImageList).
