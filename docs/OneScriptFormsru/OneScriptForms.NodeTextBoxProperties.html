ПолеВводаУзла (NodeTextBox) Свойства

Свойства класса ПолеВводаУзла (NodeTextBox) перечислены ниже. Полный список членов класса ПолеВводаУзла (NodeTextBox) смотрите в разделе ПолеВводаУзла (NodeTextBox) Члены.

Свойства

ВертикальноеВыравнивание (VerticalAlign) Возвращает или задает выравнивание текста по вертикали.
ВыравниваниеТекста (TextAlign) Возвращает или задает способ горизонтального выравнивания текста.
ЗначениеИзменено (ValueChanged) Возвращает или задает код для выполнения, при изменении значения поля ввода узла.
ИспользоватьСовместимуюОтрисовку (UseCompatibleTextRendering) Возвращает или задает значение, определяющее, следует ли использовать класс Graphics (GDI+) или класс TextRenderer (GDI) для отрисовки текста.
Колонка (ParentColumn) Объект КолонкаДереваЗначений (TreeColumn), которому принадлежит поле ввода узла.
ЛевыйОтступ (LeftMargin) Возвращает или задает размер левого отступа.
Обрезка (Trimming) Возвращает или задает значение, определяющее правила сокращения строки.
Редактируемый (EditEnabled) Возвращает или задает значение, определяющее доступно ли редактирование поля ввода узла.
Шрифт (Font) Возвращает или задает шрифт текста.
