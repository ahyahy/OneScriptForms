ФормыДляОдноСкрипта.РазрешитьСобытия (OneScriptForms.AllowEvents) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, показывающее, разрешена ли обработка событий.

Значение по умолчанию - Ложь.

Примечание

Метод ФормыДляОдноСкрипта.ЗапуститьОбработкуСобытий (OneScriptForms.StartEventProcessing) устанавливает это свойство в значение Истина, запускает обработку событий и останавливает выполнение скрипта до выхода из цикла обработки событий. Это свойство позволяет разрешить или отменить обработку событий не останавливая выполнение скрипта.

Пример

Ф.РазрешитьСобытия = Истина;
