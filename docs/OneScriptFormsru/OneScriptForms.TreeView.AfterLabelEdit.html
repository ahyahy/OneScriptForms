Дерево.ПослеРедактированияНадписи (TreeView.AfterLabelEdit) Свойство

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код для выполнения после изменения текстовой надписи узла дерева.

Примечание

Пример

Дерево1.ПослеРедактированияНадписи = Ф.Действие(ЭтотОбъект, "Дерево1_ПослеРедактированияНадписи");
