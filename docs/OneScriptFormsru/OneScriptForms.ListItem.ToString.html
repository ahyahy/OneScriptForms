ЭлементСписка.ВСтроку (ListItem.ToString) Метод

Синтаксис

ВСтроку()

Параметры

Возвращаемое значение

Тип: Строка.

Описание

Возвращает строку, представляющую текущий объект.

Примечание

Пример

Эл = ПолеВыбора1.Элементы.Элемент(ПолеВыбора1.ИндексВыбранного);
Строка = Эл.ВСтроку();
