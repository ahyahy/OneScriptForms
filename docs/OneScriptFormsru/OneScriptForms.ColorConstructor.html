Цвет (Color) Конструктор

Синтаксис

Цвет()

Цвет(<ИмяЦвета>)

Цвет(<Красный>, <Зеленый>, <Синий>)

Параметры

ИмяЦвета
Тип: Строка.
Название определенного системой цвета.
Красный
Тип: Число.
Значение красной составляющей цвета.
Зеленый
Тип: Число.
Значение зеленой составляющей цвета.
Синий
Тип: Число.
Значение синей составляющей цвета.

Описание

Создаёт новый экземпляр класса Цвет (Color).

Примечание

Пример

Цвет1 = Ф.Цвет();
Цвет2 = Ф.Цвет("Розовый");
Цвет3 = Ф.Цвет(38, 210, 180);
