Форма.РазмерАвтоМасштабирования (Form.AutoScaleBaseSize) Свойство

Использование

Чтение и запись.

Значение

Тип: Размер (Size).

Возвращает или задает базовый размер, используемый для автоматического масштабирования формы.

Примечание

Этот член был сохранен для обеспечения обратной совместимости.

Значение этого свойства используется при первоначальном создании формы. После задания свойства его нельзя изменить.

Значение свойства РазмерАвтоМасштабирования (AutoScaleBaseSize) используется во время вывода формы для расчета коэффициента масштабирования формы. Базовый размер автомасштабирования используется формой в качестве основы для сравнения с размером шрифта системы, чтобы определить, как масштабировать форму при использовании автомасштабирования.

Свойство РазмерАвтоМасштабирования (AutoScaleBaseSize) возвращает ширину и высоту для шрифта, заданного в свойстве Font формы. В Windows Forms это единственный источник сведений о средней ширине символов конкретного шрифта.

Пример

Сообщить("Форма1.РазмерАвтоМасштабирования.Ширина = " + Форма1.РазмерАвтоМасштабирования.Ширина);
Сообщить("Форма1.РазмерАвтоМасштабирования.Высота = " + Форма1.РазмерАвтоМасштабирования.Высота);
