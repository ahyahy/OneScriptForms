МассивСписок.Установить (ArrayList.Set) Метод

Синтаксис

Установить(<Индекс>, <Значение>)

Параметры

Индекс (обязательный)
Тип: Число.
Отсчитываемый от нуля индекс элемента массива.
Значение (обязательный)
Тип: Произвольный.
Значение элемента, которое будет установлено по указанному индексу.

Возвращаемое значение

Описание

Устанавливает значение элемента по указанному индексу.

Примечание

Пример

М.Установить(2, Ложь);
