Дерево.ПолучитьУзел (TreeView.GetNodeAt) Метод

Синтаксис

ПолучитьУзел(<X>, <Y>)

Параметры

X (обязательный)
Тип: Число.
Позиция X для оценки и извлечения узла.
Y (обязательный)
Тип: Число.
Позиция Y для оценки и извлечения узла.

Возвращаемое значение

Тип: УзелДерева (TreeNode).

Объект УзелДерева (TreeNode) в указанном месте, в дереве или Неопределено, если в этом месте нет узла.

Описание

Возвращает узел дерева, расположенный в точке с указанными координатами.

Примечание

Пример

Узел = Дерево1.ПолучитьУзел(10, 10);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Дерево1 = Форма1.ЭлементыУправления.Добавить(Ф.Дерево());
Узел1 = Дерево1.Узлы.Добавить("Узел1");
Узел2 = Дерево1.Узлы.Добавить("Узел2");
Узлы1 = Узел1.Узлы;
ПодУзел1 = Узлы1.Добавить("ПодУзел1");

Узел = Дерево1.ПолучитьУзел(10, 10);
Сообщить("Узел.Текст = " + Узел.Текст); // Узел.Текст = Узел1

Ф.ЗапуститьОбработкуСобытий();
