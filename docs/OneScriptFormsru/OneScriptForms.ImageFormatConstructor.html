Описание

Используйте метод ФормыДляОдноСкрипта.ФорматИзображения (OneScriptForms.ImageFormat) для создания нового экземпляра класса ФорматИзображения (ImageFormat).

Примечание

Формат нового экземпляра класса соответствует формату изображений, Guid которого состоит только из нулей.

Пример

ФорматИзображения1 = Ф.ФорматИзображения();
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ФорматИзображения1 = Ф.ФорматИзображения();
