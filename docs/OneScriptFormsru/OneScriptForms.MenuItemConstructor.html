ЭлементМеню (MenuItem) Конструктор

Синтаксис

ЭлементМеню(<Текст>, <Нажатие>, <СочетаниеКлавиш>)

Параметры

Текст (необязательный)
Тип: Строка.
Нажатие (необязательный)
Тип: Произвольный.
Параметр может быть задан типом значения Действие из библиотеки односкрипта, или типом значения Действие (Action).
СочетаниеКлавиш (необязательный)
Тип: СочетаниеКлавиш (Shortcut).

Описание

Используйте метод ФормыДляОдноСкрипта.ЭлементМеню (OneScriptForms.MenuItem) для создания нового экземпляра класса ЭлементМеню (MenuItem).

Примечание

Если задать для параметра Текст значение "-", пункт меню будет отображаться как разделитель (горизонтальная линия), а не стандартный элемент меню.

Пример

Эл = Ф.ЭлементМеню("Открыть", Ф.Действие(ЭтотОбъект, "Открыть_ПриНажатии"), Ф.СочетаниеКлавиш.ShiftF11);
