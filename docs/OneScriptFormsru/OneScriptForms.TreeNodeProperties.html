УзелДерева (TreeNode) Свойства

Свойства класса УзелДерева (TreeNode) перечислены ниже. Полный список членов класса УзелДерева (TreeNode) смотрите в разделе УзелДерева (TreeNode) Члены.

Свойства

Дерево (TreeView) Возвращает родительское дерево, которому назначен узел дерева.
Индекс (Index) Возвращает позицию узла дерева в коллекции узлов дерева.
ИндексВыбранногоИзображения (SelectedImageIndex) Возвращает или задает значение индекса в списке изображений для изображения, которое отображается, если узел дерева находится в выбранном состоянии.
ИндексИзображения (ImageIndex) Возвращает или задает значение индекса в списке изображений для изображения, отображаемого, когда узел дерева находится в невыбранном состоянии.
Метка (Tag) Возвращает объект, содержащий дополнительные данные.
ПолныйПуть (FullPath) Возвращает путь из корневого узла дерева к текущему узлу дерева.
Помечен (Checked) Возвращает или задает значение, указывающее, находится ли узел дерева в выбранном состоянии.
ПредыдущийОтображаемыйУзел (PrevVisibleNode) Возвращает предшествующий отображаемый узел дерева.
Родитель (Parent) Возвращает родительский узел дерева для текущего узла дерева.
СледующийОтображаемыйУзел (NextVisibleNode) Возвращает следующий отображаемый узел дерева.
Текст (Text) Возвращает или задает текст, отображаемый в надписи узла дерева.
Узлы (Nodes) Возвращает коллекцию объектов УзелДерева (TreeNode), назначенных текущему узлу дерева.
ШрифтУзла (NodeFont) Возвращает или задает шрифт, используемый для отображения текста надписи узла дерева.
