КолонкаТаблицы.СтильЯчейки (DataGridViewColumn.DefaultCellStyle) Свойство

Использование

Чтение и запись.

Значение

Тип: СтильЯчейки (DataGridViewCellStyle).

Возвращает или задает стиль по умолчанию для ячеек колонки.

Примечание

Пример

СтильЯчейки1 = Ф.СтильЯчейки();
СтильЯчейки1.ЦветФона = Ф.Цвет("Бирюзовый");
КолонкаПолеВвода1.СтильЯчейки = СтильЯчейки1;
