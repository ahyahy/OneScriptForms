НаблюдательФайловойСистемы.ВключаяПодкаталоги (FileSystemWatcher.IncludeSubDirectories) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, следует ли отслеживать подкаталоги в указанном пути.

Истина - вы хотите отслеживать подкаталоги, Ложь - в противном случае. Значение по умолчанию: Ложь.

Примечание

Установите ВключаяПодкаталоги (IncludeSubDirectories) в значение Истина - вы хотите отслеживать файлы и каталоги также и его подкаталогах. Установка свойства ВключаяПодкаталоги (IncludeSubDirectories) в значение Ложь помогает уменьшить количество уведомлений.

Пример

НаблюдательФайловойСистемы1.ВключаяПодкаталоги = Истина;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

НаблюдательФайловойСистемы1 = Ф.НаблюдательФайловойСистемы();
НаблюдательФайловойСистемы1.Путь = Ф.Окружение().ПолучитьПутьКаталога(Ф.ОсобаяПапка.Личное);
НаблюдательФайловойСистемы1.ВключаяПодкаталоги = Истина;

Ф.ЗапуститьОбработкуСобытий();
