ПодэлементыСпискаЭлементов.Элемент (ListViewSubItemCollection.Item) Метод

Синтаксис

Элемент(<Индекс>, <Значение>)

Параметры

Индекс (обязательный)
Тип: Число.
Индекс получаемого или устанавливаемого подэлемента в коллекции.
Значение (необязательный)
Тип: ПодэлементСпискаЭлементов (ListViewSubItem).
Устанавливаемый элемент.

Возвращаемое значение

Тип: ПодэлементСпискаЭлементов (ListViewSubItem).

Описание

Устанавливает и/или возвращает подэлемент по указанному индексу в коллекции.

Примечание

Пример

Сообщить("ПодэлементыРадуга.Элемент(2).Текст = " + ПодэлементыРадуга.Элемент(2).Текст);

СписокЭлементов1.РежимОтображения = Ф.РежимОтображения.Подробно;

Колонка5 = Колонки.Добавить(Ф.Колонка("где", 90, Ф.ГоризонтальноеВыравнивание.Лево));
Колонка5 = Колонки.Добавить(Ф.Колонка("сидят", 90, Ф.ГоризонтальноеВыравнивание.Лево));
Колонка5 = Колонки.Добавить(Ф.Колонка("фазаны", 100, Ф.ГоризонтальноеВыравнивание.Лево));
Элементы = СписокЭлементов1.Элементы;
Радуга = Элементы.Добавить(Ф.ЭлементСпискаЭлементов("Радуга"));

ПодэлементыРадуга = Радуга.Подэлементы;
ПодэлементыРадуга.Добавить(Ф.ПодэлементСпискаЭлементов("красный"));
ПодэлементыРадуга.Добавить(Ф.ПодэлементСпискаЭлементов("оранжевый"));
ПодэлементыРадуга.Добавить(Ф.ПодэлементСпискаЭлементов("желтый"));
ПодэлементыРадуга.Добавить(Ф.ПодэлементСпискаЭлементов("зеленый"));
ПодэлементыРадуга.Добавить(Ф.ПодэлементСпискаЭлементов("голубой"));
ПодэлементыРадуга.Добавить(Ф.ПодэлементСпискаЭлементов("синий"));
ПодэлементыРадуга.Добавить(Ф.ПодэлементСпискаЭлементов("фиолетовый"));
