Календарь.КонечнаяДата (MonthCalendar.SelectionEnd) Свойство

Использование

Чтение и запись.

Значение

Тип: Дата.

Возвращает или задает конечную дату для выбранного диапазона дат.

Примечание

Пример

КонечнаяДата1 = Календарь1.КонечнаяДата;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Календарь1 = Ф.Календарь();
Календарь1.Родитель = Форма1;
КонечнаяДата1 = Календарь1.КонечнаяДата;
Сообщить("Старая КонечнаяДата = " + КонечнаяДата1);
КонечнаяДата1 = КонечнаяДата1 + 86400; // прибавили один день в секундах
Сообщить("Новая КонечнаяДата = " + КонечнаяДата1);

Ф.ЗапуститьОбработкуСобытий();
