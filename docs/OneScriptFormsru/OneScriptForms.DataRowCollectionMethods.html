СтрокиДанных (DataRowCollection) Методы

Методы класса СтрокиДанных (DataRowCollection) перечислены ниже. Полный список членов класса СтрокиДанных (DataRowCollection) смотрите в разделе СтрокиДанных (DataRowCollection) Члены .

Методы

Вставить (InsertAt) Вставляет новую строку в коллекцию по указанному индексу.
Добавить (Add) Создает и добавляет указанный объект СтрокаДанных (DataRow) в коллекцию.
Очистить (Clear) Очищает коллекцию от всех строк.
Удалить (Remove) Удаляет строку из коллекции.
УдалитьПоИндексу (RemoveAt) Удаляет строку по указанному индексу из коллекции.
Элемент (Item) Возвращает указанный объект СтрокаДанных (DataRow) из коллекции.
