ПоведениеСсылки (LinkLabelLinkBehavior) Перечисление

Описание

Определяет поведение ссылки.

Члены

Имя члена Описание Значение
ВсегдаПодчеркнутый (AlwaysUnderline) В ссылке всегда отображается подчеркнутый текст. 1
НеПодчеркнутый (NeverUnderline) Текст ссылки никогда не отображается подчеркнутым. Ссылку по-прежнему можно отличить от другого текста с помощью свойства ЦветСсылки (LinkColor) элемента управления НадписьСсылка (LinkLabel). 3
ПодчеркнутыйПриНаведении (HoverUnderline) В ссылке отображается подчеркнутый текст, только когда мышь находится на тексте ссылки. 2
СистемныйПоУмолчанию (SystemDefault) Поведение данной настройки зависит от параметров, заданных с помощью диалогового окна "Свойства обозревателя" панели управления или Internet Explorer. 0
