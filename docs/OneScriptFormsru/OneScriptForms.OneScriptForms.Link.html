ФормыДляОдноСкрипта.Ссылка (OneScriptForms.Link) Метод

Синтаксис

Ссылка()

Параметры

Возвращаемое значение

Тип: Ссылка (Link).

Описание

Создаёт новый экземпляр класса Ссылка (Link).

Примечание

Пример

Ссылка1 = Ф.Ссылка();
