Вкладка.ИндексИзображения (TabPage.ImageIndex) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает индекс изображения, отображаемого на этой вкладке.

Значение по умолчанию: -1, что означает отсутствие изображения.

Примечание

Пример

Вкладка1.ИндексИзображения = 0;
