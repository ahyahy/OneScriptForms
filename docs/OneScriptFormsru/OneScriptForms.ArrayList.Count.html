МассивСписок.Количество (ArrayList.Count) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает количество элементов, содержащихся в МассивСписок (ArrayList).

Примечание

Пример

Сообщить("М2.Количество = " + М2.Количество);
