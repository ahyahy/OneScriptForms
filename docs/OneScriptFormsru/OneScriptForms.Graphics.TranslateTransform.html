Графика.Сдвинуть (Graphics.TranslateTransform) Метод

Синтаксис

Сдвинуть(<X>, <Y>)

Параметры

X (обязательный)
Тип: Число.
x координата точки сдвига.
Y (обязательный)
Тип: Число.
y координата точки сдвига.

Возвращаемое значение

Описание

Сдвигает элементы по горизонтали и вертикали.

Примечание

Пример

Графика1 = Форма1.СоздатьГрафику();
ПоХ = Форма1.КлиентШирина / 2.0;
ПоУ = Форма1.КлиентВысота / 2.0;
Графика1.Сдвинуть (ПоХ, ПоУ);

Перем Ф, Форма1;

Процедура Форма1_ПриПерерисовке() Экспорт
