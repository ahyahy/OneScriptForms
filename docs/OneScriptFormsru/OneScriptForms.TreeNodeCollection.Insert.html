УзлыДерева.Вставить (TreeNodeCollection.Insert) Метод

Синтаксис

Вставить(<Индекс>, <УзелДерева>)

Параметры

Индекс (обязательный)
Тип: Число.
УзелДерева (обязательный)
Тип: УзелДерева (TreeNode).

Возвращаемое значение

Тип: УзелДерева (TreeNode).

Описание

Вставляет указанный узел дерева по указанному индексу.

Примечание

Пример

ВставляемыйУзел = Ф.УзелДерева("ВставляемыйУзел");
Узлы1.Вставить(0, ВставляемыйУзел);
