КнопкиПанелиИнструментов (ToolBarButtonCollection) Методы

Методы класса КнопкиПанелиИнструментов (ToolBarButtonCollection) перечислены ниже. Полный список членов класса КнопкиПанелиИнструментов (ToolBarButtonCollection) смотрите в разделе КнопкиПанелиИнструментов (ToolBarButtonCollection) Члены.

Методы

Вставить (Insert) Вставляет существующую КнопкаПанелиИнструментов (ToolBarButton) в коллекции кнопок панели инструментов в указанном месте.
Добавить (Add) Добавляет новую КнопкаПанелиИнструментов (ToolBarButton) в конец коллекции кнопок панели инструментов с указанным значение свойства КнопкаПанелиИнструментов.Текст (ToolBarButton.Text).
Очистить (Clear) (унаследовано от КоллекцияОснова (CollectionBase)) Удаляет все объекты из коллекции.
Удалить (Remove) Удаляет заданную кнопку из коллекции кнопок панели инструментов.
УдалитьПоИндексу (RemoveAt) (унаследовано от КоллекцияОснова (CollectionBase)) Удаляет объект по указанному индексу из коллекции.
Элемент (Item) Устанавливает и/или возвращает КнопкаПанелиИнструментов (ToolBarButton) по указанному индексу в коллекции кнопок панели инструментов.
