Обзор ФорматИзображения (ImageFormat)

Конструктор

 ФорматИзображения (ImageFormat) Конструктор Используйте метод ФормыДляОдноСкрипта.ФорматИзображения (OneScriptForms.ImageFormat) для создания нового экземпляра класса ФорматИзображения (ImageFormat).

Свойства

Bmp (Bmp) Возвращает формат изображения точечного рисунка (BMP).
Gif (Gif) Возвращает формат изображения графического обмена (GIF).
Icon (Icon) Возвращает формат изображения значков Windows.
Jpeg (Jpeg) Возвращает формат изображения jpeg (JPEG).
Png (Png) Возвращает формат изображения png (PNG).

Методы

ВСтроку (ToString) Преобразует объект ФорматИзображения (ImageFormat) в удобную для восприятия строку.
