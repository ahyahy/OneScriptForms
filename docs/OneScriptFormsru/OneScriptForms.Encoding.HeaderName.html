Кодировка.ИмяЗаголовка (Encoding.HeaderName) Свойство

Использование

Только чтение.

Значение

Тип: Строка.

Возвращает имя текущей кодировки, которое может использоваться с тегами заголовка сообщения почтового агента.

Примечание

Пример

Кодировка1 = Ф.Кодировка().UTF8;
Сообщить("Кодировка1.ИмяЗаголовка = " + Кодировка1.ИмяЗаголовка);
