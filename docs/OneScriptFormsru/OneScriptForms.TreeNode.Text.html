УзелДерева.Текст (TreeNode.Text) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает текст, отображаемый в надписи узла дерева.

Примечание

Пример

Сообщить("ПодУзел2.Текст = " + ПодУзел2.Текст);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Дерево1 = Форма1.ЭлементыУправления.Добавить(Ф.Дерево());
Дерево1.Ширина = 200;
Узел1 = Дерево1.Узлы.Добавить("Узел1");
Узел2 = Дерево1.Узлы.Добавить("Узел2");
Узел3 = Дерево1.Узлы.Добавить("Узел3");
Узлы1 = Узел1.Узлы;
ПодУзел1 = Узлы1.Добавить("ПодУзел1");
Узлы2 = ПодУзел1.Узлы;
ПодУзел2 = Узлы2.Добавить("ПодУзел2");

Дерево1.ВыбранныйУзел = Узел2;

Сообщить("ПодУзел2.Текст = " + ПодУзел2.Текст); // ПодУзел2.Текст = ПодУзел2

Ф.ЗапуститьОбработкуСобытий();
