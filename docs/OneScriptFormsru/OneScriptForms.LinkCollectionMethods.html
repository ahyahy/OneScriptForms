КоллекцияСсылок (LinkCollection) Методы

Методы класса КоллекцияСсылок (LinkCollection) перечислены ниже. Полный список членов класса КоллекцияСсылок (LinkCollection) смотрите в разделе КоллекцияСсылок (LinkCollection) Члены.

Методы

Добавить (Add) Добавляет в коллекцию ссылку и связываемые с ней сведения.
Очистить (Clear) Удаляет все ссылки из коллекции.
Удалить (Remove) Удаляет указанную ссылку из коллекции.
УдалитьПоИндексу (RemoveAt) Удаляет ссылку, находящуюся в указанном месте коллекции.
Элемент (Item) Устанавливает и/или возвращает элемент по указанному индексу в коллекции.
