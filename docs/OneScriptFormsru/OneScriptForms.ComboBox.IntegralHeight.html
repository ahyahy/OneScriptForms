ПолеВыбора.ПодборВысоты (ComboBox.IntegralHeight) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, следует ли изменить размер элемента управления, чтобы избежать частичного отображения элементов.

Примечание

Пример

ПолеВыбора1.ПодборВысоты = Истина;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Ширина = 310;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ПолеВыбора1 = Ф.ПолеВыбора();
ПолеВыбора1.Родитель = Форма1;
ПолеВыбора1.Ширина = 300;

ЭлементыПоляВыбора1 = ПолеВыбора1.Элементы;
ЭлементыПоляВыбора1.Добавить("Москва");
ЭлементыПоляВыбора1.Добавить("Хабаровск");
ЭлементыПоляВыбора1.Добавить("Саратов");
