ЭлементыУправления.Удалить (ControlCollection.Remove) Метод

Синтаксис

Удалить(<ЭлементУправления>)

Параметры

ЭлементУправления (обязательный)
Тип: Произвольный.

Возвращаемое значение

Описание

Удаляет элемент управления из коллекции;

Примечание

Пример

Форма1.ЭлементыУправления.Удалить(Кнопка2);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Кнопка1 = Форма1.ЭлементыУправления.Добавить(Ф.Кнопка());
Кнопка1.Текст = "Кнопка1";
Кнопка1.Положение = Ф.Точка(20, 20);

Кнопка2 = Форма1.ЭлементыУправления.Добавить(Ф.Кнопка());
Кнопка2.Текст = "Кнопка2";
Кнопка2.Ниже(Кнопка1, 5);

Кнопка3 = Форма1.ЭлементыУправления.Добавить(Ф.Кнопка());
Кнопка3.Текст = "Кнопка3";
Кнопка3.Ниже(Кнопка2, 5);

Кнопка4 = Форма1.ЭлементыУправления.Добавить(Ф.Кнопка());
Кнопка4.Текст = "Кнопка4";
Кнопка4.Ниже(Кнопка3, 5);

Форма1.ЭлементыУправления.Удалить(Кнопка2);
