СортированныйСписок.Значения (SortedList.Values) Свойство

Использование

Только чтение.

Значение

Тип: МассивСписок (ArrayList).

Возвращает МассивСписок (ArrayList), содержащий значения из коллекции СортированныйСписок (SortedList).

Примечание

Пример

МассивСписок1 = СортированныйСписок1.Значения;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

СортированныйСписок1 = Ф.СортированныйСписок();
СортированныйСписок1.Добавить(1.10, "Простой стиль");

СортированныйСписок1.Добавить(4.13, 18.245);
СортированныйСписок1.Добавить(3, 18.246);
СортированныйСписок1.Добавить(5.15, Дата(1984,03,26,06,24,21));
