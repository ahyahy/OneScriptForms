ИндексыВыбранныхПоляСписка.Индекс (ListBoxSelectedIndexCollection.IndexOf) Метод

Синтаксис

Индекс(<Значение>)

Параметры

Значение (обязательный)
Тип: Число.
Отсчитываемый от нуля индекс из списка ЭлементыПоляСписка (ListBoxObjectCollection), искомый в этой коллекции.

Возвращаемое значение

Тип: Число.

Отсчитываемый от нуля индекс в коллекции; в противном случае -1.

Описание

Возвращает индекс в коллекции с указанным индексом из объекта ЭлементыПоляСписка (ListBoxObjectCollection) элемента управления ПолеСписка (ListBox).

Примечание

Пример

Индекс = ПолеСписка1.ИндексыВыбранных.Индекс(3);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ПолеСписка1 = Форма1.ЭлементыУправления.Добавить(Ф.ПолеСписка());
ПолеСписка1.Стыковка = Ф.СтильСтыковки.Заполнение;
ПолеСписка1.РежимВыбора = Ф.РежимВыбора.МножественныйПростой;
ПолеСпискаЭлементы1 = ПолеСписка1.Элементы;
Для А = 1 По 10 Цикл
	ПолеСпискаЭлементы1.Добавить("Элемент" + А);
КонецЦикла;
ПолеСписка1.ВыбранныйЭлемент = ПолеСпискаЭлементы1.Элемент(0);
ПолеСписка1.ВыбранныйЭлемент = ПолеСпискаЭлементы1.Элемент(1);
ПолеСписка1.ВыбранныйЭлемент = ПолеСпискаЭлементы1.Элемент(3);

Сообщить("Если элемент под индексом 3 есть среди выделенных, то его индекс в коллекции выделенных = " + ПолеСписка1.ИндексыВыбранных.Индекс(3));
