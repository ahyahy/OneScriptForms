ХэшТаблица (HashTable) Методы

Методы класса ХэшТаблица (HashTable) перечислены ниже. Полный список членов класса ХэшТаблица (HashTable) смотрите в разделе ХэшТаблица (HashTable) Члены.

Методы

Добавить (Add) Добавляет элемент с указанным ключом и значением в хэш-таблицу.
Очистить (Clear) Удаляет все элементы из хэш-таблицы.
Удалить (Remove) Удаляет элемент с указанным ключом из хэш-таблицы.
Установить (Set) Задает значение записи в хэш-таблице.
Элемент (Item) Возвращает значение, связанное с указанным ключом.
