АктивацияЭлемента (ItemActivation) Перечисление

Описание

Задает действие пользователя, необходимое для активизации записей в элементе управления СписокЭлементов (ListView) и обратную связь, возникающую при наведении указателя мыши на элемент.

Члены

Имя члена Описание Значение
ДваНажатия (TwoClick) Пользователь должен щелкнуть элемент дважды, чтобы активировать его. Это отличается от стандартного двойного щелчка, поскольку между щелчками может пройти некоторое время. Изменяется цвет текста элемента при перемещении пользователем указателя мыши над элементом. 2
ОдноНажатие (OneClick) Активация одним щелчком. Курсор изменяется на указатель курсора в виде руки и изменяется цвет текста элемента при перемещении пользователем указателя мыши над элементом. 1
Стандартная (Standard) Активация двойным щелчком. Без обратной связи при перемещении пользователем указателя мыши над элементом. 0
