СписокИзображений.Изображения (ImageList.Images) Метод

Синтаксис

Изображения(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.
Индекс изображения.

Возвращаемое значение

Тип: Картинка (Bitmap).

Изображение в списке, по указанному индексу.

Описание

Возвращает значение по указанному индексу.

Примечание

Пример

Изображение = СписокИзображений1.Изображения(0);
