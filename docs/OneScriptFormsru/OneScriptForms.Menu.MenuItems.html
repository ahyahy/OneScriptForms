Меню.ЭлементыМеню (Menu.MenuItems) Свойство

Использование

Только чтение.

Значение

Тип: ЭлементыМеню (MenuItemCollection).

Возвращает значение, указывающее на коллекцию объектов ЭлементМеню (MenuItem), связанных с меню.

Примечание

Пример

Элементы = Подменю_Файл.ЭлементыМеню;
