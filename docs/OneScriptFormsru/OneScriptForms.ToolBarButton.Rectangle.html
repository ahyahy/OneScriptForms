КнопкаПанелиИнструментов.Прямоугольник (ToolBarButton.Rectangle) Свойство

Использование

Только чтение.

Значение

Тип: Прямоугольник (Rectangle).

Возвращает ограничивающий прямоугольник для кнопки панели инструментов.

Примечание

Пример

Сообщить("Кнопка1.Прямоугольник.Икс = " + Кнопка1.Прямоугольник.Икс);
Сообщить("Кнопка1.Прямоугольник.Игрек = " + Кнопка1.Прямоугольник.Игрек);
Сообщить("Кнопка1.Прямоугольник.Ширина = " + Кнопка1.Прямоугольник.Ширина);
Сообщить("Кнопка1.Прямоугольник.Высота = " + Кнопка1.Прямоугольник.Высота);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ПанельИнструментов1 = Форма1.ЭлементыУправления.Добавить(Ф.ПанельИнструментов());
Кнопка1 = Ф.КнопкаПанелиИнструментов("Открыть");
ПанельИнструментов1.Кнопки.Добавить(Кнопка1);

Сообщить("Кнопка1.Прямоугольник.Икс = " + Кнопка1.Прямоугольник.Икс);
