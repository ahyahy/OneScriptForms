СсылкаЯчейки.ЦветПосещеннойСсылки (DataGridViewLinkCell.VisitedLinkColor) Свойство

Использование

Чтение и запись.

Значение

Тип: Цвет (Color).

Возвращает или задает цвет, используемый для отображения уже посещенной ссылки.

Примечание

Пример

СсылкаЯчейки.ЦветПосещеннойСсылки = Ф.Цвет("Фиолетовый");
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Ширина = 500;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
Таблица1.Родитель = Форма1;
Таблица1.Стыковка = Ф.СтильСтыковки.Заполнение;
Таблица1.АвтоНумерацияСтрок = Истина;

КолонкаСсылка1 = Ф.КолонкаСсылка();
Таблица1.Колонки.Добавить(КолонкаСсылка1);
Таблица1.Колонки.Добавить(Ф.КолонкаСсылка());
Таблица1.КоличествоСтрок = 5;
Таблица1.ДобавлятьСтроки = Ложь;

КолонкаСсылка1.Текст = "infostart.ru";
КолонкаСсылка1.ИспользоватьТекстКакСсылку = Истина;

КолонкаСсылка1.ЦветПосещеннойСсылки = Ф.Цвет("Зеленый");

СсылкаЯчейки = Таблица1.Ячейка(0, 2);
СсылкаЯчейки.ЦветПосещеннойСсылки = Ф.Цвет("Фиолетовый");

Ф.ЗапуститьОбработкуСобытий();
