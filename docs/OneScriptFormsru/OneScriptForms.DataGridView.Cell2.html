Таблица.Ячейка (DataGridView.Cell) Метод

Синтаксис

Ячейка(<НомерКолонки>, <НомерСтроки>)

Параметры

НомерКолонки (обязательный)
Тип: Число.
Отсчитываемый от нуля индекс колонки.
НомерСтроки (обязательный)
Тип: Число.
Отсчитываемый от нуля индекс строки.

Возвращаемое значение

Тип: ЯчейкаТаблицы (DataGridViewCell).

Ячейка по указанным индексам колонки и строки.

Описание

Объект ЯчейкаТаблицы (DataGridViewCell), находящийся по указанным индексам колонки и строки.

Примечание

Пример

Таблица1.Ячейка(1, 1).Значение = "лампа";

Таблица1.ТолькоЧтение = Истина;
