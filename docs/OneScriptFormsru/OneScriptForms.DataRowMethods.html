СтрокаДанных (DataRow) Методы

Методы класса СтрокаДанных (DataRow) перечислены ниже. Полный список членов класса СтрокаДанных (DataRow) смотрите в разделе СтрокаДанных (DataRow) Члены.

Методы

ЗавершитьРедактирование (EndEdit) Прекращает изменение строки.
НачатьРедактирование (BeginEdit) Начинает операцию изменения объекта СтрокаДанных (DataRow).
ОтказИзменений (RejectChanges) Отменяет все изменения, внесенные в строку после последнего вызова метода СтрокаДанных.ПринятьИзменения (DataRow.AcceptChanges).
ОтменаРедактирования (CancelEdit) Отменяет текущее редактирование строки.
ПринятьИзменения (AcceptChanges) Фиксирует все изменения, внесенные в эту строку со времени последнего вызова метода СтрокаДанных.ПринятьИзменения (DataRow.AcceptChanges).
Удалить (Delete) Удаляет строку данных.
УстановитьЭлемент (SetItem) Устанавливает данные, хранящиеся в указанной колонке.
Элемент (Item) Возвращает объект ЭлементДанных (DataItem) в указанной колонке.
