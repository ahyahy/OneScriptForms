Обзор Шрифт (Font)

Конструктор

Шрифт (Font) Конструктор Используйте метод ФормыДляОдноСкрипта.Шрифт (OneScriptForms.Font) для создания нового экземпляра класса Шрифт (Font).

Свойства

Высота (Height) Возвращает значение междустрочного интервала данного шрифта.
Имя (Name) Возвращает название шрифта.
Размер (Size) Возвращает ширину самой большой буквы шрифта.
Стиль (Style) Возвращает сведения о стиле для данного шрифта.
