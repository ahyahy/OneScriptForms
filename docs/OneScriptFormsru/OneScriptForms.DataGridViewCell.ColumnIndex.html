ЯчейкаТаблицы.ИндексКолонки (DataGridViewCell.ColumnIndex) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает индекс колонки для данной ячейки.

Индекс колонки, содержащей ячейку; -1, если ячейка не содержится в колонке.

Примечание

Пример

Сообщить("" + Таблица1.ТекущаяЯчейка.ИндексКолонки);

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();

Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

Таблица1 = Ф.Таблица();
