Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает имя элемента управления.

ПрямоугольнаяКисть1 = Ф.ПрямоугольнаяКисть(Ф.СтильШтриховки.Мешковина, Ф.Цвет().Белый, Ф.Цвет().Красный);
ПрямоугольнаяКисть1.Имя = "ПрямоугольнаяКисть1";
Сообщить("ПрямоугольнаяКисть1.Имя = " + ПрямоугольнаяКисть1.Имя);
