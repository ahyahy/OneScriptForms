КолонкаДанных.ЗначениеПоУмолчанию (DataColumn.DefaultValue) Свойство

Использование

Чтение и запись.

Значение

Тип: Произвольный.

Возвращает или задает значение по умолчанию для колонки при создании новых строк.

Примечание

Пример

Колонка2.ЗначениеПоУмолчанию = 12345678901234567890.125;
