Графика.ИзИзображения (Graphics.FromImage) Метод

Синтаксис

ИзИзображения(<Изображение>)

Параметры

Изображение (обязательный)
Тип: Картинка (Bitmap).
Объект Картинка (Bitmap), из которого создается новый объект Графика (Graphics).

Возвращаемое значение

Тип: Графика (Graphics).

Этот метод возвращает новый объект Графика (Graphics) для указанного объекта Картинка (Bitmap).

Описание

Создает новый объект Графика (Graphics) из указанного объекта Картинка (Bitmap).

Примечание

Поверхностью рисования будет служить выбранное изображение.

Следует всегда вызывать метод Освободить (Dispose) для освобождения ресурсов, созданных этим методом.

Пример

Изображение1 = Ф.Картинка(СтрМасленица1);
Графика1 = Форма1.СоздатьГрафику().ИзИзображения(Изображение1);
