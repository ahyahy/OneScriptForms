РежимВыбораТаблицы (DataGridViewSelectionMode) Перечисление

Описание

Описывает способ выбора ячеек элемента управления Таблица (DataGridView).

Члены

Имя члена Описание Значение
ЗаголовокКолонки (ColumnHeaderSelect) Колонка будет выбрана путем щелчка ячейки заголовка колонки. Отдельную ячейку можно выбрать, щелкнув ее. 4
ЗаголовокСтроки (RowHeaderSelect) Строка может быть выбрана путем щелчка заголовка строки. Отдельную ячейку можно выбрать, щелкнув ее. 3
Колонка (FullColumnSelect) Вся колонка может быть выбрана путем щелчка заголовка колонки или ячейки, содержащейся в этой колонке. 2
Строка (FullRowSelect) Вся строка может быть выбрана путем щелчка заголовка строки или ячейки, содержащейся в этой строке. 1
Ячейка (CellSelect) Можно выбрать одну или несколько отдельных ячеек. 0
