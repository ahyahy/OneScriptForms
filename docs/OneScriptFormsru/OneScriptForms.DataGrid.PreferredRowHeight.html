СеткаДанных.ПредпочтительнаяВысотаСтрок (DataGrid.PreferredRowHeight) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает предпочтительную высоту строк для элемента управления СеткаДанных (DataGrid).

Примечание

Задать значение этого свойства нужно перед установкой значений свойств ИсточникДанных (DataSource) и ЧленДанных (DataMember), либо отдельно, либо с помощью метода УстановитьСвязьДанных (SetDataBinding), иначе свойство не окажет никакого воздействия. Если создан СтильТаблицыСеткиДанных (DataGridTableStyle), свойство будет переопределено в свойстве СтильТаблицыСеткиДанных.ПредпочтительнаяВысотаСтрок (DataGridTableStyle.PreferredRowHeight).

Пример

СеткаДанных1.ПредпочтительнаяВысотаСтрок = 30;
Полный пример кода
