РежимСортировки (DataGridViewColumnSortMode) Перечисление

Описание

Указывает, как пользователь может сортировать колонку объекта Таблица (DataGridView).

Члены

Имя члена Описание Значение
Автоматически (Automatic) Пользователь может сортировать колонку, щелкнув заголовок колонки (или нажав клавишу F3 в ячейке), если только заголовки колонок не используются для выборки. Глиф сортировки отображается автоматически. 1
НеСортируемый (NotSortable) Колонка может быть отсортирована только программным способом, однако он не предназначен для сортировки, поэтому заголовок колонки не будет содержать места для глифа сортировки. 0
Программный (Programmatic) Колонка может быть отсортирована только программным способом, и заголовок колонки будет содержать место для глифа сортировки. 2
