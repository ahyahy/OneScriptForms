ТаблицыДанных (DataTableCollection) Методы

Методы класса ТаблицыДанных (DataTableCollection) перечислены ниже. Полный список членов класса ТаблицыДанных (DataTableCollection) смотрите в разделе ТаблицыДанных (DataTableCollection) Члены .

Методы

Добавить (Add) Добавляет объект ТаблицаДанных (DataTable) в коллекцию.
Удалить (Remove) Удаляет указанный объект ТаблицаДанных (DataTable) из коллекции.
УдалитьПоИндексу (RemoveAt) Удаляет таблицу с указанным индексом из коллекции.
Элемент (Item) Возвращает объект ТаблицаДанных (DataTable) по указанному индексу.
