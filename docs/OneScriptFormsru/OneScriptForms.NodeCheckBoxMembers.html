Конструктор

ФлажокУзла (NodeCheckBox) Конструктор Создаёт новый экземпляр класса ФлажокУзла (NodeCheckBox).

Свойства

Колонка (ParentColumn) Объект КолонкаДереваЗначений (TreeColumn), которому принадлежит флажок узла.
ЛевыйОтступ (LeftMargin) Возвращает или задает размер левого отступа.
Редактируемый (EditEnabled) Возвращает или задает значение, определяющее доступно ли редактирование флажка узла.
СостояниеФлажкаИзменено (CheckStateChanged) Возвращает или задает код для выполнения при изменении состояния флажка узла.
ТриСостояния (ThreeState) Возвращает или задает значение, указывающее, допускет ли флажок три контрольных состояния, а не два.

Методы

ПолучитьЗначение (GetValue) Возвращает одно из значений перечисления СостояниеФлажка (CheckState).
УстановитьЗначение (SetValue) Задает значение для флажка узла дерева значений.
