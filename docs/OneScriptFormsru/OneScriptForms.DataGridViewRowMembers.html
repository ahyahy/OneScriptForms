Обзор СтрокаТаблицы (DataGridViewRow)

Конструктор

СтрокаТаблицы (DataGridViewRow) Конструктор Создаёт новый экземпляр класса СтрокаТаблицы (DataGridViewRow).

Свойства

Выбран (Selected) Возвращает или задает значение, указывающее, выбрана ли строка.
Высота (Height) Возвращает или задает текущую высоту строки.
ВысотаРазделителя (DividerHeight) Возвращает или задает высоту разделителя строки в пикселях.
ЗаголовокСтроки (HeaderCell) Возвращает или задает ячейку заголовка строки.
Закреплено (Frozen) Возвращает или задает значение, показывающее, является ли строка замороженной.
ИзменяемыйРазмер (Resizable) Возвращает или задает значение, указывающее, могут ли пользователи изменять размер строки.
Индекс (Index) (унаследовано от ПолосаТаблицы (DataGridViewBand)) Возвращает относительную позицию диапазона в элементе управления Таблица (DataGridView).
Метка (Tag) (унаследовано от ПолосаТаблицы (DataGridViewBand)) Возвращает объект, содержащий дополнительные данные.
МинимальнаяВысота (MinimumHeight) Возвращает или задает минимальную высоту строки.
Отображается (Displayed) Возвращает значение, указывающее, отображается ли данная строка на экране.
Отображать (Visible) Возвращает или задает значение, показывающее, является ли строка видимой.
СтильЯчейки (DefaultCellStyle) Возвращает или задает стили по умолчанию для строки, которые используются для визуализации ячеек в строке, если стили не переопределены.
Таблица (DataGridView) (унаследовано от ЭлементТаблицы (DataGridViewElement)) Возвращает элемент управления Таблица (DataGridView), связанный с данным элементом.
ТолькоЧтение (ReadOnly) Возвращает или задает значение, указывающее, является ли строка доступной только для чтения.
ЭтоНоваяСтрока (IsNewRow) Возвращает значение, указывающее, является ли строка строкой для новых записей.
Ячейки (Cells) Возвращает коллекцию ячеек, заполняющих строку.

Методы

Ячейки (Cells) Возвращает объект ЯчейкаТаблицы (DataGridViewCell) по указанному индексу.
