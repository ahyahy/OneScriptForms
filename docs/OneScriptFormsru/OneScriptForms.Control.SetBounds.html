ЭлементУправления.УстановитьГраницы (Control.SetBounds) Метод

Синтаксис

УстановитьГраницы(<X>, <Y>, <Ширина>, <Высота>)

Параметры

X (обязательный)
Тип: Число.
Y (обязательный)
Тип: Число.
Ширина (обязательный)
Тип: Число.
Высота (обязательный)
Тип: Число.

Возвращаемое значение

Описание

Задает указанные границы элемента управления для указанного местоположения и размера.

Примечание

Пример

Форма1.УстановитьГраницы(20, 20, 500, 500);

Форма1.Отображать = Истина;
