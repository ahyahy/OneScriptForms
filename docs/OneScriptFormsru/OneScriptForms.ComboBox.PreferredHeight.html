ПолеВыбора.ПредпочтительнаяВысота (ComboBox.PreferredHeight) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает предпочтительную высоту ПолеВыбора (ComboBox).

Предпочтительная высота в пикселях области элемента в поле со списком.

Примечание

Предпочтительная высота - это значение, основанное на высоте шрифта и параметрах границы.

Пример

Сообщить("ПолеВыбора1.ПредпочтительнаяВысота = " + ПолеВыбора1.ПредпочтительнаяВысота);
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Ширина = 310;
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ПолеВыбора1 = Ф.ПолеВыбора();
ПолеВыбора1.Родитель = Форма1;
ПолеВыбора1.Ширина = 300;

ЭлементыПоляВыбора1 = ПолеВыбора1.Элементы;
ЭлементыПоляВыбора1.Добавить("Москва");
ЭлементыПоляВыбора1.Добавить("Хабаровск");
