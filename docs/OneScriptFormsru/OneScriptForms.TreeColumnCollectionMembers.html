Обзор КолонкиДереваЗначений (TreeColumnCollection)

Свойства

Количество (Count) Возвращает количество объектов в коллекции.

Методы

Вставить (Insert) Вставляет существующую колонку в коллекцию колонок по указанному индексу.
Добавить (Add) Добавляет колонку в коллекцию.
Индекс (IndexOf) Возвращает отсчитываемый от нуля индекс первого вхождения, найденной колонки.
Очистить (Clear) Очищает коллекцию.
Содержит (Contains) Определяет, входит ли колонка в коллекцию.
Удалить (Remove) Удаляет колонку из коллекции.
УдалитьПоИндексу (RemoveAt) Удаляет колонку по указанному индексу из коллекции.
УстановитьКолонку (SetColumn) Устанавливает колонку по указанному индексу.
