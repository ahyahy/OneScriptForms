НачальноеПоложениеФормы (FormStartPosition) Перечисление

Описание

Определяет начальное положение формы.

Члены

Имя члена Описание Значение
Вручную (Manual) Положение формы определяется свойством Положение (Location). 0
ГраницыОкнаПоУмолчанию (WindowsDefaultBounds) Положение формы и ее границы определены в Windows по умолчанию. 3
ПоложениеОкнаПоУмолчанию (WindowsDefaultLocation) Форма с заданными размерами размещается в расположении, определенном по умолчанию в Windows. 2
ЦентрРодителя (CenterParent) Форма выравнивается по центру в границах родительской формы. 4
ЦентрЭкрана (CenterScreen) Форма выравнивается по центру текущего экрана. 1
