ПолеСписка.ИндексВыбранногоИзменен (ListBox.SelectedIndexChanged) Свойство

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код для выполнения, при изменении свойства ИндексВыбранного (SelectedIndex).
