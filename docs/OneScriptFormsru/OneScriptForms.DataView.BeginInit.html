ПредставлениеДанных.НачатьИнициализацию (DataView.BeginInit) Метод

Синтаксис

НачатьИнициализацию()

Параметры

Возвращаемое значение

Описание

Запускает инициализацию ПредставлениеДанных (DataView), используемого в форме или другим компонентом. Инициализация происходит во время выполнения.

Примечание

Использование методов ЗакончитьИнициализацию (EndInit) и НачатьИнициализацию (BeginInit) предотвращает использование элемента управления перед его полной инициализацией.

Пример

ПредставлениеДанных1.НачатьИнициализацию();
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

СеткаДанных1 = Форма1.ЭлементыУправления.Добавить(Ф.СеткаДанных());
СеткаДанных1.Стыковка = Ф.СтильСтыковки.Заполнение;
	
ТаблицаДанных1 = Ф.ТаблицаДанных();
ТаблицаДанных1.ИмяТаблицы = "ТД1";

ТаблицаДанных1.Колонки.Добавить(Ф.КолонкаДанных("Как_Отобразить_Элемент", Ф.ТипДанных.Строка));
ТаблицаДанных1.Колонки.Добавить(Ф.КолонкаДанных("Значение_элемента", Ф.ТипДанных.Строка));
Для А = 1 По 5 Цикл
	ТекСтрока = ТаблицаДанных1.Строки.Добавить(ТаблицаДанных1.НоваяСтрока());
	ТекСтрока.УстановитьЭлемент(0, "СтрКнопка" + А);
	ТекСтрока.УстановитьЭлемент(1, "Кнопка" + А);
КонецЦикла;

ПредставлениеДанных1 = Ф.ПредставлениеДанных();

ПредставлениеДанных1.НачатьИнициализацию();

ПредставлениеДанных1.Таблица = ТаблицаДанных1;
СеткаДанных1.ИсточникДанных = ПредставлениеДанных1;

ПредставлениеДанных1.ЗакончитьИнициализацию();

ПредставлениеСтрокиДанных1 = ПредставлениеДанных1.ДобавитьНовуюСтроку();
ПредставлениеСтрокиДанных1.УстановитьЭлемент(0, "СтрКнопка6");
ПредставлениеСтрокиДанных1.УстановитьЭлемент("Значение_элемента", "Кнопка6");
ПредставлениеСтрокиДанных1.ЗакончитьРедактирование();

Ф.ЗапуститьОбработкуСобытий();
