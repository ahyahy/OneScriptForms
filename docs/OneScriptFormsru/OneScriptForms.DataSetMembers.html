Обзор НаборДанных (DataSet)

Конструктор

НаборДанных (DataSet) Конструктор Создаёт новый экземпляр класса НаборДанных (DataSet).

Свойства

ИмяНабораДанных (DataSetName) Возвращает или задает имя текущего набора данных.
Таблицы (Tables) Возвращает коллекцию таблиц, содержащихся в НаборДанных (DataSet).

Методы

Изменен (HasChanges) Возвращает значение, указывающее, имеет ли НаборДанных (DataSet) изменения, в том числе новые, удаленные или измененные строки.
ОтказИзменений (RejectChanges) Откат всех изменений, внесенных в НаборДанных (DataSet) с момента его создания или со времени последнего вызова метода НаборДанных.ПринятьИзменения (DataSet.AcceptChanges).
ПринятьИзменения (AcceptChanges) Фиксирует все изменения, внесенные в НаборДанных (DataSet) после его загрузки или после последнего вызова метода НаборДанных.ПринятьИзменения (DataSet.AcceptChanges).
Таблицы (Tables) Возвращает значение по указанному индексу.
