КнопкаПанелиИнструментов.Метка (ToolBarButton.Tag) Свойство

Использование

Только чтение.

Значение

Тип: Коллекция (Collection).

Возвращает объект, содержащий дополнительные данные.

Примечание

Свойство Метка (Tag) позволяет расширить возможности объекта с помощью дополнительных данных с различными типами значений.

Пример

Метка = Кнопка1.Метка;
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

ПанельИнструментов1 = Форма1.ЭлементыУправления.Добавить(Ф.ПанельИнструментов());
Кнопка1 = Ф.КнопкаПанелиИнструментов("Открыть");
ПанельИнструментов1.Кнопки.Добавить(Кнопка1);

Метка = Кнопка1.Метка;
Метка.Добавить(Кнопка1, "Объект");
Метка.Добавить("Кнопка1", "Имя");
Метка.Добавить(Истина, "Булево");
Метка.Добавить(156.35, "Число");
Метка.Добавить(Дата(1984,03,26,06,24,21), "Дата");

Сообщить("Объект = " + Кнопка1.Метка.Элемент(0).Текст);
Сообщить("Имя = " + Кнопка1.Метка.Элемент("Имя"));
Сообщить("Булево = " + Кнопка1.Метка.Элемент("Булево"));
Сообщить("Число = " + Кнопка1.Метка.Элемент("Число") / 2);
Сообщить("Дата = " + Кнопка1.Метка.Элемент("Дата"));

Ф.ЗапуститьОбработкуСобытий();
