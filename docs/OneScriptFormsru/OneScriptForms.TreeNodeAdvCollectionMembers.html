Обзор УзлыДереваЗначений (TreeNodeAdvCollection)

Свойства

Количество (Count) Возвращает количество объектов в коллекции.

Методы

Вставить (Insert) Вставляет существующий узел в коллекцию узлов по указанному индексу.
Добавить (Add) Добавляет узел в коллекцию.
Индекс (IndexOf) Возвращает отсчитываемый от нуля индекс первого вхождения, найденного узла.
Очистить (Clear) Очищает коллекцию.
Содержит (Contains) Определяет, входит ли узел в коллекцию.
Удалить (Remove) Удаляет узел из коллекции.
УдалитьПоИндексу (RemoveAt) Удаляет узел по указанному индексу из коллекции.
УстановитьУзел (SetNode) Устанавливает узел по указанному индексу.
Элемент (Item) Возвращает объект УзелДереваЗначений (TreeNodeAdv) по указанному индексу в коллекции.
