МаскаПоляВвода.СимволПароля (MaskedTextBox.PasswordChar) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает символ, отображаемый в качестве замены для данных, вводимых пользователем.

Примечание

Символ пароля не должен совпадать с текущим знаком приглашения - МаскаПоляВвода.СимволПриглашения (MaskedTextBox.PromptChar). Эти два знака должны отличаться.

Пример

МаскаПоляВвода1.СимволПароля = "r";
Полный пример кода 




ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptForms\OneScriptForms\bin\Debug\OneScriptForms.dll");
Ф = Новый ФормыДляОдноСкрипта();
Форма1 = Ф.Форма();
Форма1.Отображать = Истина;
Форма1.Показать();
Форма1.Активизировать();

МаскаПоляВвода1 = Форма1.ЭлементыУправления.Добавить(Ф.МаскаПоляВвода("LLL LLL-LLLL"));

МаскаПоляВвода1.СимволПароля = "r";
Сообщить("СимволПароля = " + МаскаПоляВвода1.СимволПароля);

Ф.ЗапуститьОбработкуСобытий();
