Цвет (Color) Методы

Методы класса Цвет (Color) перечислены ниже. Полный список членов класса Цвет (Color) смотрите в разделе Цвет (Color) Члены.

Методы

ИзArgb (FromArgb) Создает цвет из четырех 8-разрядных ARGB (альфа, красный, зеленый и синий).
ИзRgb (FromRgb) Создает цвет из указанного 8-битного значения цвета (красный, зеленый и синий). Значение альфа неявно 255 (полностью непрозрачный).
ИзИмени (FromName) Создает цвет из указанного имени предопределенного цвета.
ПолучитьArgb (ToArgb) Возвращает 32-битное ARGB-значение цвета.
