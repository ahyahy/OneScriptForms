ЭлементУправления (Control) Члены

Обзор ЭлементУправления (Control)

Свойства

ВерсияПродукта (ProductVersion) Возвращает версию сборки.
Верх (Top) Возвращает или задает y-координату верхнего края элемента управления в пикселях.
Высота (Height) Возвращает или задает высоту элемента управления.
ВысотаШрифта (FontHeight) Возвращает высоту шрифта, используемого элементом управления.
Границы (Bounds) Возвращает или задает размер и расположение элемента управления.
ДвойноеНажатие (DoubleClick) Возвращает или задает код, когда элемент управления дважды щелкнут.
Доступность (Enabled) Возвращает или задает значение, указывающее, может ли элемент управления реагировать на действия пользователя.
ЖирныйШрифт (FontBold) Возвращает или задает, является ли стиль шрифта, используемый элементом управления, полужирным.
Захват (Capture) Возвращает или задает значение, указывающее, захватил ли элемент мышь.
Имя (Name) Возвращает или задает имя элемента управления.
ИмяПродукта (ProductName) Возвращает имя сборки, содержащей элемент управления.
ИмяШрифта (FontName) Возвращает или задает имя шрифта, используемого элементом управления.
ИспользоватьКурсорОжидания (UseWaitCursor) Возвращает или задает значение, указывающее, следует ли использовать курсор ожидания для текущего элемента управления и всех дочерних элементов управления.
КлавишаВверх (KeyUp) Возвращает или задает код для события, которое происходит, когда отпускается клавиша, если элемент управления имеет фокус.
КлавишаВниз (KeyDown) Возвращает или задает код, выполняемый при нажатии клавиши в то время как элемент управления имеет фокус.
КлавишаНажата (KeyPress) Возвращает или задает код, выполняемый при нажатии клавиши в то время как элемент управления имеет фокус.
КлиентВысота (ClientHeight) Возвращает или задает высоту клиентской области элемента управления.
КлиентПрямоугольник (ClientRectangle) Возвращает прямоугольник, представляющий клиентскую область элемента управления.
КлиентРазмер (ClientSize) Возвращает или задает высоту и ширину клиентской области элемента управления.
КлиентШирина (ClientWidth) Возвращает или задает ширину (в пикселях) клиентской области элемента управления.
КнопкиМыши (MouseButtons) Возвращает значение, указывающее, какая из кнопок мыши была нажата.
КонтекстноеМеню (ContextMenu) Возвращает или задает контекстное меню, связанное с элементом управления.
Курсор (Cursor) Возвращает или задает курсор, который отображается, когда указатель мыши находится над элементом управления.
Лево (Left) Возвращает или задает x-координату левого края элемента управления в пикселях.
Метка (Tag) Возвращает объект, содержащий дополнительные данные.
МышьНадЭлементом (MouseEnter) Возвращает или задает код для выполнения, когда указатель мыши находится над элементом управления.
МышьПокинулаЭлемент (MouseLeave) Возвращает или задает код для выполнения, когда указатель мыши покидает пределы элемента управления.
Нажатие (Click) Возвращает или задает код для выполнения при нажатии элемента управления.
Низ (Bottom) Возвращает расстояние между нижним краем элемента управления и верхним краем клиентской области контейнера.
ОсновнойЦвет (ForeColor) Возвращает или задает цвет переднего плана элемента управления.
Отображать (Visible) Возвращает или задает значение, указывающее, отображаются ли элемент управления и все его дочерние элементы управления. Истина, если элемент управления отображается; в противном случае - Ложь.
ПозицияМыши (MousePosition) Возвращает позицию курсора мыши в экранных координатах.
Положение (Location) Возвращает или задает координаты верхнего левого угла элемента управления относительно верхнего левого угла его контейнера.
ПоложениеИзменено (LocationChanged) Возвращает или задает код для выполнения при изменении свойства Положение (Location).
ПорядокОбхода (TabIndex) Возвращает или задает последовательность перехода по клавише TAB между элементами управления внутри контейнера.
Право (Right) Возвращает расстояние между правым краем элемента управления и левым краем его контейнера.
ПриВходе (Enter) Возвращает или задает код для выполнения при входе в элемент управления.
ПриЗадержкеМыши (MouseHover) Возвращает или задает код для выполнения когда указатель мыши задерживается на элементе управления.
ПриНажатииКнопкиМыши (MouseDown) Возвращает или задает код для выполнения при нажатии кнопки мыши, если указатель мыши находится на элементе управления.
ПриОтпусканииМыши (MouseUp) Возвращает или задает код для выполнения при отпускании кнопки мыши, когда указатель мыши находится на элементе управления.
ПриПеремещении (Move) Возвращает или задает код для выполнения при перемещении элемента управления.
ПриПеремещенииМыши (MouseMove) Возвращает или задает код для выполнения, когда указатель мыши переместился над элементом управления.
ПриПерерисовке (Paint) Возвращает или задает код для выполнения, когда элемент управления перерисован.
ПриПотереФокуса (LostFocus) Возвращает или задает код для выполнения, когда элемент управления теряет фокус.
ПриУходе (Leave) Возвращает или задает код для выполнения, когда фокус ввода покидает элемент управления.
Размер (Size) Возвращает или задает высоту и ширину элемента управления.
РазмерИзменен (SizeChanged) Возвращает или задает код для выполнения при изменении значения свойства Размер (Size).
РазмерШрифта (FontSize) Возвращает или задает размер шрифта, используемого элементом управления.
Родитель (Parent) Возвращает или задает родительский контейнер элемента управления.
Стыковка (Dock) Возвращает или задает, к какому краю родительского контейнера прикреплен элемент управления.
Сфокусирован (Focused) Возвращает значение, указывающее, имеет ли объект фокусировку.
ТабФокус (TabStop) Возвращает или задает значение, указывающее, может ли пользователь перевести фокус на данный элемент управления при помощи клавиши TAB.
Текст (Text) Возвращает или задает текст, связанный с этим элементом управления.
ТекстИзменен (TextChanged) Возвращает или задает код для выполнения, при изменении свойства Текст (Text).
Тип (Type) (унаследовано от Компонент (Component)) Возвращает тип текущего экземпляра.
Фокусируемый (CanFocus) Возвращает значение, указывающее, может ли элемент управления получать фокус.
ФоновоеИзображение (BackgroundImage) Возвращает или задает фоновое изображение, отображаемое в элементе управления.
ЦветФона (BackColor) Возвращает или задает цвет фона для элемента управления.
Ширина (Width) Возвращает или задает ширину элемента управления.
Шрифт (Font) Возвращает или задает шрифт текста, отображаемый элементом управления.
ЭлементВерхнегоУровня (TopLevelControl) Возвращает родительский элемент управления, не имеющий другого родительского элемента управления Форма (Form). Как правило, им является внешний объект Форма (Form), в котором содержится элемент управления.
ЭлементДобавлен (ControlAdded) Возвращает или задает код для выполнения при добавлении нового элемента управления в ЭлементыУправления (ControlCollection).
ЭлементУдален (ControlRemoved) Возвращает или задает код для выполнения при удалении элемента управления из ЭлементыУправления (ControlCollection).
ЭлементыУправления (Controls) Возвращает коллекцию элементов управления, содержащихся в элементе управления.
Якорь (Anchor) Возвращает или задает границы контейнера, с которым связан элемент управления, и определяет способ изменения размеров элемента управления при изменении размеров его родительского элемента.

Методы

Актуализировать (Refresh) Заставляет элемент управления аннулировать клиентскую область и немедленно перерисовать себя и любые дочерние элементы управления.
Аннулировать (Invalidate) Делает недействительной всю поверхность элемента управления и вызывает его перерисовку.
ВозобновитьРазмещение (ResumeLayout) Возобновляет обычную логику макета.
ВосстановитьФоновоеИзображение (ResetBackgroundImage) Сбрасывает свойство ФоновоеИзображение (BackgroundImage) на значение по умолчанию.
Выбрать (Select) Активирует элемент управления.
ВыполнитьРазмещение (PerformLayout) Вызывает в элементе управления принудительное применение логики макета ко всем его дочерним элементам управления.
ДочернийПоКоординатам (GetChildAtPoint) Создает объект Графика (Graphics) для элемента управления.
ЗавершитьОбновление (EndUpdate) Возобновление прорисовки элемента управления после того как прорисовка была остановлена методом НачатьОбновление (BeginUpdate).
НаЗаднийПлан (SendToBack) Отправляет элемент управления в конец z-порядка.
НайтиФорму (FindForm) Возвращает форму, в которой находится элемент управления.
НайтиЭлемент (FindControl) Выполняет поиск по имени элемента управления в коллекции элементов управления текущего контейнера.
НаПереднийПлан (BringToFront) Помещает элемент управления в начало z-порядка.
НачатьОбновление (BeginUpdate) Предотвращает прорисовку элемента управления до тех пор, пока не будет вызван метод ЗавершитьОбновление (EndUpdate).
Обновить (Update) Вызывает перерисовку элементом управления недопустимых областей клиентской области.
ОбновитьСтили (UpdateStyles) Вызывает принудительное повторное применение назначенных стилей к элементу управления.
Освободить (Dispose) (унаследовано от Компонент (Component)) Освобождает ресурсы, используемые Компонент (Component).
Показать (Show) Отображает элемент управления пользователю.
ПолучитьСтиль (GetStyle) Возвращает значение указанного бита стиля для данного элемента управления.
ПриостановитьРазмещение (SuspendLayout) Временно приостанавливает логику компоновки для элемента управления.
Скрыть (Hide) Скрывает элемент управления от пользователя.
СледующийЭлемент (GetNextControl) Возвращает следующий или предыдущий элемент среди дочерних элементов управления в последовательности клавиши TAB.
СоздатьГрафику (CreateGraphics) Создает объект Графика (Graphics) для элемента управления.
СоздатьЭлемент (CreateControl) Вызывает принудительное создание видимого элемента управления, включая создание дескриптора и всех видимых дочерних элементов.
ТочкаНаКлиенте (PointToClient) Вычисляет местоположение указанной экранной точки в клиентские координаты.
ТочкаНаЭкране (PointToScreen) Вычисляет местоположение указанной точки клиента в экранные координаты.
УстановитьГраницы (SetBounds) Задает указанные границы элемента управления для указанного местоположения и размера.
УстановитьСтиль (SetStyle) Включает или выключает указанный стиль.
Фокус (Focus) Устанавливает фокус ввода на элемент управления.
Центр (Center) Центрирует элемент управления внутри своего контейнера.
ЭлементУправления (Control) Возвращает элемент управления ЭлементУправления (Control) по указанному индексу.
ЭлементыУправления (Controls) Возвращает элемент управления с указанным индексом в коллекции.
