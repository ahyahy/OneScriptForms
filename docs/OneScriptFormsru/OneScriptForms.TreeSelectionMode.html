РежимВыбораДереваЗначений (TreeSelectionMode) Перечисление

Описание

Описывает способ выбора узлов элемента управления ДеревоЗначений (TreeViewAdv).

Члены

Имя члена Описание Значение
Множественный (Multi) Можно выбрать несколько узлов одновременно. 1
МножественныйДляРодителя (MultiSameParent) Можно выбрать несколько узлов одновременно в пределах одного родителя. 2
Одиночный (Single) Можно выбрать только один узел. 0
