ЭлементыМеню.Содержит (MenuItemCollection.Contains) Метод

Синтаксис

Содержит(<Элемент>)

Параметры

Элемент (обязательный)
Тип: ЭлементМеню (MenuItem).
Объект ЭлементМеню (MenuItem) для поиска в коллекции.

Возвращаемое значение

Тип: Булево.

Истина - ЭлементМеню (MenuItem) является членом коллекции, Ложь - в противном случае.

Описание

Определяет, является ли указанный ЭлементМеню (MenuItem) членом коллекции.

Примечание

Пример

Содержит = ГлавноеМеню1.ЭлементыМеню.Содержит(Подменю_Файл);
