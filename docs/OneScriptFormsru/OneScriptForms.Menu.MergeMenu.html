Меню.СлитьМеню (Menu.MergeMenu) Метод

Синтаксис

СлитьМеню(<Меню>)

Параметры

Меню (обязательный)
Тип: Произвольный.
Объект ГлавноеМеню (MainMenu) или КонтекстноеМеню (ContextMenu), чьи элементы меню объединяются с элементами текущего меню.

Возвращаемое значение

Описание

Объединяет объекты ЭлементМеню (MenuItem) одного меню с текущим меню.

Примечание

Этот метод выполняет слияние объектов ЭлементМеню (MenuItem) из одного меню с текущим меню. Объекты ЭлементМеню (MenuItem) объединяются в соответствии со значениями свойств ЭлементМеню.ТипСлияния (MenuItem.MergeType) и ЭлементМеню.ПорядокСлияния (MenuItem.MergeOrder).

Этот метод может быть вызван классами ГлавноеМеню (MainMenu) или КонтекстноеМеню (ContextMenu) для объединения двух или более объектов, производных от класса Меню (Menu).

Пример

Слитое.СлитьМеню(Меню2.КлонироватьМеню());
