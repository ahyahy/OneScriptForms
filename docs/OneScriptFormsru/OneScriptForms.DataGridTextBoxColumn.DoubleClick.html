СтильКолонкиПолеВвода.ДвойноеНажатие (DataGridTextBoxColumn.DoubleClick) Свойство

Использование

Чтение и запись.

Значение

Тип: Произвольный.

Возвращает или задает код для выполнения, когда ячейка колонки дважды щелкнута.

Примечание

Свойство может быть задано типом значения Действие из библиотеки односкрипта, или типом значения Действие (Action).

Процедура обработки события должна быть объявлена с ключевым словом Экспорт.

Пример

СтильКолонкиПолеВвода1.ДвойноеНажатие = Ф.Действие(ЭтотОбъект, "СтильКолонкиПолеВвода1_ДвойноеНажатие");
