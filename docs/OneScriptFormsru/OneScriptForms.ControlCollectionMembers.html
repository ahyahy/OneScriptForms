Обзор ЭлементыУправления (ControlCollection)

Свойства

Количество (Count) (унаследовано от КоллекцияОснова (CollectionBase)) Возвращает общее количество объектов в коллекции.

Методы

Добавить (Add) Добавляет элемент управления в коллекцию.
ДобавитьКнопку (AddButton) Добавляет новый элемент управления Кнопка (Button) с указанным текстом, размером и местоположением.
Индекс (IndexOf) Возвращает отсчитываемый от нуля индекс в коллекции для найденного элемента.
Очистить (Clear) (унаследовано от КоллекцияОснова (CollectionBase)) Удаляет все объекты из коллекции.
Содержит (Contains) Определяет, является ли указанный элемент управления членом коллекции.
Удалить (Remove) Удаляет элемент управления из коллекции;
УдалитьПоИндексу (RemoveAt) (унаследовано от КоллекцияОснова (CollectionBase)) Удаляет объект по указанному индексу из коллекции.
УстановитьИндексДочернего (SetChildIndex) Устанавливает индекс в коллекции для дочернего элемента управления.
Элемент (Item) Возвращает элемент управления с указанным индексом в коллекции.
