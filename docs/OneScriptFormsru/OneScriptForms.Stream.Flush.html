Поток.Сбросить (Stream.Flush) Метод

Синтаксис

Сбросить()

Параметры

Возвращаемое значение

Описание

Очищает все буферы данного потока и вызывает запись данных буферов в базовое устройство.

Примечание

Пример

Поток1.Сбросить();
