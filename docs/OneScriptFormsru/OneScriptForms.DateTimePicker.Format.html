ПолеКалендаря.Формат (DateTimePicker.Format) Свойство

Использование

Чтение и запись.

Значение

Тип: ФорматПоляКалендаря (FormatDateTimePicker).

Возвращает или задает формат даты и времени, отображаемых в элементе управления.

Одно из значений перечисления ФорматПоляКалендаря (FormatDateTimePicker). Значение по умолчанию - Длинный (Long).

Для того, чтобы свойство ПользовательскийФормат (CustomFormat) повлияло на форматирование отображаемой даты и времени, свойству Формат (Format) должно быть присвоено значение Пользовательский (Custom).

Примечание

Пример

ПолеКалендаря1.Формат = Ф.ФорматПоляКалендаря.Короткий;
