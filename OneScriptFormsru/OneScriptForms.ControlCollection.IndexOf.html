ЭлементыУправления.Индекс (ControlCollection.IndexOf) Метод

Синтаксис

Индекс(<Значение>)

Параметры

Значение (обязательный)
Тип: Произвольный.
Значение элемента для поиска в коллекции.

Возвращаемое значение

Тип: Число.

Отсчитываемый от нуля индекс элемента в коллекции; в противном случае -1.

Описание

Возвращает отсчитываемый от нуля индекс в коллекции для найденного элемента.

Примечание

Пример

Индекс1 = Форма1.ЭлементыУправления.Индекс(Кнопка2);
