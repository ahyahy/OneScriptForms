ЭлементыУправления.Элемент (ControlCollection.Item) Метод

Синтаксис

Элемент(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.
Индекс элемента управления в коллекции.

Возвращаемое значение

Тип: Произвольный.

Описание

Возвращает элемент управления с указанным индексом в коллекции.

Примечание

Пример

Сообщить("Текст = " + Форма1.ЭлементыУправления.Элемент(2).Текст);
